# 《深空信号》角色设定

## 主要角色

### 林远 - 船长

| 项目 | 内容 |
|------|------|
| **年龄** | 42岁 |
| **外貌** | 国字脸，短发微白，深邃眼神，左眉有一道旧伤疤（火星事故留下） |
| **服装** | 深蓝色船长制服，肩章四道金杠，胸前佩戴地球徽章 |
| **装备** | 便携式数据终端（老旧但可靠），父亲遗留的机械表 |

**人物小传：**
林远出生于月球殖民地，父亲是第一批火星移民工程师。12岁时，父亲在火星尘暴中为救同事牺牲，临终前将机械表交给他，说："人类必须向前，但别忘记回头看。"这句话成为林远的人生信条。他加入深空探测计划，既是为了延续父亲的遗志，也是为了证明人类可以安全地探索未知。然而，内心深处，他对"未知"有着近乎执念的渴望——这是他对父亲未竟事业的继承，也是他对童年创伤的补偿。

**性格成因：** 父亲的牺牲让他既渴望探索（继承遗志），又害怕失去（过度谨慎的矛盾）
**核心价值观：** 探索是人类的使命，但生命比发现更重要
**动机与欲望：** 成为第一个接触外星文明的人类船长，名留青史

---

### 苏晴 - 首席科学家

| 项目 | 内容 |
|------|------|
| **年龄** | 35岁 |
| **外貌** | 齐肩黑发，锐利眼神，常年佩戴防辐射眼镜，手指有数据笔茧 |
| **服装** | 白色科研制服，左臂佩戴量子计算机接口手环 |
| **装备** | 全息数据分析仪，随身携带父亲的笔记本（手写物理公式） |

**人物小传：**
苏晴是地球顶尖物理学家之女，从小在实验室长大。母亲因一次实验事故去世，父亲从此变得极度谨慎，教导她："数据不会撒谎，但直觉会杀人。"苏晴继承了父亲的理性，却压抑了自己的感性。她加入"黎明号"是为了寻找外星文明的科学证据，证明人类在宇宙中并不孤独。但当真正面对未知时，她发现自己最大的恐惧不是死亡，而是犯错——就像母亲那样。

**性格成因：** 母亲的意外让她相信理性高于一切，情感是危险的
**核心价值观：** 科学必须严谨，任何发现都需要验证
**动机与欲望：** 找到确凿的外星文明证据，完成母亲未竟的研究
