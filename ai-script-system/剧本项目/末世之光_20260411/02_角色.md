## 🎭 角色 A：阿尘

### 基本信息

| 属性 | 设定 |
|------|------|
| **姓名** | 阿尘（自称，真名已遗忘） |
| **年龄** | 38岁（末世发生时31岁，已生存7年） |
| **生日** | 末世前1988年4月（白羊座） |
| **原职业** | 核电站高级工程师 |
| **现身份** | 废土拾荒者 |
| **生存年限** | 独自在废土生存7年 |

### 详细背景故事

**末世前（7年前）：**
- 已婚，妻子林婉（35岁，教师），儿子小宇（3岁）
- 居住在东部沿海城市，生活平静
- 职业是核电站工程师，熟悉辐射防护知识
- 性格温和内敛，工作认真负责

**灾难日（7年前）：**
- "大坠落"事件：陨石群撞击地球，引发核冬天
- 当时正在核电站值班，因地下掩体幸存
- 地表通讯中断，与家人失散
- 3个月后返回地面，城市已成废墟
- 在废墟中找到家，妻儿踪迹全无

**生存期（7年间）：**
- 第1年：疯狂寻找家人，一无所获
- 第2-3年：加入幸存者团体，因目睹背叛而离开
- 第4-7年：独自生存，成为专业拾荒者
- 学会辨识辐射区、寻找净化水源、躲避变异生物
- 性格逐渐麻木，但从未完全放弃人性

**心理状态：**
- 表面：冷漠、麻木、只信自己
- 深层：强烈的幸存愧疚（为什么是我活了下来）
- 潜意识：通过保护婴儿赎罪，弥补没能保护家人的遗憾

#### 日常体征

- **呼吸：** 轻微哮喘音（辐射影响肺部），剧烈运动后咳嗽
- **体味：** 混合了汗水、金属、霉变和轻微辐射的刺鼻味道
- **体温：** 常年偏低（36.1°C），手脚冰凉
- **睡眠：** 每天4-5小时，浅眠，任何响动都会惊醒
- **饮食习惯：** 每天一顿，生吃或简单加热，不挑食但量极少
- **排泄：** 因缺水而尿液深黄且量少，大便干硬（每3-4天一次）

## 👶 角色 B：小光（婴儿）

### 基本信息

| 属性 | 设定 | 备注 |
|------|------|------|
| **姓名** | 无名，阿尘后来叫他"小光" | 因婴儿对光敏感，也象征希望之光 |
| **年龄** | 精确7个月（根据牙齿和发育程度） | 末世后出生的第一代 |
| **性别** | 不明确（襁褓遮挡，阿尘也不关心） | 观众可自行想象 |
| **体重** | 约6.5kg（偏轻，营养不良） | 正常7个月婴儿应为8-9kg |
| **身长** | 65cm | 略低于平均值 |
| **身份** | 末世奇迹 | 不知为何独自在超市存活 |
| **象征** | 希望、未来、人性的光辉、救赎 | 多重象征意义 |

### 身世之谜（推测）

**发现地点：** 废弃超市的储藏室货架上

**存活条件分析：**
- 货架上有未开封的奶粉罐头（阿尘发现时还有3罐）
- 角落有接雨水的容器（超市屋顶漏水）
- 周围有 Adults 的痕迹但人已不见
- 婴儿身上有咬过的布条（可能曾用来止血或包扎）

**父母去向推测（留给观众想象）：**
1. 父母外出寻找食物，遭遇变异生物未能返回
2. 父母受伤，自知时日无多，将婴儿藏在安全处
3. 父母一方变异，另一方带其离开或...

**名字"小光"的由来：**
- 阿尘抱着婴儿走出超市时，一缕阳光照在婴儿脸上
- 婴儿第一次睁开眼睛（出生以来可能没见过光）
- 阿尘喃喃自语："...光..."
- 结局时正式命名

|------|------|
| **体型** | 腹部异常膨大（多次生育） |
| **特征** | 腹部皮肤撕裂，可见内脏轮廓 |
| **地位** | 生育工具，受群体保护 |
| **状态** | 目前怀孕，攻击性极强 |
| **注意** | 阿尘不知道这条是母犬，否则会更警惕 |

#### 其他成员（2-3只）

- **"瞎眼"：** 双眼完全失明，靠嗅觉和听觉行动
- **"瘸腿"：** 右后腿骨折愈合变形，跑起来一瘸一拐
- **"秃尾"：** 尾巴完全脱落，屁股光秃

### 感官能力

| 感官 | 能力 | 细节 |

| **嗅觉** | 可追踪3天前的气味 | 能分辨恐惧气味（肾上腺素） |
| **痛觉** | 几乎丧失 | 受伤不会退缩，更危险 |
| **味觉** | 只有苦味感知 | 喜欢腐肉的发酵酸味 |

### 气味特征（详细描述）

**近距离（10米内）：**
- 腐烂血液的铁锈味
- 化学药剂的刺鼻酸味
- 腐败肉体的硫化氢臭（像臭鸡蛋）
- 它们自己的尿液标记（领地意识）

**阿尘的识别（50米外）：**
- 阿尘能闻到那种独特的"甜腻腐臭"
- 这是他7年来最恐惧的味道
- 一闻到立即进入战斗/逃跑模式

### 行为模式（狩猎流程）

- 评估猎物威胁等级

**阶段2：试探（第3场景中段）**
- 增加犬数至4-5只
- 缩小包围圈至50米
- 发出低吼，测试猎物心理

**阶段3：追击（第3场景高潮）**

**阶段4：围杀（第4场景）**

- 如果猎物逃脱，放弃追击（领地意识不强）

## 📖 人物小传（核心补充）

### 阿尘的完整人物小传

**【童年】**
陈默（阿尘原名）生于1988年4月，山东济南人。父亲是工厂技术员，母亲是纺织厂工人。家境普通，但父母感情很好。

小默从小就安静，喜欢躲在角落里看大人说话。他能记住每个人说话的习惯——王阿姨紧张时会摸耳垂，李叔叔说谎时会眨眼。

8岁那年，父亲在工厂事故中受伤，腿瘸了。小默看着父亲从家里的顶梁柱变成沉默寡言的人。他第一次明白：人可以挺过灾难，但会被灾难改变。

**【求学】**
2006年，陈默考上清华大学核工程专业。全村第一个考上清华的，摆了三天酒席。

在大学，他依然安静，但成绩优异。他不是最聪明的，但最勤奋。实验室的师兄说："陈默做实验，每个步骤都要写三遍检查，慢是慢，但从不出错。"

2010年读研，2013年毕业，进入东部沿海核电站工作。

**【成家】**
2014年，26岁的陈默经同事介绍，认识了林婉。

林婉是小学语文老师，比他小3岁，爱笑，说话声音温柔。他们第一次约会，陈默紧张得把咖啡洒了，林婉笑着说："你这人，连紧张都这么认真。"

2015年结婚，2017年儿子陈宇出生。小宇3个月时，陈默值夜班，在监控室里看着家里的摄像头——林婉抱着儿子哄睡，他截了屏，设为手机壁纸。

**【灾难日 - 2021年3月15日】**
那天陈默31岁，正在核电站地下控制室值班。

凌晨2点，警报响起。不是核电站的问题——是天空。陨石群撞击地球，大气层被撕裂，通讯中断。

陈默和同事们躲在掩体里3个月。出去时，世界变了。城市成了废墟，天空总是灰的，植物开始变异。

他走回住处——房子塌了，没有人。邻居说，地震时很多人往西边跑了，后来再也没回来。

他找了一年。从废墟到荒野，从城市到乡村。他学会了辨识辐射区、净化水、躲避变异生物。他找到了一些幸存者，但都不是林婉和小宇。

**【第一年的创伤】**
第一年的记忆是模糊的。陈默后来想，可能是大脑的保护机制。

他记得自己在废墟里喊，喊到嗓子出血。记得自己杀过一只变异犬，不是为了吃，是为了发泄。记得自己曾在深夜把枪（后来丢了）抵在头上，但最终没有扣扳机。

"因为我不知道他们是不是还活着，"他后来想，"如果我死了，他们回来找我怎么办？"

**【第二年的觉醒】**
第二年，陈默遇到了一群幸存者。

领头的是个退伍军人，叫老张。老张说："你一个人活不下去的，跟我们一起。"

陈默加入了。他发现自己有用——懂辐射、会修机器、做事靠谱。他开始觉得，也许活下去有意义。

但年底发生了分歧。团队里有人提议去抢另一个小团队的物资，老张同意了。陈默反对："他们也是幸存者。"

老张说："不抢他们，我们就得死。你想死吗？"

陈默离开了。那天晚上，他看着天上的灰云，明白了一件事：灾难不只改变了世界，也改变了人。有些人，再也变不回来了。

**【第三至七年：阿尘的诞生】**
第三年，陈默不再用自己的名字。他自称"阿尘"——尘埃的尘，意思是"无足轻重，随风飘荡"。

他成了专业拾荒者。一个人，一把匕首，一个背包。走遍了方圆百公里的废墟。

他形成了自己的生存法则：
- 不与人深交（失去的痛苦太难受）
- 不存多余的感情（只会拖累自己）
- 每天检查辐射检测仪（活着是唯一的任务）
- 晚上写日记（写给谁看？不知道，只是习惯了）

他以为自己会一直这样下去，直到辐射病发作，或者某次拾荒时运气不好。

**【人物转变的伏笔】**
但有些东西，他没意识到：

- 他会把多余的罐头放在明显的位置，给更弱的拾荒者
- 他救过一个被变异生物追的少年，给他指了避难所的方向
- 他的日记里，开始越来越多地提到"如果小宇还活着，现在应该10岁了"

他不是完全麻木了。只是他把感情封存在一个箱子里，锁起来，假装不存在。

**【遇见小光前的心理状态】**
故事开始时，阿尘38岁，独自生存7年。

他的身体：营养不良，慢性辐射病（偶尔咳血），但能跑能跳。

他的心理：表面麻木，深层是未愈合的创伤。他每天都在问"为什么是我活下来"，但从不允许自己哭。

他有一个秘密：每天睡前，他会对着黑暗说"晚安，婉婉。晚安，小宇。"就像他们还在身边。

**【遇见小光的意义】**
当他抱起小光的那一刻，所有的封印都崩开了。

不是因为小光像小宇（不像，小宇出生时胖嘟嘟的，小光瘦小），而是因为——

他终于可以做点什么了。

不是为了赎罪（"我没保护好他们"），不是为了补偿（"这次我要做好"），而是因为——

"这个孩子需要我。"

简单。直接。像一道光，刺进他封了7年的黑暗。

**【人物弧光的本质】**
阿尘的转变，不是从"坏人"变成"好人"。

他一直是个好人。即使在最黑暗的时候，他也没有放弃底线。

他的转变是：从"否认自己有感情"到"接受自己有感情"。

从"我不需要任何人"到"我愿意保护一个人"。

从"活着就是任务"到"活着有意义"。

结局时，他说"小光"，不是在叫婴儿的名字。他是在叫那道光——

那道穿透末世黑暗、照进他心里的光。

### 小光的身世之谜（推测性小传）

**【出生前】**
小光的父母是谁？我们从线索推测。

**父亲的线索：**
- 超市储藏室有男性工作服（电工制服）
- 货架上有手工制作的木质玩具（会木工）
- 墙壁上有用螺丝刀刻的字："坚持住"（笔迹有力）
- 地上有军用水壶（可能是退伍军人或户外爱好者）

推测：父亲可能是退伍军人，会电工和木工，性格坚毅。

**母亲的线索：**
- 襁褓缝制精细，针脚均匀（会缝纫）
- 角落有女性杂志（怀孕期看的）
- 有收集的野花标本（爱美、乐观）
- 婴儿被藏在最隐蔽的货架后（母亲的保护本能）

推测：母亲可能是教师或护士，性格温柔但坚强。

**【出生】**
小光出生在末世第6年（2027年左右）。

母亲在超市里用临时搭建的产床生下她。没有医生，没有药物，只有父亲和一套从废墟里找来的旧工具书。

父亲接生的。他后来对母亲说："她的手，一出生就抓住了我的手指。"

**【前6个月】**
超市成了他们的家。

父亲每天外出寻找物资和净化水。母亲照顾婴儿，用杂志上的知识做育儿笔记。

他们在货架上刻下身高线。第一条是满月时画的，第二条是百天时画的。

小光的名字可能是"小希"或"小安"——父母希望她能平安长大。

**【最后的时刻】**
末世第7年的某天，父亲外出后没有回来。

母亲抱着婴儿等了3天。第4天，她听到外面有变异生物的叫声，越来越近。

她做了决定：把婴儿藏在最隐蔽的货架后，用货架挡住，只留一个缝隙透气。

她在婴儿身边放了最后3罐奶粉、一个接雨水的容器、一张纸条。

纸条上写着："如果有人发现她，请照顾好她。她叫______（被血迹模糊）。谢谢。"

然后她走了出去，引开变异生物。

她没有回来。

**【独自存活的7天】**
小光独自在货架后存活了约7天。

她哭过，但很快学会了安静——可能是因为变异生物的声音让她知道，出声是危险的。

她喝完了接雨水的容器里的水（可能之前母亲每天换）。

她饿，但还能忍受——7个月的婴儿可以短暂承受饥饿。

她大部分时间都在睡觉。在梦中，她可能还记得母亲的体温。

**【被阿尘发现时的状态】**
当阿尘搬开货架时，小光没有哭。

她只是睁大眼睛，看着这个陌生人。

然后，她伸出手，像出生时抓住父亲的手指一样，抓住了阿尘的手指。

**【象征意义】**
小光不是"阿尘的救赎工具"。

她是一个真实存在过的生命——有父母、有过去、有未被讲述的故事。

她的存活是奇迹，但奇迹背后是父母的牺牲。

她的笑是对那个牺牲的最好回应："谢谢你们保护我，我会活下去。"

结局时，当她在避难所里笑，那不只是"天真无邪的笑"。

那是跨越了死亡、黑暗、孤独之后，对生命本身的庆祝。
