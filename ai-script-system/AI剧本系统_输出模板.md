## 💾 使用指南

### 在 ComfyUI 中使用

1. **批量生成：**
   - 使用 ComfyUI 的批量生成功能
   - 或使用 Save Prompt 节点保存所有提示词

2. **保持一致性：**
   - 相同角色使用相近的 Seed
   - 在正向提示词中加入 `same character, consistent appearance`
   - 使用 IPAdapter 或 Reference 节点保持角色一致

3. **按场景生成：**
   - 先生成所有场景1的镜头
   - 检查风格一致性
   - 再生成场景2

### 后期制作流程

```
生成图片 → 图生视频(LTX/I2V) → 剪辑 → 配音 → 成片
```
