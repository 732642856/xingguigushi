## 使用流程

### Step 1: 提供基本信息

我会询问您：
1. **主题**：想讲什么故事？
2. **类型**：短剧 / 电视剧 / 电影 / 宣传片？
3. **时长**：预计多长时间？

### Step 2: 选择运行模式

```
🎛️ 请选择运行模式：

1️⃣ 自动模式 - 系统自动完成所有步骤，直接输出最终结果
              （适合：需求明确、信任系统判断、追求效率）

2️⃣ 手动模式 - 每个步骤完成后暂停，等您确认后再继续
              （适合：精细打磨、需要调整、学习过程）
```

### Step 3: 等待结果

**自动模式：**
- 系统自动执行 5 个步骤
- 管理Agent在每个步骤进行质量核查
- 最终输出完整的项目文件

**手动模式：**
- 每个步骤完成后展示结果
- 您可以随时说"继续"或"修改：xxx"
- 完全可控的创作过程

---

## 运行中可随时使用的指令

| 指令 | 说明 |
|------|------|
| `继续` | 确认当前步骤，进入下一步（手动模式） |
| `修改：xxx` | 指出需要修改的地方，当前步骤重新生成 |
| `回到大纲` / `回到第1步` | 跳转到指定步骤 |
| `切换自动` | 切换到自动模式，剩余步骤自动执行 |
| `切换手动` | 切换到手动模式，剩余步骤逐个确认 |
| `保存` | 保存当前所有内容到文件 |
| `重新开始` | 清空所有，从头开始 |
| `帮助` | 显示可用的命令列表 |
