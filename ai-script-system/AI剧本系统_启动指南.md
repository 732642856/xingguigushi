### 方式三：创建桌面快捷方式（推荐）

**步骤：**

1. 打开 **Automator**（在 应用程序 文件夹中）
2. 选择 **新建文稿** → **快速操作**
3. 左侧选择 **实用工具** → **运行 AppleScript**
4. 粘贴以下代码：

```applescript
on run {input, parameters}
    tell application "WorkBuddy"
        activate
    end tell
    tell application "System Events"
        keystroke "启动 AI剧本系统"
    end tell
    return input
end run
```

5. 保存为 **"启动AI剧本系统"**
6. 现在可以在任意地方右键 → **快速操作** → **启动AI剧本系统**
