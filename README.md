## 🎯 关于星语

星语是一套基于中国传统术数文化的故事创作工具集，包含：
- 世宿情缘（二十八星宿）
- 八门谋生（奇门遁甲）
- 六壬探案（六壬神课）
- 军途（军事谋略）
- 情感戏（情感戏剧创作）
- 民俗（民俗文化）
- 星轨人生（紫微斗数角色设计）
