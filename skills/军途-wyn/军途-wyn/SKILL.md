# 军途-wyn

军事题材创作工具。以兵法体系为底层框架，整合古代/近代/现代军事元素，融合专业编剧方法论，提供从军事题材选择到完整战争故事的全流程创作支持。

---

## 零、格式路由（第一步）

在进入工作流之前，先确定用户的创作需求：

| 时代 | 核心元素 | 适用题材 |
|------|---------|---------|
| **古代军事** | 冷兵器、战阵、兵法、谋略 | 历史战争、史诗、侠义 |
| **近代军事** | 热兵器、一战/二战、民国军阀 | 战争史诗、谍战、传记 |
| **现代军事** | 信息化作战、特种部队、当代军旅 | 现代战争、军旅、特种作战 |

**选择方式**：
- 用户说"古代战争故事"→ 进入古代军事模块
- 用户说"二战/谍战"→ 进入近代军事模块
- 用户说"特种部队/当代军旅"→ 进入现代军事模块
- 用户说"穿越/混搭"→ 可以组合多个模块

以**孙子兵法**+**三十六计**为战略核心，映射到创作：

| 兵法概念 | 军事创作映射 | 戏剧功能 |
|---------|------------|---------|
| 知己知彼 | 情报战、心理战 | 建立信息差，制造悬念 |
| 兵贵神速 | 闪电战、突袭 | 制造紧张感 |
| 以逸待劳 | 防守反击、诱敌深入 | 积累情绪，等待爆发 |
| 声东击西 | 佯攻、欺骗作战 | 反转的关键 |
| 走为上计 | 战略性撤退 | 情节转折，人物成长 |
| 围魏救赵 | 围点打援 | 战术创新的高光时刻 |
| 暗度陈仓 | 隐蔽行动 | 伏笔揭示 |
| 反间计 | 间谍反水 | 背叛与信任 |

### 二、战争阶段结构（六阶段）

```
一、战前准备（建置）
  - 背景铺垫、势力铺排
  - 主角的"旧世界"——战前状态
  - 开场钩子：一个预示战争的细节

二、战略部署（发展前半）
  - 战前会议、情报分析
  - 战术制定、物资准备
  - 主角进入"战争世界"

三、战术执行（发展后半）
  - 首次战斗、小规模冲突
  - 试探对方实力
  - 情报更新，战术调整

四、战斗胶着（高潮前）
  - 主力决战、势力对决
  - 损失出现，代价显现
  - 核心冲突爆发

五、转折与决战（高潮）
  - 战术创新/援军到达/意外变量
  - 终极对决
  - 弧光完成——选择与牺牲

六、战后处理（结局）
  - 胜负揭晓
  - 代价与代价
  - 新世界/新常态
```

### 三、军事冲突类型库

| 冲突类型 | 描述 | 适用阶段 |
|---------|------|---------|
| 命令与执行 | 上级的命令vs现实的困难 | 任何阶段 |
| 个人与集体 | 小我的生存vs大战的胜利 | 发展-高潮 |
| 战术与战略 | 局部胜利vs全局利益 | 胶着-转折 |
| 正义与效率 | 人道主义vs战争效率 | 任何阶段 |
| 信任与背叛 | 间谍、叛变、内部矛盾 | 任何阶段 |
| 荣誉与生存 | 战死vs撤退 | 高潮-结局 |

## 铁律（优先级最高）

### 写作红线

**绝对不要**：
- 写心理描写（"他害怕了"、"她内心纠结"）
- 写括号暗示（"（其实他不想死）"）
- 角色用台词解释战术或战略
- "神剧"套路：一个人打败一个连、手撕鬼子
- 说教式战争总结
- 任何无法被摄影机拍到的内容

**必须做到**：
- 战争通过**行动**展示——不说"他很勇敢"，写他在做什么
- 战术通过**场景和细节**展示——不解释"他在布阵"，写布阵的过程
- 台词有军事辨识度——不同兵种、不同军衔说话方式不同
- 死亡有代价——不是随便死一个配角
- 用**信息差**制造戏剧张力

### 工作流纪律

- 严格按步骤顺序执行，不跳步
- 每一步完成后暂停，等待用户的 [通过 / 修改 / 自检] 指令
- 每步结束后提醒用户可以要求 [自检]

## 二、军事故事七步工作流

### 步骤一：军事题材与破题

**目标**：确定时代背景 + 建立军事戏剧动作

**操作流程**：
1. 确定时代背景（古代/近代/现代）
2. 确定军事类型（战略/战术/战争/军旅）
3. 确定核心冲突（个人vs集体/正义vs效率/生存vs使命）
4. 建立军事戏剧动作

**军事戏剧动作设计原则**：
- 目标：胜利/生存/完成任务/保家卫国
- 阻碍：敌方实力/内部矛盾/资源不足/道德困境
- 好的军事戏剧动作：目标清晰，障碍多层（敌方→内部→自身→意外）

**输出格式**：

### 步骤二：军事人物深度与弧光

**目标**：建立军事角色的完整内在张力

**军事角色的特殊维度**：

军事角色的核心矛盾：**"军人职责vs人性本能"** 和 **"战争逻辑vs道德逻辑"**

```markdown
## 角色：XXX

- 身份：[士兵/军官/特种兵/平民]

- 外在表现：[表面给人什么感觉]
- 内在真实：[真实的恐惧/渴望]

### 戏剧维度
- Want（外在需求）：胜利/生存/完成任务/回家
- Need（内在欲望）：战争的意义/人性/救赎
- Ghost（前史创伤）：什么经历让这个人上了战场
- Lie（角色相信的谎言）："战争是光荣的"/"我能活着回去"/"命令必须服从"
- Flaw（性格缺陷）：冲动/冷血/过度服从/ PTSD

### 弧光
- 起点：[A状态——战前状态]
- 伪变化：[获得荣誉但失去人性？]
- 终点：[B状态——战后状态]
- 弧光类型：
  - 成长弧光：从"害怕战争"到"承担战争"
  - 悲剧弧光：从"荣誉"到"创伤"
  - 反弧光：从"战争英雄"到"战争机器"

### 军事特质
- 作战风格：[进攻/防守/谋略]
- 说话方式：[军事术语/沉默/简洁]
- 破防时刻：[什么情况下会崩溃]
```

### 步骤四：战役结构大纲

**目标**：搭建军事故事骨架

**结构选型指南**：

| 军事类型 | 推荐结构 | 原因 |
|---------|---------|------|
| 战略型 | **战争阶段式** | 天然对应战争的多阶段递进 |
| 战术型 | **Save the Cat / Story Circle** | 标准商业片节奏 |
| 军旅型 | **内在节拍表** | 角色成长驱动 |
| 史诗型 | **McKee 故事原理** | 多线叙事，宏大视角 |
| 爽文型 | **类型杂交** | 前半紧张，后半爽感 |

**军事战争六阶段结构**：

```
一、战前准备（建置）
  角色旧世界——战前状态、势力铺排
  开场钩子：一个预示战争的细节（征兵令、战报、告别）
  同时展示角色的军事能力
  双轨节奏：情节松+情感中（表面平静）

二、战略部署（发展前半）
  主角进入战争世界
  第一次战斗——小规模冲突
  双轨节奏：情节中+情感轻（上升期）

三、战术执行（发展后半）
  主力参战，冲突升级
  情报更新，战术调整
  双轨节奏：情节紧+情感中

四、战斗胶着（高潮前）
  核心冲突爆发
  损失出现，代价显现
  角色面临生死抉择
  双轨节奏：情节紧+情感重

五、转折与决战（高潮）
  战术创新/援军到达/意外变量
  终极对决
  弧光完成——选择与牺牲
  双轨节奏：情节爆发+情感爆发（不能太久）

六、战后处理（结局）
  胜负揭晓
  代价与抚恤
  新世界/新常态——与开场对照
  双轨节奏：情节松+情感重
```

**Subplot交织**：

| Subplot | 军事功能 | 交织规则 |
|---------|---------|---------|
| **A Plot（主线）** | 军事行动——战役胜负 | 与B Plot在胶着期交汇 |
| **B Plot（情感线）** | 战友情/爱情/家庭，承载主题 | 在胶着期和死门期给予/收回 |
| **C Plot（暗线）** | 真正的幕后棋局 | 在高潮期揭示 |

### 步骤五：战斗场景拆解

**军事场景标注格式**：

**军事场景特殊技法**：

- **信息差场景**：观众知道主角不知道的情报——"敌方有埋伏！"

### 步骤六：场景写作

**军事写作规则**：

- 战争通过**行动**展示，不通过**解释**
- 战术通过**细节**展示——写布阵的过程，写瞄准的动作
- 台词有军事辨识度——不同兵种说话方式不同
- 死亡有代价——每个死亡都有意义
- **军事幽默**（如适用）："灾难越大，语气越平"——战争中的日常

**潜台词对话示例**：

**直白（错误）**：
> "我们已经损失了三分之一的兵力，必须撤退。"

**潜台词（正确）**：
> "三排还剩几个？"
> "...够了。"
> "我问的是还剩几个！"
> "报告连长，够了。"

### 步骤七：剧本医生

**军事故事自检清单**：

#### 自检一：核心戏剧动作
- [ ] 是否建立了清晰的军事目标+阻碍？
- [ ] 战争赌注是否足够高？
- [ ] 信息差设计是否有效？

#### 自检二：人物弧光
- [ ] Want和Need是否清晰且对立？
- [ ] Ghost→Lie→Flaw链条是否完整？
- [ ] 弧光是否是不可逆的变化？
- [ ] 是否有至少一次"伪变化"？
- [ ] 军事角色的特殊维度（职责vs人性）是否体现？

#### 自检三：结构与节奏
- [ ] 六阶段是否清晰推进？
- [ ] 开场钩子是否有效？
- [ ] Subplot是否有效交织？
- [ ] 双轨节奏是否有张弛对比？

#### 自检四：军事逻辑
- [ ] 战术是否有因果关系？（不是巧合）
- [ ] 军事细节是否合理？（不能有"神剧"bug）
- [ ] 伤亡代价是否有体现？

#### 自检五：写作质量
- [ ] 是否有心理描写？（如有则Fail）
- [ ] 对话是否过于直白？
- [ ] 战争是否通过行动展示而非解释？
- [ ] 不同角色的台词风格是否有区分？

## 三、军事节奏管理（剧集专用）

### 集间情绪基调类型库

| 基调 | 情节特征 | 适用阶段 |
|------|---------|---------|
| **备战型** | 战前准备、训练 | 前期 |
| **开拔型** | 出发、向战场移动 | 前期 |
| **遭遇型** | 小规模冲突、试探 | 中前 |
| **对峙型** | 持续战斗、僵持 | 中段 |
| **焦灼型** | 损失加大、信心动摇 | 中后 |
| **转折型** | 意外变量、反转 | 转折 |
| **决战型** | 终极对决 | 高潮 |
| **胜利型** | 胜利/失败结果 | 结局 |
| **伤亡型** | 代价、抚恤 | 结局 |
| **和平型** | 战后重建、反思 | 结局 |

### 节奏波浪原则

- 不能连续3集以上同一基调
- 紧张之后必须有呼吸
- 胜利/失败不能太久——观众需要喘气

### 双轨节奏在军事故事中的应用

1. **情节紧+情感轻**：战斗进行中——紧张但理性
2. **情节松+情感重**：战斗后的安静——伤亡、反思
3. **情节紧+情感重**：决战——高潮，不能太久
4. **情节松+情感轻**：战前/和平时期——呼吸
