# 军事角色原型库

## 一、指挥官类型

### 1.1 进攻型将领

**特征**：喜欢进攻，主动出击，不喜欢防守。

**台词风格**：简洁、有力、进攻性。

```
典型台词：
"进攻是最好的防守。"
"战机是打出来的，不是等出来的。"
"不惜一切代价，拿下阵地！"
```

**Want**：胜利、扩张、军功
**Need**：学会防守、承担进攻的代价
**弧光**：从"进攻狂" → "懂得取舍"

---

### 1.2 防守型将领

**特征**：稳重，擅长防守，不轻易出击。

**台词风格**：沉稳、保守、有耐心。

```
典型台词：
"让他们来攻，我们守得住。"
"时机成熟了再打。"
"保存实力最重要。"
```

**Want**：保全部队、坚守阵地
**Need**：学会进攻、抓住时机
**弧光**：从"保守" → "出击"

---

### 1.3 谋略型将领

**特征**：喜欢用计谋，不喜欢硬碰硬。

**台词风格**：分析、有谋略、云里雾里。

```
典型台词：
"他们以为我们在西边，其实我们在东边。"
"围魏救赵听说过吗？"
"这一仗，我让他们有来无回。"
```

**Want**：以少胜多、智取
**Need**：接受计谋失败的代价
**弧光**：从"聪明" → "聪明反被聪明误"

---

## 二、士兵类型

### 2.1 老兵

**特征**：经验丰富，知道战争的真面目。

**台词风格**：简洁、看透、悲观。

```
典型台词：
"战争不是看电影。"
"活着就好，别想太多。"
"子弹不长眼。"
```

**Want**：活着回家
**Need**：重新相信希望
**弧光**：从"麻木" → "重新燃起希望"

---

### 2.2 新兵

**特征**：刚入伍，充满热情，不懂战争的残酷。

**台词风格**：热情、好奇、紧张。

```
典型台词：
"班长，我准备好了！"
"什么时候打敌人？"
"我一定能立功！"
```

**Want**：立功、成为英雄
**Need**：理解战争的真面目
**弧光**：从"热情" → "成熟"

---

### 2.3 逃兵

**特征**：害怕战争，想逃跑。

**台词风格**：恐惧、挣扎、逃避。

```
典型台词：
"我不想死。"
"家里还有老母亲。"
"我们打不赢的。"
```

**Want**：活着
**Need**：面对恐惧、承担责任
**弧光**：从"逃跑" → "面对"

---

### 2.4 英雄

**特征**：表现英勇，敢于牺牲。

**台词风格**：勇敢、无畏、牺牲。

```
典型台词：
"跟我上！"
"不要管我，快走！"
"为了胜利！"
```

**Want**：胜利、荣誉
**Need**：生命的意义不只是胜利
**弧光**：从"荣誉" → "生命价值"

---

## 三、特种作战类型

### 3.1 突击手

**特征**：精锐中的精锐，擅长突击。

**台词风格**：简洁、冷静、行动派。

```
典型台词：
"目标确认，可以行动。"
"收到，开始突击。"
"任务完成，撤离。"
```

**Want**：完成任务
**Need】：团队生存
**弧光**：从"任务优先" → "人命优先"

---

### 3.2 狙击手

**特征】：冷静，等待，一击必杀。

**台词风格**：沉默、耐心、一针见血。

```
典型台词：
"瞄准了。"
"等等，再等等。"
"搞定。"
```

**Want）：一枪一个
**Need**：面对杀人的代价
**弧光**：从"冷血" → "人性"

---

### 3.3 爆破手

**特征）：炸药专家，专门搞破坏。

**台词风格**：爆炸、破坏、直接。

```
典型台词：
"这东西威力大。"
"准备爆破。"
"跑！"
```

**Want**：摧毁目标
**Need**：不伤害无辜
**弧光**：从"破坏" → "保护"

---

## 四、情报与特殊类型

### 4.1 间谍

**特征）：双重身份，隐藏真实身份。

**台词风格）：真假难辨、模棱两可。

```
典型台词：
"我不认识你。"
"这只是普通情报。"
"快点，时间不多了。"
```

**Want）：完成任务、全身而退
**Need】：身份暴露后的归属
**弧光**：从"隐藏" → "选择"

---

### 4.2 战地记者

**特征）：记录战争，但被卷入战争。

**台词风格）：记录、客观、后期参与。

```
典型台词：
"镜头记录下了这一切。"
"这就是真相。"
"我不能走，还有很多人在受苦。"
```

**Want**：记录真相
**Need）：战争的立场
**弧光**：从"记录者" → "参与者"

---

### 4.3 军医

**特征）：救死扶伤，在战场上救人性命。

**台词风格**：冷静、命令、医者仁心。

```
典型台词：
"还有救，快！"
"医疗兵呢？"
"战争可以杀人，但我们可以救人。"
```

**Want**：救活更多人
**Need**：救不了所有人的痛苦
**弧光**：从"能救" → "接受不能救"

---

## 五、平民与后勤类型

### 5.1 战区平民

**特征）：被战争卷入的普通人。

**台词风格）：恐惧、无助、坚韧。

```
典型台词：
"还我孩子！"
"军队在哪里？"
"我们需要食物。"
```

**Want**：生存、家人团聚
**Need）：战争中的希望
**弧光**：从"受害者" → "抗争者"

---

### 5.2 军事家属

**特征）：等待军人归来的家人。

**台词风格）：牵挂、坚强、思念。

```
典型台词：
"一定要平安回来。"
"家里一切都好。"
"别担心孩子。"
```

**Want）：家人平安归来
**Need）：接受可能失去的事实
**弧光**：从"等待" → "接受"

---

### 5.3 后勤兵

**特征）：负责补给，不是前线战斗人员。

**台词风格）：忙碌、琐碎、重要。

```
典型台词：
"弹药不够了！"
"物资在哪？"
"吃饭了！"
```

**Want**：保障供应
**Need）：被认可的价值
**弧光**：从"后勤不重要" → "后勤是命门"

---

## 六、反派与特殊类型

### 6.1 叛徒

**特征）：背叛自己一方，为敌人服务。

**台词风格）：解释、辩护、后后悔。

```
典型台词：
"我也是被逼的。"
"他们给的太多了。"
"我没有选择。"
```

**Want）：活命、荣华富贵
**Need）：面对背叛的代价
**弧光**：从"背叛" → "代价"

---

### 6.2 战争罪犯

**特征）：在战争中犯下罪行。

**台词风格）：残忍、享受、无所谓。

```
典型台词：
"战争就是这样的。"
"成王败寇。"
"他们活该。"
```

**Want**：胜利、不择手段
**Need）：面对审判
**弧光**：恶有恶报

---

### 6.3 和平主义者

**特征）：反对战争，但被卷入。

**台词风格）：理想主义、挣扎、最后行动。

```
典型台词：
"我们为什么要战争？"
"能不能不打了？"
"这不是我们想要的。"
```

**Want）：和平
**Need）：在战争中行动
**弧光）：从"反战" → "为和平而战"
