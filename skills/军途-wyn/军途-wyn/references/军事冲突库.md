# 军事冲突库

## 一、命令与执行冲突

### 1.1 不合理命令
```
冲突：上级命令明显送死
场景：指挥官下令进攻明显守不住的阵地
人物：指挥官 vs 执行者
解决：执行/抗命/想办法变通
```

### 1.2 命令被隐瞒
```
冲突：关键命令被隐瞒
场景：部分人知道战略意图，部分人被蒙在鼓里
人物：知情者 vs 不知情者
解决：暴露/理解/牺牲
```

### 1.3 命令冲突
```
冲突：两个上级的命令矛盾
场景：A上级让进攻，B上级让撤退
人物：执行者 vs 两个上级
解决：选择一边/汇报/自行判断
```

### 1.4 命令延迟
```
冲突：命令传达延迟导致战机丧失
场景：进攻命令到达时已经太晚
人物：发令者 vs 接收者 vs 战场
解决：追责/理解/补救
```
