# 战斗场景模板库

## 一、古代战斗场景

### 1.1 战阵对决型
```
场景：两军列阵，正面交锋
元素：战鼓、旗帜、冲锋、战阵
时长度：5-15分钟
节奏：情节紧+情感轻→重
古代兵种：步兵+骑兵+弓兵配合
```

### 1.2 攻城战型
```
场景：攻城方vs守城方
元素：云梯、投石机、城门、巷战
时长度：可拉长（多集）
节奏：情节紧（持续）+情感重
关键：攻城伤亡vs守城伤亡
```

### 1.3 夜袭营型
```
场景：夜间偷袭敌军营地
元素：隐蔽、潜入、放火、突袭
时长度：3-10分钟
节奏：情节紧+情感中
关键：被发现的风险
```

### 1.4 单挑型
```
场景：两军阵前武将单挑
元素：武艺、兵器、勇气
时长度：2-5分钟
节奏：情节紧+情感中
关键：单挑结果影响军心
```

### 1.5 水战型
```
场景：水上作战
元素：战船、火攻、接舷战
时长度：5-15分钟
节奏：情节紧+情感重
关键：风向、水流
```

### 1.6 埋伏型
```
场景：伏兵四起
元素：诱饵、包围、歼灭
时长度：3-10分钟
节奏：情节紧（爆发）+情感重
关键：诱饵的牺牲
```

## 二、近代战斗场景

### 2.1 堑壕战型
```
场景：一战式堑壕对峙
元素：射击、弹坑、毒气、冲锋
时长度：可拉长
节奏：情节紧（持续）+情感重
关键：消耗战、伤亡惨重
```

### 2.2 闪电战型
```
场景：装甲部队快速突袭
元素：坦克、空降、包围
时长度：3-10分钟
节奏：情节紧+情感轻
关键：速度vs抵抗
```

### 2.3 巷战型
```
场景：城市街道争夺
元素：街垒、狙击、建筑物
时长度：可拉长
节奏：情节紧+情感重
关键：每个建筑物的争夺
```

### 2.4 登陆战型
```
场景：两栖登陆作战
元素：登陆艇、海滩、碉堡
时长度：5-15分钟
节奏：情节紧+情感重
关键：登陆地带的伤亡
```

### 2.5 空袭型
```
场景：轰炸机编队空袭
元素：轰炸、拦截、地面烟火
时长度：3-10分钟
节奏：情节紧+情感轻（轰炸方）/重（被炸方）
关键：平民伤亡
```

### 2.6 海战型
```
场景：海面舰艇对决
元素：炮击、鱼雷、潜艇
时长度：5-15分钟
节奏：情节紧+情感重
关键：击沉vs被击沉
```

### 2.7 谍战型
```
场景：情报人员行动
元素：暗号、接头、跟踪、营救
时长度：3-10分钟
节奏：情节紧（暗涌）+情感重
关键：身份暴露
```

## 三、现代战斗场景

```
元素：渗透、潜伏、突击、撤离
时长度：3-10分钟
节奏：情节紧+情感轻
关键：行动隐蔽性
```

```
场景：恐怖分子占据大楼
元素：人质、狙击、突击、谈判
时长度：5-15分钟
节奏：情节紧+情感重
关键：人质安全
```

### 3.3 军演对抗型
```
场景：红蓝对抗演习
元素：战术对抗、信息战、心理战
时长度：可拉长
节奏：情节中+情感中
关键：输赢vs学到
```

```
场景：边境对峙/小规模冲突
```

### 3.5 电子战型

### 3.6 维和行动型

```
场景：联合国维和任务

时长度：可拉长

```

### 3.7 撤侨行动型
```
场景：海外撤侨
元素：使馆、机场、海港
时长度：3-10分钟
节奏：情节紧+情感重
关键：带同胞回家
```

### 3.8 救灾行动型
```
场景：自然灾害救援
元素：搜救、医疗、运送
时长度：可拉长
节奏：情节中+情感重
```

## 四、通用战斗场景模板

### 4.1 战前动员型
```
场景：战斗前的动员讲话
元素：命令、激励、决心
时长度：1-3分钟
节奏：情节松+情感重
关键：提升士气
```

### 4.2 战前侦察型
```
场景：战前侦察敌情
元素：探查、回报、制定计划
时长度：2-5分钟
节奏：情节松+情感中
关键：情报准确性
```

### 4.3 行军型
```
场景：部队向战场移动
元素：行军、休息、赶路
时长度：可长可短
节奏：情节松+情感轻
关键：行军中的小事
```

### 4.4 战前会议型
```
场景：战术讨论/命令下达
元素：地图、分析、部署
时长度：2-5分钟
节奏：情节松+情感轻
关键：战术制定
```

### 4.5 战场等待型
```
场景：等待进攻命令
元素：紧张、沉默、准备
时长度：1-3分钟
节奏：情节松+情感重（紧张）
关键：等待的压迫感
```

### 4.6 第一枪型
```
场景：战斗打响第一枪
元素：开火、冲锋、倒下
时长度：1-2分钟
节奏：情节紧（爆发）+情感重
关键：第一枪的意义
```

### 4.7 战场救护型
```
场景：战场上的急救
元素：包扎、搬运、死亡
时长度：2-5分钟
节奏：情节中+情感重
关键：救不救得活
```

### 4.8 战场通讯型
```
场景：战场通讯/求援
元素：电台、信号、等待
时长度：1-3分钟
节奏：情节紧+情感重
关键：通讯中断
```

### 4.9 战场对峙型
```
场景：双方僵持不下
元素：射击、等待、消耗
时长度：可拉长
节奏：情节紧（持续）+情感中
关键：谁先忍不住
```

### 4.10 战场溃败型
```
场景：一方溃败
元素：逃跑、追击、投降
时长度：3-10分钟
节奏：情节紧（爆发）+情感重
关键：溃败的原因
```

### 4.11 战场胜利型
```
场景：战斗胜利
元素：欢呼、表彰、庆祝
时长度：2-5分钟
节奏：情节松+情感重→轻
关键：胜利的代价
```

### 4.12 战场失败型
```
场景：战斗失败
元素：沉默、伤亡、反思
时长度：2-5分钟
节奏：情节松+情感重
关键：失败的原因
```

### 4.13 战后清理型
```
场景：战场清理
元素：收尸、统计、伤员
时长度：2-5分钟
节奏：情节松+情感重
关键：代价的具象化
```

### 4.14 战时慰问型
```
场景：后方慰问
元素：食物、表演、慰问品
时长度：1-3分钟
节奏：情节松+情感轻
关键：对比前线
```

### 4.15 战时家书型
```
场景：写家书/收到家书
元素：思念、隐瞒、谎言
时长度：1-3分钟
节奏：情节松+情感重
```

```
节奏：情节紧+情感重
```

```
关键：能量管理
```

```
时长度：3-10分钟
节奏：情节紧（暗涌）+情感中
```
