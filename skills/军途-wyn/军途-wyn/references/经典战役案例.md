## 一、古代经典战役

### 1.1 巨鹿之战（秦末）
```
时间：公元前207年
双方：项羽楚军 vs 秦军
兵力：楚军5万 vs 秦军40万
结果：楚军大胜，秦军主力尽毁
奇门映射：破军（破坏）+ 勇敢（项羽）
可提取情节：破釜沉舟、以少胜多、将领魅力
```

### 1.2 官渡之战（三国）
```
时间：200年
双方：曹操 vs 袁绍
兵力：曹操2万 vs 袁绍10万
结果：曹操大胜，统一北方
奇门映射：智谋（曹操）+ 情报（许攸）
可提取情节：夜袭乌巢、情报战、曹操的决断
```

### 1.3 淝水之战（东晋）
```
时间：383年
双方：东晋8万 vs 前秦80万
结果：东晋大胜，前秦灭亡
奇门映射：计谋（谢安）+ 心理战
可提取情节：心理战瓦解、苻坚的失败、风声鹤唳
```

### 1.4 赤壁之战（三国）
```
时间：208年
双方：孙刘联军5万 vs 曹操20万
结果：孙刘大胜，曹操败退
奇门映射：火攻（周瑜）+ 东风（天时）
可提取情节：连环计、苦肉计、借东风
```

### 1.5 岳飞抗金（南宋）
```
时间：1130年后
双方：岳家军 vs 金军
特点：精忠报国、战术出色
奇门映射：忠诚（岳飞）+ 训练
可提取情节：郾城大捷、十二道金牌、莫须有
```
