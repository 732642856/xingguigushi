# 《渡者无归》第四幕 · 轮回乐园

## 节点一：过渡 · 从崩溃缝隙坠入

### Node_X_奈何桥不是桥

**触发条件**：从任意压抑路线的**系统崩溃瞬间**坠入——
- 汉骨层的石壁裂缝中渗出金色液体
- 歌姬剧场的横梁断裂后露出纸扎结构
- 内化阿缨后胸腔暖白光穿透灵体肋骨

**场景设定**：黄昏与黎明之间的金色时刻，宣纸河流向上流淌。

**视觉资产**：
- `BG_X` (1920×1080): `A river made of yellowed rice paper flowing upward in golden hour light, warm chrysanthemum-colored water, glowing pebbles of shed memories visible on the riverbed, upside-down sky of slowly rotating sky-lanterns connected by threads to translucent ghosts standing on the banks, Song dynasty pavilions made of stacked wish-papers lining the shores, absurdly warm and impossible, Eastern Alice in Wonderland, fluid painterly animation, warm golden chiaroscuro, 4K, 16:9`

**音频反转**：
- 基础层：颂钵60Hz保留，但叠加**竹笛与二胡的温暖应答**
- Binaural Beats从60/63Hz转为**432Hz自然和谐音**
- 新层：零散的、带着笑声的拍手，皮影戏锣鼓，糖人吹气的嘶嘶声

**剧本文本**：
- **系统残留（最后一次冰冷播报）**："错误。盲区数据溢出。渡者...不在服务区。"
- **主角（困惑）**："这里不黑。这里...在笑？"
- **阿缨（内化态，第一次以非控诉的语气）**："你听到的是祝福的声音。不是给你的。是给那些...已经放下了的。"
- **新NPC（纸船上的撑篙老人）**："小伙子，你攥了两千年的拳头，要不要先松开？糖人要化了。"

**交互热区（过渡语法：从“审判”到“探索”）**：

| 热区 | 位置 | 动作 | 变量变化 | 跳转 |
|---|---|---|---|---|
| 松开拳头（主角自身） | `bottom:20%; left:40%; width:20%; height:30%` | 主角五指缓缓张开，靛蓝灵质从指缝流出，滴落河面变成金色 | Humanity+1, System-2, 首次进入乐园 | Node_Y1_糖人铺 |
| 抓向灯笼（天空） | `top:10%; left:30%; width:40%; height:30%` | 主角伸手触碰一盏河灯，灯上的祝福涌入视网膜 | 获得首件"祈愿包裹" | Node_Y1_糖人铺（携带包裹） |
| 沉入河底（河流） | `top:40%; left:20%; width:60%; height:40%` | 主角被宣纸河流温柔地卷入水下，看到河底发光的记忆鹅卵石 | 解锁隐藏"遗忘种子" | Node_Y3_祈愿花园（直达） |

**技术要点**：
- CSS动画：`@keyframes float-upward` 让宣纸河流产生向上流动的错觉
- 音频过渡：`gainNode.gain.linearRampToValueAtTime(0, audioCtx.currentTime + 3)` 让低频颂钵在3秒内淡出，同时竹笛在3秒内淡入
- 颜色过渡：画面从`filter: grayscale(70%)`在8秒内过渡到`grayscale(0%)`，模拟从地府到人间的视觉适应

## 节点二：忘川市集 · 糖人铺

### Node_Y1_糖人铺：前世的甜

**场景设定**：市集第一摊位。老爷爷用**忘川水+记忆粉末**吹糖人。每一个糖人都是亡魂前世的模样。

**视觉资产**：
- `BG_Y1` (1920×1080): `A Chinese afterlife market stall on floating paper boats, warm chrysanthemum-colored river flowing upward, an elderly man blowing sugar figurines from glowing memory-powder, translucent customers waiting with paper lanterns, qing dynasty vermillion and malachite green palette, soft tissue-paper texture, warm absurd and tender, Eastern Alice in Wonderland, fluid painterly animation, warm golden chiaroscuro, 4K, 16:9`
- `T_Y1_糖人阿缨` (1024×1024 PNG透明): `A translucent sugar figurine of a young Chinese woman in Song dynasty robes, purple neck ligature visible as a faint crack in the sugar, warm amber glow from within, delicate and fragile, transparent background`

**音频设计**：
- 糖人吹制声：嘶嘶的气流声 + 糖壳凝固时的**清脆裂响**（像冰块入热茶）
- 市集氛围：远处皮影戏的锣鼓、近处算盘珠子的拨动（但算的不是钱，是**祝福的数量**）
- 爷爷的嗓音：沙哑但温暖，带着**永远无法完成的慈爱**

**剧本文本**：

**老爷爷**：
> "每一个来我这儿的，都想要一个'自己'。可糖人太甜了，前世太苦了，混在一起...就化得快。"
> *（他看向主角）*
> "你不一样。你身上有两个味。一个是墨——判官的墨。一个是糖... somebody else's sugar."
> *（指向主角胸腔）*
> "要我吹一个她吗？吹一个...住在你肺里的那位？"

**阿缨（内化态，第一次以"味道"出现）**：
> *（不是声音，是主角舌尖上突然涌现的味觉——苦、涩、然后是一丝回甘）*
> "你尝到了吗？那是景和二年霜降的味道。城门外的风，混着糖炒栗子的烟。你当时说...说回来就娶我。那句话是甜的。后来的两千卷宗...是苦的。"

**老爷爷的糖人吹制过程（交互核心）**：

这不是点击选择，而是一个**三段式手势交互**：

1. **选色**：观众在三种颜色的糖锅前悬停——
   - 赭红色（阿缨的城楼/血/记忆）
   - 靛青色（主角的判官墨/愧疚）
   - 藤黄色（霜降的糖炒栗子/未完成的承诺）
   
2. **塑形**：老爷爷开始吹制。观众需要**按住屏幕/鼠标**模拟"吹气"的持续时间——
   - 吹得太短（<2秒）：糖人不成形，是一团模糊的糖块。老爷爷说："太快了。来不及看清她。"
   - 吹得正好（2-4秒）：糖人成形，但会有一道**天生的裂缝**（对应阿缨的勒痕/玉佩的断口）。老爷爷说："完美都是有缝的。缝里面才有光。"
   - 吹得太长（>4秒）：糖人吹破了，糖壳碎裂。老爷爷说："太用力了。想把两千年吹进一口气里，会炸的。"

3. **品尝**：糖人完成后，观众需要**点击糖人"吃"一口**。
   - 第一口：尝到的是**阿缨的甜味**——系统第一次允许主角"摄入"另一个灵魂的味道。
   - 第二口：尝到的是**自己的墨苦味**——愧疚感。
   - 第三口：两种味道混在一起，变成一种**新的、从未有过的味道**。阿缨的声音从主角的味蕾上传到耳蜗："这是...我们的味道。不是她的。不是他的。是'我们'的。"

**交互热区**：

| 热区 | 位置 | 动作 | 变量变化 | 结果 |
|---|---|---|---|---|
| 赭红色糖锅 | `left:20%; top:40%; width:15%; height:20%` | 选择阿缨的记忆色 | Aying_Memory_Color="ochre" | 糖人呈暖红色 |
| 靛青色糖锅 | `left:42%; top:40%; width:15%; height:20%` | 选择主角的愧疚色 | Protagonist_Memory_Color="indigo" | 糖人呈冷青色 |
| 藤黄色糖锅 | `left:65%; top:40%; width:15%; height:20%` | 选择承诺的甜蜜色 | Promise_Color="藤黄" | 糖人呈暖金色 |
| 糖人主体 | `center:50%; top:55%; width:12%; height:18%` | 点击"品尝" | Taste_Memory=true | 触发味觉叙事 |
| 老爷爷的手 | `left:35%; top:50%; width:10%; height:15%` | 悬停查看他的手纹 | 发现手纹是无数织了一半的毛衣针脚 | 解锁"织补"伏笔 |

**味觉音频（Web Audio API）**：
```javascript
function playTasteSound(flavor) {
  const ctx = new AudioContext();
  if (flavor === 'sweet') {
    // 甜味 = 高频 bell + 温暖谐波
    const osc = ctx.createOscillator();
    osc.frequency.value = 880; // A5
    osc.type = 'sine';
    const gain = ctx.createGain();
    gain.gain.setValueAtTime(0.15, ctx.currentTime);
    gain.gain.exponentialRampToValueAtTime(0.001, ctx.currentTime + 3);
    osc.connect(gain).connect(ctx.destination);
    osc.start();
  } else if (flavor === 'bitter') {
    // 苦味 = 低频 grind + 不和谐音程
    const osc = ctx.createOscillator();
    osc.frequency.value = 110; // A2
    osc.type = 'sawtooth';
    const filter = ctx.createBiquadFilter();
    filter.type = 'lowpass';
    filter.frequency.value = 400;
    const gain = ctx.createGain();
    gain.gain.setValueAtTime(0.1, ctx.currentTime);
    gain.gain.exponentialRampToValueAtTime(0.001, ctx.currentTime + 2);
    osc.connect(filter).connect(gain).connect(ctx.destination);
    osc.start();
  } else if (flavor === 'fusion') {
    // 融合味 = 两个频率的和声，从冲突到和谐
    const osc1 = ctx.createOscillator(); osc1.frequency.value = 440;
    const osc2 = ctx.createOscillator(); osc2.frequency.value = 523; // C5，与A4形成小三度（从紧张到温暖）
    const gain = ctx.createGain();
    gain.gain.setValueAtTime(0.1, ctx.currentTime);
    // 2秒后频率滑向和谐
    osc2.frequency.linearRampToValueAtTime(554, ctx.currentTime + 2); // C#5，形成大三度
    gain.gain.exponentialRampToValueAtTime(0.001, ctx.currentTime + 4);
    osc1.connect(gain); osc2.connect(gain);
    gain.connect(ctx.destination);
    osc1.start(); osc2.start();
  }
}
```

## 节点三：忘川市集 · 遗憾风筝

### Node_Y2_遗憾风筝：不剪断的线

**场景设定**：市集第二区域。天空不是天空，是**由无数风筝组成的、永远不会散去的云层**。每一只风筝下面都系着一根线，线的末端系在地面上的一个小木桩上——每个亡魂都可以把自己的遗憾系在这里。

**核心规则**：**风筝不飞走，是因为遗憾不需要被消灭，只需要被看见。**

**视觉资产**：
- `BG_Y2` (1920×1080): `A Chinese afterlife kite field, thousands of paper kites covering the upside-down sky like clouds, each kite bearing calligraphy wishes, threads connecting to small wooden stakes on the ground, translucent ghosts sitting beneath their kites looking up, warm golden hour light filtering through rice-paper wings, qing dynasty vermillion and soft yellow palette, absurdly tender, Eastern Alice in Wonderland, fluid painterly animation, warm chiaroscuro, 4K, 16:9`
- `T_Y2_霜降风筝` (1024×1024 PNG透明): `An ancient Chinese paper kite with wooden frame, calligraphy reading "景和二年霜降" on its wing, thread trailing below, warm amber glow, delicate rice-paper texture, transparent background`

**音频设计**：
- 风筝翅膀在风中的**纸张摩擦声**——不是尖锐的，是**温柔的、像翻书页一样的沙沙声**
- 线的**轻微颤动音**——像竖琴的最低一根弦被微风拨动
- 亡魂们的**低语**：不是对话，是各自对着自己的风筝喃喃自语，形成一种**无意识的多声部合唱**

**剧本文本**：

**风筝管理员（一个永远14岁的女孩，死于永远没寄出的情书）**：
> "这些风筝，飞不走的。我们也不想它们飞走。飞走了，就等于说'我不在意了'。可我们在意。我们只是...不疼了而已。"
> *（她指向主角）*
> "你有风筝吗？不是那种会飞很高的。是那种...系在城楼柱子上的？"

**主角找到"霜降风筝"（交互核心）**：

主角在无数风筝中，通过**滑动/拖拽画面**寻找。画面会随着拖动而**微微倾斜**（模仿风筝视角的摇晃感）。当主角的手指/鼠标悬停在某只风筝上时，那只风筝的翅膀会**微微亮起**，并传来**那只风筝主人的最后一声叹息**。

最终，主角找到一只**靛青色风筝**——那是他自己的。

风筝上的字不是"景和二年霜降"，而是：
> **"待北境平，以此为聘。——持衡，景和二年霜降。"**

线的末端不是系在木桩上，是**系在一块城楼的砖上**。那块砖上面有一道**勒痕**——和阿缨脖子上的完全吻合。

**交互设计**：

| 热区 | 位置 | 动作 | 变量变化 | 结果 |
|---|---|---|---|---|
| 滑动寻筝 | 全画面 | 拖拽画面浏览风筝群 | 每悬停一只触发一声叹息 | 收集声音记忆 |
| 靛青风筝 | `top:35%; left:45%; width:10%; height:15%` | 触碰自己的风筝 | Found_Own_Kite=true | 风筝亮起，显现文字 |
| 风筝线 | `top:50%; left:48%; width:2%; height:30%` | 沿着线向下拖拽 | 发现线末端系着城砖 | 解锁"城砖勒痕" |
| 不剪断 | 线末端城砖 | 点击选择"不剪" | Regret_Kept=true | 保留遗憾，获得"线"道具 |
| 剪断 | 线末端城砖 | 点击选择"剪断" | Regret_Cut=true | 遗憾消散，风筝飞入天空 |

**剪断与不剪断的双线后果**：

**不剪断**：
> 主角保留了自己的遗憾。阿缨的声音第一次以**轻松的语气**说："你留着吧。我也留着。我们就这样...系着。不是绑着。是系着。"
> 主角获得道具：**"未完成的线"**——一根永远系在两人之间的、看不见但摸得着的线。在后续节点中，当主角迷路时，可以**沿着这根线找回阿缨的位置**。

**剪断**：
> 风筝飞向天空，越飞越高，最终变成一颗**星星**。阿缨的声音说："它自由了。可自由的代价是...不再被看见。你真的想让我变成星星吗？星星很好，可是很冷。"
> 主角失去与这只风筝的连接，但获得**"空"的状态**——在后续"织补毛衣"节点中，如果选择"空"，奶奶会织出一件**特别轻盈的毛衣**（但保暖性不足，暗示心理防御薄弱）。

## 节点四：轮回登记处 · 织补毛衣

### Node_Y3_织补处：奶奶的针

**场景设定**：巨大的庙会礼堂。老奶奶们坐在高柜台后，用织毛衣的针为亡魂织**"来世福袋毛衣"**。排队的人在下棋、写字、编辫子。

**视觉资产**：
- `BG_Y3` (1920×1080): `A grand Chinese temple hall as playful afterlife registry, warm golden interior papered with red couplets, elderly women in floral cotton jackets at high counters knitting with glowing needles, translucent ghosts playing chess and braiding hair while waiting in queue, warm paper lantern light, absurdly cozy, Eastern Alice in Wonderland, fluid painterly animation, warm chiaroscuro, 4K, 16:9`

**音频设计**：
- 织毛衣的**竹针碰撞声**：咔哒、咔哒、咔哒——形成稳定的、催眠般的节奏
- 算盘声（但算的是祝福数量）
- 排队亡魂的**闲聊**："你下一世想当什么？" "我想当一颗不被摘的梅子。"

**剧本文本**：

**负责主角的奶奶**：
> "小伙子，你的毛衣破了太久了。两千年的洞，补起来要费点功夫。"
> *（她举起织了一半的毛衣——上面已经有一些花纹：玉佩的形状、一卷宗的形状、一个问号）*
> "下一世，你想带着什么走？只能带一样。"
> *（她凑近，压低声音）*

// 开放式输入 + 毛衣编织可视化
const userInput = document.getElementById('next-life-input');
const sweater = document.getElementById('sweater-canvas');

userInput.addEventListener('keydown', (e) => {
  if (e.key === 'Enter') {
    const value = userInput.value.trim();
    if (!value) {
      subtitle.show("不能说'没有'。'没有'也是一种选择。写下来。");
      return;
    }
    
    State.Next_Life_Cargo = value;
    
    // 根据输入决定织入位置
    const weaveConfig = getWeaveConfig(value);
    animateWeaving(sweater, weaveConfig);
    
    // 奶奶回应
    setTimeout(() => {
      subtitle.show(weaveConfig.grandmaResponse, {color: '#dcc8b0'});
      // 抛出毛衣动画
      sweater.classList.add('thrown');
      setTimeout(() => switchScene('Node_Y4_祈愿花园'), 4000);
    }, 3000);
  }
});

function getWeaveConfig(input) {
  if (input.includes('阿缨')) return {
    position: 'collar', color: '#8B3A2A', grandmaResponse: "好。把她织在领口。最贴近心跳的地方。但她不能替你心跳。"
  };

    position: 'cuff', color: '#2A4A6A', grandmaResponse: "好。熔成一颗纽扣。缝在袖口。每次抬手，都能看见。"
  };
  if (input.includes('记忆')) return {
    position: 'none', color: null, grandmaResponse: "记忆是毛线本身。你带的是记忆织出来的东西。再说一样。"
  };
  if (input.includes('笔')) return {
    position: 'inner-rib', color: '#1a1a1a', grandmaResponse: "...你还是要写？好。织成一根肋骨。内置的。别人看不见。"
  };
  if (input === '' || input.match(/空|无|nothing|empty/i)) return {
    position: 'chest-hole', color: 'transparent', grandmaResponse: "好。留一个洞。最轻的毛衣。也是最冷的。"
  };
  return {
    position: 'back', color: '#C8A030', grandmaResponse: `好。'${input}'织成花纹。字越多，花纹越密。太重了，下一世会累。`
  };
}
