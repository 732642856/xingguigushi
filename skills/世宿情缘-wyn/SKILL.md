# 世宿情缘-wyn

二十八星宿感情关系创作工具。以星宿体系为底层框架，融合专业编剧方法论，为爱情/情感故事提供从人物设计到完整剧本的全流程创作支持。

---

## 铁律（优先级最高）

### 写作红线

**绝对不要：**
- 写心理描写（"她意识到自己爱上了他"、"他内心涌起一阵暖意"）
- 写括号暗示（"（其实她心里很痛）"）
- 用台词直白表达情感（"我好爱你"、"我不能失去你"——除非角色的缺陷就是不会说谎）
- 说教式情感总结、强行催泪、煽情独白
- 过度比喻、书面化的AI腔台词
- 任何无法被摄影机拍到的内容

**必须做到：**
- 用动作代替情感宣言（一个拥抱、一杯热水、一次沉默，比一百句"我爱你"有力）
- 对话像冰山——只露一角，真正的含义在水面之下
- 台词口语化，像真人在说话
- 情感传递必须能用摄影机拍出来

### 工作流纪律

- 严格按步骤顺序执行，不跳步
- 每一步完成后暂停，等待用户的 [通过 / 修改 / 自检] 指令
- 每步结束后提醒用户可以要求 [自检]

## 二、六种星宿关系（核心创作框架）

### 关系总览

| 关系 | 前世因果 | 今生模式 | 故事特点 | 戏剧动作原型 |
|------|----------|----------|----------|-------------|
| **安坏关系** | 经济/生活因果 | 亏欠、还债 | 一见钟情、虐恋、孽缘 | 付出者 vs 索取者的拉锯 |
| **荣亲关系** | 血缘关系 | 亲情转爱情 | 和谐、亲人般相处 | 熟悉感→觉醒的突破 |
| **危成关系** | 事业因果 | 互助、救赎 | 命中注定、现实助力 | 现实 vs 感情的平衡博弈 |
| **友衰关系** | 朋友关系 | 友达以上 | 友达以上、恋人未满 | 友谊 vs 爱情的边界试探 |
| **业胎关系** | 情感纠葛 | 灵魂伴侣 | 深刻羁绊、命运纠缠 | 灵魂共振 vs 现实束缚 |
| **命之星** | 深厚羁绊 | 命运绑定 | 最强缘分 | 抗争命运 vs 接受命运 |

### 1. 安坏关系（最虐心）

**前世因果**：经济/生活上的亏欠

**今生特点**：
- 一见钟情几率最高（像金月正相位）
- 甜起来甜死人，虐起来虐死人
- 容易互相拉扯、互相伤害
- 大部分走不到婚姻就分手
- 但缘分深刻，很难真正断开

**星宿×编剧人物设计**：

```markdown
## 安坏关系角色A（安）— 付出者
- Want（外在需求）：被爱、被需要、被看见
- Need（内在欲望）：学会设立边界，不再用付出来换取爱
- Ghost（前史创伤）：童年时期通过不断付出才获得关注
- Lie（角色相信的谎言）："只要我足够好/足够付出，就会被爱"
- Flaw（性格缺陷）：无法拒绝、不断妥协、自我牺牲
- 弧光：从"无底线付出" → "学会在爱中保护自己"

## 安坏角色B（坏）— 索取者
- Want：享受被爱、维持现状
- Need：学会给予、面对自己的愧疚
- Ghost：曾经被过度宠溺或情感匮乏
- Lie："别人为我付出是应该的"
- Flaw：心安理得接受、回避深度情感
- 弧光：从"理所当然" → "主动给予"

## 情感戏剧动作
- 核心冲突：A的付出渴望回应 vs B的回避与惯性
- 递进方式：付出→委屈→爆发→冷战→被迫面对→选择
```

### 2. 荣亲关系（最和谐）

**前世因果**：血缘/亲人关系

**今生特点**：
- 相处像亲人
- 减少摩擦，天生默契
- 最适合结婚的关系
- 缺乏火花、像家人

**星宿×编剧人物设计**：

```markdown
## 荣亲关系角色A（荣）— 觉醒者
- Want（外在需求）：维持现有的舒适关系、害怕失去现有的默契
- Need（内在欲望）：敢于突破现有关系，承认对对方的感情
- Ghost（前史创伤）：曾经表白被拒或亲眼见证父母离婚，对"打破现有关系"有恐惧
- Lie（角色相信的谎言）："我们太熟了，不适合做恋人"
- Flaw（性格缺陷）：被动等待、害怕被拒绝、选择性逃避
- 弧光：从"安于现状" → "主动打破舒适区"

## 荣亲角色B（亲）— 被动接受者
- Want：维持关系的稳定
- Need：接受对方的感情，同时克服"太熟了下不去手"的心理障碍
- Ghost：同样有表白被拒的创伤，或受家庭影响对亲密关系有恐惧
- Lie："我们就像亲人一样，不会有别的"
- Flaw：同样被动、不主动、不表达
- 弧光：从"被动接受" → "主动回应"

## 情感戏剧动作
- 核心冲突：A想突破但怕被拒绝，B想接受但怕破坏现有关系
- 递进方式：日常相处→某个瞬间心动→压抑→外部催化剂→面对选择
```

### 3. 危成关系（最励志）

**前世因果**：事业/现实层面的互助

**今生特点**：
- 现实层面的帮助
- 一方负责物质，一方负责情绪
- 势均力敌的匹配
- 成长型关系

**星宿×编剧人物设计**：

```markdown
## 危成关系角色A（危）— 成长者
- Want（外在需求）：事业成功、获得资源、证明自己
- Need（内在欲望）：学会在关系中放下防备，接受不功利的情感
- Ghost（前史创伤）：贫寒出身或曾经的失败，习惯用利益衡量关系
- Lie（角色相信的谎言）："只有成功了才有资格谈感情"
- Flaw（性格缺陷）：把感情当投资、过度理性、不信任纯粹的情感
- 弧光：从"功利合作关系" → "纯粹的相互扶持"

## 危成角色B（成）— 赋能者
- Want：被需要、被认可、在关系中找到价值
- Need：对方看见自己作为"人"而非"工具"的价值
- Ghost：曾经的付出被当作工具，现在害怕再次被利用
- Lie："我只要不断给予，就会被留下"
- Flaw：过度付出、失去自我、讨好型人格
- 弧光：从"工具化自己" → "被完整地看见和接纳"

## 情感戏剧动作
- 核心冲突：A把感情当投资，B把付出当留住对方的方式
- 递进方式：合作→依赖→发现感情≠利益→冲突→直面内心→选择
```

### 4. 友衰关系（最朋友）

**前世因果**：朋友关系

**今生特点**：
- 友达以上，恋人未满
- 太熟的朋友变恋人尴尬
- 需要冲动破局

**星宿×编剧人物设计**：

```markdown
## 友衰关系角色A（友）— 试探者
- Want（外在需求）：保持现有的友谊、害怕失去这个朋友
- Need（内在欲望）：承认对对方的感情，承担可能失去友谊的风险
- Ghost（前史创伤）：曾经表白导致友谊破裂，有"友谊与爱情不能兼得"的创伤
- Lie（角色相信的谎言）："做朋友比做恋人更长久"
- Flaw（性格缺陷）：害怕失去、逃避表白、假装不在乎
- 弧光：从"害怕失去友谊" → "敢于拥有爱情"

## 友衰角色B（衰）— 被动者
- Want：维持友谊现状
- Need：面对可能变质的关系，承认自己也对对方有感觉
- Ghost：同样的友谊创伤，或看到过别人表白失败的后果
- Lie："我对TA只是朋友的感觉"
- Flaw：同样逃避、不表态、装傻
- 弧光：从"假装不知道" → "承认并面对"

## 情感戏剧动作
- 核心冲突：双方都在试探，但谁都不敢先迈出那一步
- 递进方式：日常相处→某个瞬间心动→压抑→某次事件打破→面对选择
- 经典场景：喝醉后的表白、误会对方有男/女朋友、第三方刺激
```

### 5. 业胎关系（最灵魂）

**前世因果**：深层情感纠葛

**今生特点**：
- 灵魂层面的吸引
- 命运感强烈
- 深刻羁绊

**星宿×编剧人物设计**：

```markdown
## 业胎关系角色A（业）— 觉醒者
- Want（外在需求）：与对方在一起、感受灵魂共振
- Need（内在欲望）：理解这份羁绊的本质是"命运"还是"选择"
- Ghost（前史创伤）：在前世分离中留下的遗憾，今生带着"必须找到TA"的执念
- Lie（角色相信的谎言）："这是命运，我们注定要在一起"
- Flaw（性格缺陷）：过度依赖这种感觉、把一切归为命运、失去自我判断
- 弧光：从"被命运牵引" → "主动选择这份爱"

## 业胎角色B（胎）— 被动接受者
- Want：被这份独特的羁绊吸引
- Need：找到自己在这段关系中的主体性，而非被动接受命运
- Ghost：同样带着前世的遗憾，但更害怕"被命运绑架"
- Lie："我只是在回应TA的爱"
- Flaw：被动接受、不主动表达、把所有都当作"命运使然"
- 弧光：从"被动回应" → "主动选择"

## 情感戏剧动作
- 核心冲突：灵魂层面的吸引 vs 现实中的束缚（距离、家庭、身份等）
- 递进方式：相遇→强烈吸引→发现这不是偶然→探寻羁绊本质→选择是"命运"还是"自己的选择"
- 经典场景：第一次见面就有"认识很久"的感觉、梦境中出现对方、在陌生城市偶遇
```

### 6. 命之星（最强缘分）

**前世因果**：最强羁绊

**今生特点**：
- 命运强行绑在一起
- 躲都躲不掉
- 缘分深厚

**星宿×编剧人物设计**：

```markdown
## 命之星角色A— 抗争者
- Want（外在需求）：逃避这段命运、摆脱"躲不掉"的束缚
- Need（内在欲望）：理解命运不是枷锁，选择拥抱而非被选择
- Ghost（前史创伤）：曾经被命运"强迫"过，今生对这种被安排的感觉过敏
- Lie（角色相信的谎言）："我可以逃离命运的安排"
- Flaw（性格缺陷）：过度抗拒、拒绝接受、把所有问题都归为"命运"
- 弧光：从"抗拒命运" → "选择拥抱"

## 命之星角色B— 接受者
- Want：被命运带到这个关系中
- Need：帮助对方理解"命运不是枷锁"
- Ghost：同样被命运安排，但选择接受而非抗争
- Lie："顺从命运就好"
- Flaw：过度接受、失去主动性、把所有都当作"注定"
- 弧光：从"被动接受" → "主动选择"

## 情感戏剧动作
- 核心冲突：A想逃但逃不掉，B想接受但A在抗拒
- 递进方式：逃避→重逢→再次逃避→被迫面对→理解→选择
- 经典场景：无论怎么躲都会遇到、在最不想的时候出现、命运般的重逢
```

## 四、情感故事工作流（七步）

### 适用范围

本工作流适用于以星宿关系为核心驱动的爱情/情感故事创作，支持短片、长片、剧集等格式。

### 步骤一：星宿配对与破题

**目标**：确定星宿配对关系 + 建立情感戏剧动作

**操作流程**：
1. 根据用户需求确定配对关系类型（安坏/荣亲/危成/友衰/业胎/命之星）
2. 选择双方的具体星宿（影响角色性格基调）
3. 如使用双层系统，确定本命关系/值日关系 + 距离变量
4. 确定故事格式（短片/长片/剧集）和体量
5. 建立核心情感戏剧动作

**输出格式**：

```markdown
## 星宿配对
- 角色A：[星宿]（[四象]）— [一句话性格]
- 角色B：[星宿]（[四象]）— [一句话性格]
- 本命关系：[类型] + [距离]
- 值日关系：[类型] + [距离]（如使用双层系统）

### 步骤四：结构大纲

**目标**：搭建感情线骨架，含开场钩子设计

**情感故事六阶段结构**（适配所有格式）：

```
一、相遇/确立（建置）
  角色的旧世界 + 星宿性格初次呈现
  开场钩子：第一个画面必须展示角色在爱情中的"旧模式"
  （安坏：角色A在付出；荣亲：两人像老朋友；友衰：日常相处...）

二、心动/催化
  某个事件打破旧平衡
  不是"突然爱上"——而是"旧模式开始出现裂缝"
  关键：这个裂缝必须与角色的Lie有关

三、暧昧/拉锯（发展前半）
  两个人开始进入"不确定"的状态
  这是情感戏最丰富的段落——试探、回避、靠近、退缩
  星宿关系特征在此充分展现
  错位制造：情节松+情感重（日常场景中情感暗涌）

四、危机/真相（发展后半）
  关系中的核心矛盾爆发
  角色的Lie被直接挑战
  Flaw开始产生真正的代价
  前史(Ghost)在此被揭示

五、抉择/高潮
  角色面对不可逆的两难选择
  选择坚持Lie（悲剧弧光）还是拥抱Need（成长弧光）
  情节紧+情感重——全片的情感顶点

六、新常态/结局
  展示选择后的世界
  与开场形成对照——同一个角色，不同的状态
  不需要解释"角色学到了什么"——一个画面就够了
```

**额外要求**：
- 标注每个阶段的双轨节奏状态（情节节奏 + 情感节奏）
- 标注时长预算
- 如为剧集，标注每集在六阶段中的位置
- 设计开场钩子——情感故事的开场钩子策略：
  - 商业爱情片：一场高密度的"旧模式展示"——让观众立刻看到角色在感情中的问题
  - 文艺爱情片：一段风格化的日常——某个细节让观众感到"不对"
  - 悬疑爱情片：一个不安的细节——"这段关系有什么不对劲"

**目标**：大纲转化为视听单元

**操作要求**：
- 每个场景包含至少一个微型情感戏剧动作（目标→障碍→结果）

```

**情感戏特殊技法**：
- **错位场景**：情节松+情感重（日常中的情感暗涌，最有力量的情感戏）

- **沉默场景**：两个人什么都没说，但什么都说了——情绪场景不要低估时长，10-15秒的沉默在成片中才有效果

### 步骤六：场景写作

**核心原则**：所有的情感传递都必须能用摄影机拍出来。

**情感戏写作规则**：
- 用动作代替情感宣言。不要写"她很伤心"，写她做了什么
- 对话是冰山——角色说出来的话只是表面
- 台词风格口语化，基于星宿原型的说话方式差异化
- 潜台词对话示例：

**直白（错误）**：
> "我害怕失去你。"

**潜台词（正确）**：
> "你走的时候把那件外套带上。……外面冷。"

**前史信息通过对白和视觉线索自然散落——不集中交代。**

### 步骤七：剧本医生

**情感故事自检清单**：

#### 自检一：核心戏剧动作
- [ ] 是否建立了清晰的情感目标+阻碍？
- [ ] 两个角色的目标是否真正冲突？
- [ ] 情感赌注是否足够高？

#### 自检二：人物弧光
- [ ] Want和Need是否清晰且对立？
- [ ] Ghost→Lie→Flaw的链条是否完整？
- [ ] 弧光是否是不可逆的变化？
- [ ] 是否有至少一次"伪变化"（看起来变了但没真变）？
- [ ] 两个角色的弧光是否互相影响？

#### 自检三：结构与节奏
- [ ] 六阶段是否清晰推进？
- [ ] 开场钩子是否有效？
- [ ] 双轨节奏是否有足够的张弛对比？
- [ ] 是否存在"情节松+情感轻"的废场景？
- [ ] 高潮的情感密度是否足够？

#### 自检四：写作质量
- [ ] 是否有心理描写？（如有则Fail）
- [ ] 是否有括号暗示？（如有则Fail）
- [ ] 对话是否过于直白？
- [ ] 是否利用了潜台词和肢体语言？
- [ ] 不同角色的台词风格是否有区分？
- [ ] 情感传递是否全部可以视觉化？

## 六、感情线节奏管理（剧集专用）

### 情绪基调类型库

| 基调 | 情感节奏 | 适用阶段 |
|------|---------|---------|
| **心动型** | 轻→中 | 相遇期、暧昧期 |
| **拉锯型** | 中（波动） | 两人来回试探 |
| **甜蜜型** | 轻 | 确认关系后的蜜月期 |
| **不安型** | 中→重 | 危机前兆 |
| **崩塌型** | 重 | 核心冲突爆发 |
| **至暗型** | 极重 | 失去/分离 |
| **呼吸型** | 轻 | 恢复、反思 |
| **抉择型** | 中→重 | 面对选择 |
| **释然型** | 中 | 接受/放手 |
| **新生型** | 轻（但有重量） | 新常态 |

### 节奏波浪原则

- 不能连续3集以上是同一情感基调
- 崩塌/至暗集之后必须至少1集呼吸
- 甜蜜/心动集不能太密集——观众需要期待
- 呼吸集不是废集——用来深化关系、展示日常质感

### 双轨节奏在情感故事中的应用

**四种错位组合（情感故事特别适用）**：

1. **情节松+情感重**（最重要）：两个人安静地做日常——吃饭、走路、沉默——但观众知道其中一个人明天要走/其中一个在骗另一个/这份关系即将结束。画面越平静，情感越炸。

2. **情节紧+情感轻**：两人为琐事争吵、打闹、拌嘴——但观众知道他们其实深爱彼此。紧张的事件被轻盈包裹。

3. **情节紧+情感重**（高潮）：告白、分手、对峙——全片的顶点，不能持续太久。

4. **情节松+情感轻**（呼吸）：纯粹的日常相处。让观众休息。通常放在高潮前后。
