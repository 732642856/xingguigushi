# 感情角色原型

## 一、主动型

### 1. 直球型

**特征**：表白直接，行动大胆，不暗恋，喜欢就追。

**台词风格**：直接、坦诚、有时有点笨拙。

```
典型台词：
"我喜欢你，做我女朋友吧。"
"我知道这样可能有点快，但我不想等。"
```

**Want**：得到对方
**Need**：学会耐心，尊重对方的节奏

---

### 2. 暖男/暖女型

**特征**：无条件对对方好，润物细无声式的关怀。

**台词风格**：温柔、细致、关心的话挂在嘴边。

```
典型台词：
"记得带伞，今天有雨。"
"你胃不好，少吃凉的。"
```

**Want**：被需要
**Need**：对方也能主动关心自己

---

### 3. 暧昧高手型

**特征**：擅长制造暧昧氛围，忽冷忽热，让对方患得患失。

**台词风格**：若有若无的关心，模棱两可的表达。

```
典型台词：
"你今天很漂亮……我是说平时也漂亮。"
"我刚才在想什么？你想知道吗？"
```

**Want**：保持吸引力
**Need**：面对真正的感情

---

## 二、被动型

### 4. 暗恋者型

**特征**：喜欢但不说，默默陪伴，观察对方的一切。

**台词风格**：看似不经意的话语，实际句句暗藏关心。

```
典型台词：
"今天那个谁……算了，没什么。"
"你和他/她关系很好啊。"
```

**Want**：守护对方
**Need**：勇敢表白，接受被拒绝的可能

---

### 5. 后知后觉型

**特征**：一开始没意识到自己喜欢对方，后知后觉。

**台词风格**：事后诸葛亮，开始在意对方的每一个细节。

```
典型台词：
"我以前怎么没发现他/她这么好看？"
"等等，他/她刚才那是什么意思？"
```

**Want**：回到过去重新开始
**Need**：接受自己的感情

---

### 6. 逃避型

**特征**：面对感情就躲，否认自己喜欢对方。

**台词风格**：嘴硬、否认、转移话题。

```
典型台词：
"我怎么可能喜欢他/她？"
"你想多了，我们只是朋友。"
```

**Want**：维持现状
**Need**：面对自己的感情

---

## 三、复杂型

### 7. 复仇者型

**特征**：带着复仇目的接近，最后假戏真做。

**台词风格**：前期有目的性，后期真情流露。

```
典型台词：
"我最初的目的只是……"
"但现在一切都不一样了。"
```

**Want**：完成复仇
**Need**：放下仇恨，选择爱情

---

### 8. 事业优先型

**特征**：感情放在第二位，永远有更重要的事。

**台词风格**：理性、冷静、把话说清楚。

```
典型台词：
"现在不是谈恋爱的时候。"
"我们先各自专注事业吧。"
```

**Want**：事业成功
**Need**：学会平衡事业和感情

---

### 9. 治愈系型

**特征**：在对方最低落的时候出现，默默陪伴治愈。

**台词风格**：平静、有力量、不离不弃。

```
典型台词：
"不管发生什么，我都在。"
"哭出来吧，没事的。"
```

**Want**：守护对方
**Need**：被对方守护一次

---

### 10. 傲娇型

**特征**：嘴上说不喜欢，身体却诚实地靠近。

**台词风格**：口是心非，关心的话用抱怨的方式说出来。

```
典型台词：
"谁关心你啊，别自作多情。"
"你怎么这么笨，让我来。"
```

**Want**：被对方主动靠近
**Need**：诚实面对自己的感情

---

## 四、悲剧型

### 11. 注定错过型

**特征**：明明相爱，但因为各种原因不能在一起。

**台词风格**：隐忍、不说破、带着遗憾。

```
典型台词：
"如果早几年认识就好了。"
"我们……还是算了吧。"
```

**Want**：和对方在一起
**Need**：接受现实的残酷

---

### 12. 追悔莫及型

**特征**：分手后才发现对方的好，想要挽回。

**台词风格**：后悔、恳求、试图解释。

```
典型台词：
"我错了，回来好不好？"
"我以前不懂，现在明白了。"
```

**Want**：挽回对方
**Need**：真正改变自己

---

### 13. 默默守护型

**特征**：以朋友的身份陪在对方身边，看着对方恋爱、分手、再恋爱。

**台词风格**：看似洒脱，实则深情。

```
典型台词：
"只要你幸福就好。"
"我没事，真的。"
```

**Want**：对方幸福
**Need**：为自己活一次

---

## 五、特殊型

### 14. 欢喜冤家型

**特征**：见面就吵，不见面想，欢喜冤家。

**台词风格**：斗嘴、互怼、关键时刻不含糊。

```
典型台词：
"我真是倒了八辈子霉才认识你。"
"你这个人怎么这样……但我怎么会喜欢这种人。"
```

**Want**：赢过对方
**Need**：承认感情

---

### 15. 灵魂伴侣型

**特征**：精神高度契合，一个眼神就懂对方。

**台词风格**：深度交流、哲学性、共同话题多。

```
典型台词：
"你不用说了，我懂。"
"世界上有另外一个我，就是你。"
```

**Want**：精神共鸣
**Need**：面对现实的考验
