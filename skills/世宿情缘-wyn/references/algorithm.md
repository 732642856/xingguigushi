# 世宿情缘算法

## 关系计算

### 本命关系
```
公式：(A星宿 - B星宿) % 6
0: 命之星
1: 友衰
2: 荣亲
3: 危成
4: 安坏
5: 業胎
```

### 距离计算
```
abs(A度 - B度) < 15: 近
15-30: 中
> 30: 远
```

## 组合映射

### 近距离模板
- 故事张力：缘分深、业力重
- 叙事节奏：快

### 中距离模板
- 故事张力：正常
- 叙事节奏：稳

### 远距离模板
- 故事张力：需要外力
- 叙事节奏：慢热
