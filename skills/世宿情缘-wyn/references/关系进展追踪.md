## 四、触发下一阶段的条件

### 安坏关系

```
相遇 → 甜蜜：一次深情对话/肢体接触
甜蜜 → 拉扯：第一次"不被在乎"
拉扯 → 真相：发现前世证据
真相 → 结局：接受or放弃
```

### 荣亲关系

```
认识 → 相处：确定关系
相处 → 矛盾：生活细节冲突
矛盾 → 融合：共同面对危机
融合 → 归宿：结婚/离婚
```

### 危成关系

```
认识 → 深入：第一次合作成功
深入 → 差距：一方明显进步
差距 → 巅峰：共同完成目标
巅峰 → 结局：继续合作or分道扬镳
```

### 友衰关系

```
朋友 → 暧昧：单独相处/肢体接触
暧昧 → 表白：某个触发事件
表白 → 确认：成功/失败
确认 → 定型：长期关系/退回朋友
```

### 业胎关系

```
相遇 → 确认：第一次感应验证
确认 → 深入：共同经历生死
深入 → 觉醒：发现多世记忆
觉醒 → 绑定：最终选择
```

### 命之星

```
相遇 → 验证：各种"巧合"验证
验证 → 考验：外部压力
考验 → 抉择：终极选择
抉择 → 结局：命运决定
```
