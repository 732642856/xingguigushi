# 感情场景模板库

## 一、相遇场景

### 1.1 命中注定型
```
场景：一见钟情
元素：目光相接、心动时刻、命运感
时长度：30秒-1分钟
双轨节奏：情节松+情感重
```

### 1.2 日久生情型
```
场景：日常相处中慢慢产生感情
元素：共同经历、互相照顾、习惯
时长度：可拉长
双轨节奏：情节松（日常）+情感渐重
```

### 1.3 冤家路窄型
```
场景：从互相讨厌到喜欢
元素：冲突、斗嘴、打脸
时长度：1-3分钟
双轨节奏：情节紧+情感变化
```

### 1.4 命中注定+冤家组合型
```
场景：命运安排相遇，但第一印象很糟
元素：命运感+反转
时长度：1-2分钟
双轨节奏：情节中+情感起伏
```

## 二、暧昧场景

### 2.1 制造机会型
```
场景：故意制造独处机会
元素：借口、紧张、笨拙
时长度：30秒-2分钟
双轨节奏：情节松+情感中
```

### 2.2 肢体接触型
```
场景：不小心或故意的身体接触
元素：触碰、尴尬、心动
时长度：30秒-1分钟
双轨节奏：情节松+情感重
```

### 2.3 深夜谈心型
```
场景：深夜单独相处，敞开心扉
元素：脆弱时刻、信任、建立
时长度：2-5分钟
双轨节奏：情节松+情感重
```

### 2.4 吃醋型
```
场景：一方吃醋但不肯承认
元素：酸话、否认、行为出卖
时长度：1-2分钟
双轨节奏：情节中+情感重
```

## 三、表白场景

### 3.1 直球表白型
```
场景：直接说出喜欢
元素：勇气、紧张、结果
时长度：1-3分钟
双轨节奏：情节松+情感重→爆发
```

### 3.2 行动表白型
```
场景：用行动表达感情
元素：礼物、惊喜、守护
时长度：2-5分钟
双轨节奏：情节中+情感重
```

### 3.3 借酒表白型
```
场景：借酒壮胆说出来
元素：酒、勇气、事后装傻
时长度：1-2分钟
双轨节奏：情节松+情感爆发
```

### 3.4 告白被拒型
```
场景：表白但被拒绝
元素：尴尬、失落、沉默
时长度：1-3分钟
双轨节奏：情节松+情感重
```

## 四、约会场景

### 4.1 第一次约会型
```
场景：紧张、生疏、试探
元素：选择地点、找话题、买单
时长度：3-10分钟
双轨节奏：情节中+情感波动
```

### 4.2 日常约会型
```
场景：普通的一天，但因为是和你
元素：平凡但甜蜜、相处细节
时长度：可长可短
双轨节奏：情节松+情感轻
```

### 4.3 雨中约会型
```
场景：雨中奔跑、躲雨
元素：浪漫、狼狈、靠近
时长度：2-5分钟
双轨节奏：情节中+情感重
```

### 4.4 重大日子约会型
```
场景：生日、纪念日等
元素：准备、惊喜、感动
时长度：3-10分钟
双轨节奏：情节中+情感重
```

## 五、争吵场景

### 5.1 冷战型
```
场景：互不理睬
元素：沉默、距离、想开口但没开口
时长度：可拉长
双轨节奏：情节松+情感重
```

### 5.2 激烈争吵型
```
场景：大声争吵
元素：重话、伤害、后悔
时长度：1-3分钟
双轨节奏：情节紧+情感重
```

### 5.3 假吵真做型
```
场景：表面吵架实际调情
元素：斗嘴、暧昧、旁观者清
时长度：1-3分钟
双轨节奏：情节紧+情感轻
```

## 六、分手场景

### 6.1 和平分手型
```
场景：理性分手
元素：感谢、祝福、释然
时长度：2-5分钟
双轨节奏：情节松+情感重
```

### 6.2 挽留型
```
场景：一方要分手，一方挽留
元素：恳求、拒绝、挣扎
时长度：2-5分钟
双轨节奏：情节中+情感重
```

### 6.3 突然分手型
```
场景：没有任何预兆的分手
元素：震惊、不敢相信、崩溃
时长度：1-3分钟
双轨节奏：情节紧+情感重
```

## 七、重逢场景

### 7.1 多年后重逢型
```
场景：多年后的相遇
元素：变化、感慨、物是人非
时长度：2-5分钟
双轨节奏：情节松+情感重
```

### 7.2 分手后重逢型
```
场景：分手后再次见面
元素：尴尬、装作不在乎、内心活动
时长度：1-3分钟
双轨节奏：情节中+情感重
```

### 7.3 命运重逢型
```
场景：怎么躲都躲不掉的重逢
元素：命运感、无奈、最终面对
时长度：2-5分钟
双轨节奏：情节中+情感起伏
```

## 八、亲密场景

### 8.1 第一次亲密型
```
场景：两性关系的第一步
元素：紧张、笨拙、温柔
时长度：可暗示，不需要具体展示
双轨节奏：情节松+情感重
```

### 8.2 日常亲密型
```
场景：情侣间的日常亲密
元素：自然、习惯、甜蜜
时长度：可长可短
双轨节奏：情节松+情感轻
```

### 8.3 复合后亲密型
```
场景：分手后又在一起
元素：珍惜、感慨、不容易
时长度：2-5分钟
双轨节奏：情节松+情感重
```

## 九、结局场景

### 9.1 圆满结局型
```
场景：有情人终成眷属
元素：幸福、承诺、新开始
时长度：2-5分钟
双轨节奏：情节松+情感重→轻
```

### 9.2 开放结局型
```
场景：故事结束但未来未知
元素：展望、留白
时长度：30秒-1分钟
双轨节奏：情节松+情感中
```

### 9.3 悲剧结局型
```
场景：相爱的人不能在一起
元素：遗憾、回忆、释然
时长度：2-5分钟
双轨节奏：情节松+情感重
```
