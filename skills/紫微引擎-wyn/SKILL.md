### 2. 命宫 / 身宫定位 ✅ 跨派共识

**安命宫：**  
从寅宫（index=2）起正月，**顺时针**数至生月宫止；  
再从该宫起子时，**逆时针**数至生时支止，即为命宫。

**安身宫：**  
从寅宫起正月，**顺时针**数至生月宫止；  
再从该宫起子时，**顺时针**数至生时支止，即为身宫。

**算法：**
```
命宫地支index = (寅index + 生月 - 1 - 时支index + 12) % 12
               = (2 + month - 1 - hourZhiIndex + 12) % 12

身宫地支index = (寅index + 生月 - 1 + 时支index) % 12
               = (2 + month - 1 + hourZhiIndex) % 12
```

地支顺序（顺时针）：子=0, 丑=1, 寅=2, 卯=3, 辰=4, 巳=5, 午=6, 未=7, 申=8, 酉=9, 戌=10, 亥=11

### 6. 安紫微星 ✅ 跨派共识

**口诀：** 生日除局商为月，一自寅起紫微定。只加不减到整除，阳退阴进记心中。

**算法：**
1. `q = Math.floor(day / ju)` → 商数（从1起，寅=1）
2. `r = day % ju` → 余数
3. 若 `r === 0`：紫微在 index = (2 + q - 1) % 12（寅起第q位）
4. 若 `r !== 0`：找最小正整数 `k` 使 `(day + k) % ju === 0`
   - `k` 为**奇数**：紫微 = 商位**逆退** k 位
   - `k` 为**偶数**：紫微 = 商位**顺进** k 位

## 四、三方四正 ✅ 跨派共识

**三合关系（四组）：**
- 子辰申（迁/疾/父）
- 丑巳酉（兄/子/官→因命宫位置而异）
- 寅午戌（命/财/官）
- 卯未亥（兄/夫/田）

**对宫（六冲）：**  
子↔午，丑↔未，寅↔申，卯↔酉，辰↔戌，巳↔亥

**算法：**
```javascript
// 给定宫index，求三方四正的四个宫index
function sanFangSiZheng(idx) {
  const opposite = (idx + 6) % 12;
  const triA = (idx + 4) % 12;
  const triB = (idx + 8) % 12;
  return [idx, triA, triB, opposite];
}
```

## 五、大限推演 ✅ 跨派共识

**起运岁数** = 五行局数（水二=2，木三=3，金四=4，土五=5，火六=6）

**行限方向：**
- 阳男 / 阴女：**顺行**（沿命宫→兄弟→夫妻方向，即逆时针地支）
- 阴男 / 阳女：**逆行**（沿命宫→父母→福德方向，即顺时针地支）

每限十年。大限宫干另起四化（同生年四化表，改用大限宫干查表）。

**阴阳判断：**
- 阳年（甲丙戊庚壬）出生男命 = 阳男
- 阴年（乙丁己辛癸）出生男命 = 阴男
- 阳年出生女命 = 阳女
- 阴年出生女命 = 阴女

## 六、流年 / 流月 / 流日 / 流时

### 流年 ✅ 跨派共识
太岁所临地支宫位即为流年命宫。  
`liuNianLifePalace = 太岁年支 index`

### 流月（斗君法） ✅ 跨派共识
斗君 = 流年命宫起寅月，**逆数**至出生月止，斗君所在宫位。  
以斗君宫为流年正月，**顺数**得各月宫位。  
`斗君index = (liuNianLifePalace - (birthMonth - 1) + 12) % 12`

### 流日
以流月命宫为起点，依地支顺数，每宫管若干日（按月总天数÷12分配）。

### 流时
以流日命宫为起点，每宫一时辰（子=23-1点）。

## 七、飞星宫干飞化（飞星派核心）

### 基本规则

每个宫位的宫干可产生四化，落入目标宫即"飞化入XX宫"。

- **自化**：宫干飞化落在本宫 → 能量自消，不聚
- **飞化出去**：落入他宫 → 形成因果链

### 禄转忌

1. A宫化禄入B宫
2. B宫宫干化忌入C宫
3. → **禄转忌**：A宫的资源/福气，因B宫的执念，流失于C宫

### 忌转忌

1. A宫化忌入B宫
2. B宫宫干化忌入C宫
3. → **忌转忌**：C宫受双重冲击，凶象加剧

### 自化含义（梁若瑜体系）

| 自化类型 | 象义 |
|---------|------|
| 自化禄 | 随性散漫，能量消散，头热尾冷 |
| 自化权 | 膨风装腔，意志力反复，主破耗 |
| 自化科 | 优柔寡断，文质但少定见 |
| 自化忌 | 不记恨，不坚持，无所谓，易吃亏 |

## 八、天地人三盘联动原则（飞星派）

1. **本命盘为体（天），大限盘为地，流年盘为人**  
   三盘共用一张盘，禄权科忌贯穿三才时空。

   - 本命迁移宫（对宫）冲命宫 → 凶  
   - 大限末宫冲本命命宫 → 凶  
   - 流年命宫冲本命命宫（犯太岁）→ 凶

3. **大限四化必须回本命盘解释**，严禁本命隔离论流年。

4. **先后天相应**：本命双象成组（禄忌同宫/对宫），大限流年飞化呼应，则事必发生。
