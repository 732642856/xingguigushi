# 角色原型参考

基于《无证之罪》《隐秘的角落》《阴门铃医》等作品的经典角色类型。

---

## 一、经典角色类型

### 1. 骆闻型（冷峻复仇者）

**特征**：外表温和，内里坚硬；有自己的道德准则，不被法律约束；为达目的愿意付出任何代价。

**台词风格**：平静、不解释、不辩论，只陈述事实。

```
典型台词：
"有些事，做了就做了。"
"我不是好人，但我也不是坏人。我只是……有债要还的人。"
```

**道德坐标**：X偏阴 / Y偏冷
**人物弧光**：复仇者 → 被复仇的对象的善意所动摇 → 自我牺牲式的了断

---

### 2. 严良型（灰色执法者）

**特征**：警察中的"异类"——不正统、不按规则办事，但有自己的一套；和犯罪者往往只有一线之隔。

**台词风格**：不正经中藏着锐利；爱用比喻和俗语；不严肃，但一针见血。

```
典型台词：
"法律？那玩意儿是给好人用的。我？我用的是经验。"
"你问我为什么不当警察了？因为我发现，有些案子，抓到人不代表抓到真相。"
```

**道德坐标**：X中性偏阳 / Y偏热
**人物弧光**：混日子 → 被卷入真相 → 不惜代价追寻

---

### 3. 郭羽型（理想主义者的堕落）

**特征**：起点善良，正直，有原则；被现实一次次打脸；逐步放弃底线，但每次放弃都伴随着痛苦和自我辩护。

**台词风格**：初期书面化、理想主义；中期开始出现犹豫和自我矛盾；后期变得冷硬，但偶尔会露出曾经的那个自己。

```
典型台词（早期）：
"我相信法律。我相信正义。"

典型台词（中期）：
"我只是想……活下去。有错吗？"

典型台词（后期）：
"别跟我说什么对错。对错是赢了之后才有的东西。"
```

**道德坐标**：从阳/热 → 逐渐向阴/冷滑落
**人物弧光**：正直律师 → 被现实教育 → 不得不"变坏" → 在某个人/某件事上找回初心

---

### 4. 朱慧如型（被命运碾压的女性）

**特征**：不是受害者，但处境让她成为"案件的一部分"；有韧性，有自己的底线；在极端环境下展现出意想不到的坚强或软弱。

**台词风格**：平静、克制；很少主动解释自己；话不多，但每句都有分量。

```
典型台词：
"你知道什么叫没得选吗？"

"我不是好人，也不是你想象的那种可怜人。我只是……运气不好的人。"
```

**道德坐标**：X中性 / Y偏热（有温度但被压制）
**人物弧光**：被动承受 → 被迫做出选择 → 承担选择的代价 → 找回主动权

---

### 5. 种老四型（骗子父亲）

**特征**：用谎言构建世界；骗术是他的生存方式；但对"家人"有真实的情感；双重身份造成巨大内心冲突。

**台词风格**（面对外人）：满嘴玄学、云山雾罩、文化包装
**台词风格**（面对儿子）：粗俗、直接、底层智慧

```
典型台词（包装模式）：
"此乃命中劫数，非寻常药石可解……"

典型台词（真实模式）：
"你个傻小子，记住——爹骗谁都不骗你。"
```

**道德坐标**：X极阴 / Y极热（表面冷酷，内心最柔软）
**人物弧光**：用谎言维持体面 → 被真相击穿 → 用真实的方式完成守护

---

### 6. 阿河型（清醒的旁观者）

**特征**：有特殊能力（遥视/感知）；能看穿别人看不到的东西；却也因此更孤独；自嘲是他的保护色。

**台词风格**：自嘲、观察锐利、冷眼看世界；偶尔流露温情。

```
典型台词：
"我看人很准——比他们自己看自己还准。所以我知道，大部分人，根本不想被看穿。"

"你知道最讽刺的是什么吗？我能看见所有人的未来，唯独看不见自己的。"

"我爸说我是垃圾堆里捡的。这话我信，因为我的记忆就是从生锈的铁皮桶旁边开始的。"
```

**道德坐标**：X偏阳（清醒者）/ Y偏冷（旁观者）
**人物弧光**：被动接收能力 → 主动选择如何使用能力 → 接受"看透一切但依然选择善良"的宿命

---

### 7. 黄毛型（小人物的挣扎）

**特征**：底层混混，看似凶狠，实则是被利用的工具；偶尔展现道义，但在大是大非面前往往选错边。

**台词风格**：匪气、江湖气、夸张、好面子。

```
典型台词：
"我跟你说，在这片儿，我说的话，比警察管用。"
"你不服？行，咱找个地方练练。"
"我就是个小混混，你别指望我有什么觉悟。"
```

**道德坐标**：X偏阴 / Y偏热（讲义气，但不辨是非）
**人物弧光**：被利用的工具 → 在某件事上展现了意外的担当 → 为此付出代价

## 二、角色关系化学反应

| 组合 | 化学反应 | 戏剧张力 |
|---|---|---|
| 骗子父亲 + 超视儿子 | 父亲用谎言包装儿子的感知，儿子的真实感知不断挑战父亲的包装 | 真相 vs. 善意的谎言 |
| 复仇者 + 灰色执法者 | 两人都在法律边缘，但走向相反 | 殊途同归的宿命感 |
| 理想主义者 + 被命运碾压者 | 一个在堕落，一个在坚守 | 选择 vs. 没得选 |
| 小人物 + 大家族 | 蚂蚁撼大象式的错位冲突 | 荒诞 + 悲凉 |

## 三、人物出场设计

### 首次出场原则

**每个角色的第一次出场，必须回答三个问题：**

```
1. 他是干什么的？（身份锚定）
2. 他在这个场景里的目的是什么？（动机锚定）
3. 他的哪一点让观众记住他？（印象锚定）
```

**示例（种老四首次出场）：**
```
场景：派出所门口，警察身后

身份：被叫来接儿子的人（后来揭示：养父）
动机：把阿河从派出所弄出来（后来揭示：不只是捞人，还有更深的目的）
印象锚定：他穿着比儿子还讲究，却满嘴跑火车——这人有点意思
```

### 群戏角色分配

```
核心角色（3-5人）：全程跟进，有独立弧光
次要角色（每集2-3人）：功能性出场，完成特定任务后淡出
背景角色：点缀真实感，不要求记忆点
```
