# 人物设定模板库

## 一、核心特质模板

### 性格细节生成模板

根据主星组合，生成3-5个具体性格特征：

**紫微星系性格模板：**
```
【骄傲自尊】内心深处有强烈的自尊心，不轻易向人低头，即使面对困境也保持着一份傲骨。
【掌控欲强】喜欢掌控局面，无论是工作还是生活，都希望能够主导事情的发展方向。
【耳根软】表面上强势，但内心容易被亲近的人影响，对信任的人言听计从。
【多疑谨慎】对人和事都保持一定的警惕，不会轻易相信他人，需要时间建立信任。
【仁慈宽厚】虽然骄傲，但内心善良，对弱者有同情心，愿意伸出援手。
```

**杀破狼性格模板：**
```
【不安于室】内心深处有一股躁动，无法忍受平淡无奇的生活，总是寻求刺激和变化。
【横冲直撞】做事果断，不拖泥带水，但有时过于冲动，容易得罪人。
【百炼成钢】经历越多挫折越坚强，有着超乎常人的抗压能力和恢复力。
【冒险精神】对未知充满好奇，愿意冒险尝试新事物，不惧失败。
【大开大合】人生大起大落，要么辉煌要么低谷，很少有平淡期。
```

**机月同梁性格模板：**
```
【温和细腻】性格温和，心思细腻，善于察言观色，能够体贴他人感受。
【安分守己】不喜欢冒险，追求稳定，愿意在熟悉的领域深耕细作。
【耐心细致】做事有耐心，能够处理繁琐复杂的工作，不急躁。
【随遇而安】适应能力强，能够在各种环境中找到自己的位置。
【知足常乐】对生活要求不高，容易满足，心态平和。
```

**巨日格局性格模板：**
```
【外向爽朗】性格外向，待人热情，容易与人打成一片。
【正义感强】有强烈的正义感，看不惯不公平的事情，喜欢打抱不平。
【乐观进取】无论遇到什么困难，都能保持乐观的心态，积极面对。
【叛逆心态】不喜欢被束缚，对传统和权威有质疑精神。
【同情心重】对弱者有强烈的同情心，愿意为民请命。
```

### 行事风格生成模板

**领导型（紫微、天府）：**
```
【决策方式】喜欢听取多方意见后做出决定，但一旦决定就不轻易改变。
【处事特点】注重面子和形象，做事讲究排场，不喜欢小家子气。
【管理风格】倾向于集权管理，重要事情亲力亲为，对下属要求严格。
【危机处理】面对危机时能够保持冷静，但有时会过于自信而忽视风险。
```

**谋士型（天机、天梁）：**
```
【决策方式】深思熟虑，多方权衡，不轻易下决定，但一旦决定就考虑周全。
【处事特点】注重细节，善于从复杂情况中找到关键点。
【管理风格】倾向于幕后策划，不喜欢站在台前，但掌控全局。
【危机处理】面对危机时能够冷静分析，找到最佳解决方案。
```

**实干型（武曲、七杀）：**
```
【决策方式】果断迅速，不拖泥带水，凭直觉和经验做决定。
【处事特点】注重实效，不喜欢空谈，用行动证明自己。
【管理风格】雷厉风行，要求执行力，对拖延和借口零容忍。
【危机处理】面对危机时敢于冒险，用勇气和行动化解危机。
```

**服务型（天同、天相）：**
```
【决策方式】考虑他人感受，倾向于折中和妥协，避免冲突。
【处事特点】注重和谐，善于协调各方关系，是团队的润滑剂。
【管理风格】服务型领导，关心下属，营造和谐的工作氛围。
【危机处理】面对危机时倾向于保守，用温和方式化解矛盾。
```

- 领导才能：天生具有领导气质，能够凝聚人心，在党内有很高的威望。
