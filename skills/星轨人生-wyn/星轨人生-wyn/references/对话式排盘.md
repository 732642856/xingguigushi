# 星轨人生 - 对话式排盘入口

简化版的紫微斗数排盘，用自然语言快速获取命盘信息。

---

## 一、对话式排盘模板

### 快速问法（直接复制使用）

```
"帮我看一个1975年农历3月17戌时的男性命盘"
"1990年农历8月初5寅时女生怎么排"
"1985年农历5月20午时的命盘"
"帮我查一个1978年2月14卯时的"
```

### 自动解析规则

| 输入项 | 识别方式 | 示例 |
|--------|----------|------|
| 年份 | 4位数字 | 1975、1990 |
| 农历月 | "农历"+数字 或 直接数字 | 农历3月、8月 |
| 农历日 | "初"+"数字" 或 "十五"等 | 初5、初十、十五 |
| 时辰 | 地支：子丑寅卯辰巳午未申酉戌亥 | 戌时、寅时 |
| 性别 | 男/女/男性/女性 | 男性、女生 |

### 输出格式

```markdown
## 命盘简报

### 基本信息
- 出生：1975年农历3月17戌时
- 性别：男
- 年龄：51岁（2026年）

### 命宫
- 命宫主星：[主星名]
- 落入宫位：[宫位名]
- 性格特点：[简要描述]

### 重点宫位
| 宫位 | 主星 | 四化 | 关键词 |
|------|------|------|--------|
| 财帛宫 | ... | ... | ... |
| 官禄宫 | ... | ... | ... |
| 夫妻宫 | ... | ... | ... |

### 2026年流年
- 大限：[XX-XX岁]
- 流年宫位：[宫位]
- 流年星曜：[星名]
- 运势提示：[简要]

### 人物设定建议
基于此命盘，建议创作角色：
- 性格倾向：[...]
- 命运主题：[...]
- 人生节点：[...]
```

## 二、常用查询句式

### 完整版（最详细）

```
"帮我排一个完整的紫微命盘，1975年农历3月17戌时，男，要包含大限流年"
```

### 简版（只要主星）

```
"1975年3月17戌时 男 命宫主星是什么"
```

### 运势版（只看流年）

```
"1975年农历3月戌时的人2026年运势怎么样"
```

### 对比版（两人对比）

```
"帮我对比两个人的命盘，男1975年农历3月17戌时，和女1990年农历8月初5寅时"
```

## 四、使用示例

### 示例1：快速查询

**输入**：
```
帮我看一个1975年农历3月17戌时的男性
```

**输出**：
```
## 命盘简报

出生：1975年农历3月17戌时 | 男 | 2026年51岁

命宫：午宫（子宫顺时针第6位）
主星：紫微星
辅星：天机星
四化：贪狼化禄

性格特点：紫微坐命者，领导力强，有权威感...

2026年流年运势：
- 处于辛酉大限（41-50岁）
- 流年行至父母宫
- 贪狼化禄，适合开展新项目
```

### 示例2：创作角色

**输入**：
```
我要写一个商战小说，需要一个1985年农历5月20午时出生的女性角色，她是职场精英
```

**输出**：
```
## 角色设定：职场精英女性

### 命盘特征
- 命宫：[主星]+[辅星]
- 官禄宫：[星曜]+[四化]

### 性格
[基于命盘的详细性格描述]

### 命运轨迹
- 事业高峰期：XX岁
- 关键转折点：XX岁
- 最终结局：[事业型/家庭型/平衡型]

### 与其他角色关系建议
- 与[类型1]角色的关系模式：[...]
- 与[类型2]角色的关系模式：[...]
```

---

## 五、注意事项

1. **农历/公历**：如果是公历日期，需注明"公历"
2. **时辰**：紫微斗数以地支记时，子时（23:00-01:00）需分早晚子时
3. **性别**：影响星曜阴阳性和四化方向
4. **流年**：需要提供"当前年份"才能计算流年运势
