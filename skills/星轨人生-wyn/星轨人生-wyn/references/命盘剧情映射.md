## 三、四化与剧情走向

### 四化 → 命运转折

| 四化 | 剧情作用 | 典型转折 |
|------|----------|----------|
| 化禄 | 收获/机遇 | 获得资源、贵人相助、意外收获 |
| 化权 | 争夺/掌控 | 权力争夺、地位上升、控制权 |
| 化科 | 声誉/帮助 | 名声提升、获得认可、得到帮助 |
| 化忌 | 失去/困境 | 损失、挫折、困境、挑战 |

### 四化组合 → 剧情类型

| 组合 | 剧情走向 | 示例 |
|------|----------|------|
| 禄+权 | 上升路线 | 创业者获得投资，事业发展 |
| 禄+科 | 声名鹊起 | 意外成名，获得认可 |
| 禄+忌 | 得失之间 | 获得后失去，或因祸得福 |
| 权+科 | 权力与名声 | 位高权重又受尊重 |
| 权+忌 | 权力斗争 | 争权失败，或权力带来代价 |
| 科+忌 | 名誉受损 | 被打压、被误解 |

## 四、大限与剧情阶段

### 大限周期 → 人生阶段

| 大限 | 人生阶段 | 剧情节奏 |
|------|----------|----------|
| 第1大限（0-10岁） | 童年/起源 | 背景设定、伏笔埋设 |
| 第2大限（11-20岁） | 成长/求学 | 性格形成、梦想萌芽 |
| 第3大限（21-30岁） | 起步/探索 | 初入社会、碰壁/机遇 |
| 第4大限（31-40岁） | 奋斗/崛起 | 事业拼搏、关键抉择 |
| 第5大限（41-50岁） | 巅峰/危机 | 权力巅峰、危机显现 |
| 第6大限（51-60岁） | 沉淀/传承 | 巅峰对决、传承安排 |
| 第7大限（61-70岁） | 收尾/回顾 | 结局铺垫、最终抉择 |

### 剧情设计示例

```
角色A（1975年生，2026年51岁）：
- 处于第5大限（41-50岁）：人生巅峰期
- 2026流年：流年宫位[宫位名]
- 适合的剧情阶段：权力巅峰、面对危机、最终抉择
```
