# 情感戏高级创作

> 进阶情感戏创作技巧

---

## 一、潜文本情感设计

### "不说"的情感

```
最有力的情感对白是没说完的话

"我曾经..."
（不说了，观众自己补全）

"你果然..."
（不说完，让对方猜）
```

### 无声情感

```
不用"我爱你"

- 眼神
- 动作
- 细节
- 习惯
```

### 情感多层次

```
表层：行为/对白
中层：动机/心理
深层：潜意识/命盘
```

## 二、经典情感结构

### 经典甜宠结构

```
第1-5集：相遇 → 吸引
第6-10集：暧昧 → 告白
第11-15集：热恋 → 波折
第16-20集：危机 → 解决 → HE
```

### 经典虐恋结构（安坏）

```
第1-3集：甜蜜开场
第4-8集：危机隐现
第9-12集：真相揭开 → 虐心
第13-15集：选择 → TE/BE
```

### 命运感结构（业胎/命之星）

```
开头：暗示命运
中段：挣扎
结尾：超越/放下
```

## 三、高级场景设计

### "糖"场景

```
特征：甜而不腻

要素：
- 日常细节
- 专属习惯
- 私人笑点
- 互相宠溺
```

### "虐"场景

```
特征：痛而不俗

要素：
- 不说出口的委屈
- 误会+不解释
- 为了对方好而推开
- 真相揭露的瞬间
```

### "命运"场景

```
特征：宿命感

要素：
- 宿命论暗示
- 星宿关系体现
- 命盘呼应
- 关键时刻选择
```

## 四、自检清单

### 情感自检

```
□ 对白是否太直白？
□ 情感是否有层次？
□ 星宿关系是否体现？
□ 命盘是否呼应？
□ 转折是否自然？
□ 结局是否合理？
```

### 红线检查

```
□ 没有过度工业糖精
□ 虐点不刻意
□ 不绿茶/不中央空调
□ 价值观正确
□ 过审风险
```
