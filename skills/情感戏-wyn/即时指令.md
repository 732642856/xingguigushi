# 情感戏即时指令

> 快速出活，不用加载

---

## 核心指令

### "星宿配对"
```
输入：两个角色基础信息
输出：
- 星宿关系类型
- 相处模式
- 关系张力点
```

### "角色命盘"
```
输入：角色属性
输出：
- 命宫主星
- 性格+情感模式
- 匹配建议
```

### "情感线"
```
输入：起始+目标
输出：
- 三段式发展
- 关键转折点
```

### "告白"
```
输入：场景/角色
输出：
- 告白对白（含潜文本）
- 场景描写
```

### "分手"
```
输入：原因/角色
输出：
- 分手对白
- 情感变化
```

### "甜场景"
```
输入：日常/细节
输出：
- 甜蜜互动
- 细节设计
```

### "虐场景"
```
输入：矛盾点
输出：
- 虐心设计
- 内心独白
```
