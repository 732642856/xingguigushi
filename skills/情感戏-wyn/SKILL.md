# 情感戏-wyn

## 三、情感线设计

### 相遇模式

```
相遇类型：
- 宿命型：命之星/业胎，前世羁绊
- 成长型：荣亲/危成，日久生情
- 冲击型：安坏/友衰，一见钟情/不打不相识
```

### 发展公式

```
甜蜜期（1-3集）→ 危机期（4-6集）→ 真相期（7-9集）→ 选择期（10集）
```

### 结局设计

```
HE：破镜重圆 / 放下过去
BE：为爱放手 / 命运弄人
OE：留白 / 开放式
```
