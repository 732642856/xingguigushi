# 虚拟币与房产泡沫：两个时代的财富幻觉

## 虚拟币生态

### ICO泡沫（2017-2018）
首次代币发行，无监管野蛮生长
- 全球ICO融资超200亿美元
- 90%项目归零，"空气币"泛滥
- 中国2017年9月4日叫停ICO

### 币圈割韭菜套路
1. **拉盘砸盘**：庄家控盘，吸引散户后抛售
2. **传销币**：邀请返佣，层级奖励，本质庞氏
3. **假交易所**：数据造假，用户充值后无法提现
4. **钱包跑路**：DeFi项目卷款

### PlusToken崩盘（2019）
- 号称"智能搬砖"，实际资金盘
- 400亿人民币，200万受害者
- 主犯潜逃海外，最终被捕

## 房产泡沫

### 日本泡沫（1985-1991）
"广场协议"后日元升值 → 热钱涌入房地产 → 东京地价可买全美
- 1991年泡沫破裂，房价暴跌50-70%
- 银行坏账，"失去的30年"
- 教训：低利率+过度信贷=灾难

### 中国房产
- 海南泡沫（1993）：地产热后烂尾楼遍地
- 鬼城现象：鄂尔多斯、曹妃甸
- 2020年后：恒大危机、碧桂园债务，行业洗牌

### 泡沫共性
1. 杠杆过高（首付贷、零首付）
2. 投机主导（炒房团、代持）
3. 政策突变（加息、限购）触发破裂
4. 最终买单：刚需接盘，银行坏账
