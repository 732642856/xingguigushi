# 零售革命：从2元店到科技新贵

## 2元店经济学
核心逻辑：极致成本控制 + 薄利多销 + 冲动消费

### 盈利模式
- 进货成本0.3-0.8元，售价2元，毛利率60-85%
- 走量：日均流水3000-8000元
- 选址：人流密集（学校、菜市场、城中村）

### 货源秘密
- 义乌小商品市场
- 尾货清仓（过季、瑕疵品）
- 山寨货（无品牌溢价）

## 拼多多模式（2015-）
农村包围城市 + 社交裂变 + 极致低价

### 核心策略
- **拼团**：微信分享，降低获客成本
- **C2M**：消费者直连工厂，砍掉中间商
- **下沉市场**：三四线城市+农村，蓝海竞争

### 争议
- 假货问题（拼多多早期外号"拼夕夕"）
- 员工猝死、加班文化
- "仅退款"政策倒逼商家内卷

## 宜家模式（IKEA）
仓储式销售 + 平板包装 + DIY组装

### 核心创新
- **成本控制**：顾客自取、自助组装，节省人工
- **场景营销**：样板间激发购买欲望
- **瑞典设计**：简洁、实用、低价

## 科技新贵

### 马斯克（Elon Musk）
- PayPal → Tesla → SpaceX → X
- 第一性原理：回归物理本质重新设计
- 政府补贴依赖：电动车补贴、NASA合同

### 中国科技巨头
- 阿里：电商平台 → 云计算 → 金融科技
- 腾讯：社交 → 游戏 → 投资帝国
- 字节：算法推荐 → 全球化（TikTok）

## 剧本素材
- 2元店老板：从摆地摊到连锁品牌
- 拼多多卖家：价格战中的生存游戏
- 科技公司：从车库创业到垄断巨头
