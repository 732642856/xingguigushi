# 江湖骗术：江湖三十年揭秘

> 李幺傻《江湖三十年》（又名《中国式骗局》）：亲历者讲述民国江湖往事

## 江湖八大门

### 上四门（阳门）
1. **惊门**：算命、看相、测字、风水
2. **疲门**：行医、卖药、江湖郎中
3. **飘门**：戏法、杂技、变戏法
4. **册门**：文书、造假、代写、刻章

### 下四门（阴门）
5. **风门**：强盗、土匪、剪径
6. **火门**：炼丹、炼金、火药
7. **爵门**：诈骗、碰瓷、放鸽子
8. **要门**：乞丐、装可怜、假残废

## 经典骗局

### 仙人跳
美女色诱 → 同伙闯入"捉奸" → 勒索钱财
现代变体：酒托、饭托、杀猪盘

### 碰瓷
假装被车撞/被物品砸伤 → 索要赔偿
专业碰瓷党：选择无监控路段、老年人易成功

### 放鸽子
假意介绍对象/工作/生意 → 收取中介费后消失
现代变体：招聘诈骗、婚恋诈骗

### 拆白党
民国骗局：俊男美女勾引富家子/阔太太 → 骗取钱财后消失
本质：情感诈骗的早期形态

### 做旧
伪造古董、假文物 → 高价卖给收藏者
产业链：盗墓→做旧→拍卖行洗白

### 放鹞子
先给目标小恩小惠（免费礼品、中奖信息）→ 建立信任 → 大额诈骗
现代变体：保健品诈骗、养老诈骗

## 江湖切口（黑话）
- 并肩子：朋友、同伙
- 肥羊：有钱的目标
- 开瓜：下手、动手
- 风紧：情况紧急，快跑
- 点子：目标人物

## 剧本素材
- 老千：从学艺到金盆洗手的一生
- 算命先生：察言观色的心理学大师
- 碰瓷专业户：演技与风险的平衡
- 现代杀猪盘：爱情陷阱与数字鸿沟
