# 紫微斗数角色设计（144命盘版）

## 概述

本技能专为剧本/小说编剧打造，基于紫微斗数144种标准命盘、108颗星曜体系，实现"人物基础属性输入→自动匹配紫微命盘→生成完整人物设定"的全流程智能创作工具。

**核心特点：**
- 数据100%来自本地紫微斗数命盘库，精准、可溯源、不瞎编
- 采用传统三合派紫微斗数体系：14主星+108颗全星曜
- 固定使用144种标准基本命盘（72主星组合×2阴阳）
- 区分男命、女命不同解读逻辑

## 使用流程

### 第一步：收集基础属性

引导用户选择/输入以下编剧创作核心属性：

**必选属性：**
- 性别（男/女）：直接影响命盘匹配与解读逻辑
- 年龄阶段（少年/青年/中年/老年）：影响人生节点描述

**可选属性：**
- 职业（商人/政客/艺人/武将/文人/工薪/江湖/权贵/平民）
- 性格倾向（内敛/张扬/谋略/善良/果敢/优柔）
- 家庭背景（富贵/工薪/寒门/单亲/孤儿）
- 人生基调（顺遂/坎坷/大起大落/中年转运）

### 第二步：命盘匹配

根据用户所选属性，从144命盘库中匹配最贴合的命盘。

**命盘匹配逻辑：**

1. **主星组合匹配**
   - 杀破狼格局（七杀/破军/贪狼）：适合武将、江湖、大起大落、果敢性格
   - 紫府廉武相格局（紫微/天府/廉贞/武曲/天相）：适合权贵、政客、谋略、富贵背景
   - 机月同梁格局（天机/太阴/天同/天梁）：适合文人、工薪、内敛、顺遂人生
   - 巨日格局（巨门/太阳）：适合艺人、政客、张扬、中年转运
   - 命无正曜：适合平民、坎坷、随波逐流

2. **辅助星曜匹配**
   - 六吉星（文昌/文曲/左辅/右弼/天魁/天钺）：增强贵人运、才华、领导力
   - 六煞星（擎羊/陀罗/火星/铃星/地空/地劫）：增加波折、冲突、变动

3. **性别差异**
   - 男命：侧重事业、财运、权力
   - 女命：侧重情感、家庭、人际关系

### 第三步：命盘确认

向用户展示匹配结果：
- 命盘名称（如：紫微在午、武曲贪狼在未）
- 主星组合
- 核心格局
- 男女命差异提示

询问用户：【确认使用】或【重新匹配】

### 第四步：生成完整人物设定

用户确认命盘后，基于该命盘的星曜格局、命理逻辑，生成可直接用于剧本的人物设定。

**生成内容结构：**

1. **核心特质**
   - 性格细节（3-5个具体特征）
   - 行事风格（决策方式、处事特点）
   - 天赋/短板
   - 价值观

2. **家庭背景**
   - 父母关系
   - 原生家庭影响
   - 成长环境

3. **人生关键节点**
   - 幼年重大事件
   - 青年转折
   - 中年机遇/坎坷
   - 晚年归宿

4. **人际与情感**
   - 贵人运
   - 小人运
   - 情感走向
   - 社交模式

5. **事业财运**
   - 职业方向
   - 财运起伏
   - 适合领域

### 第五步：结果调整

根据用户反馈微调内容，直至满意。

### 核心格局速查

**杀破狼格局**
- 构成：命宫有七杀、破军、贪狼（一颗或两颗，不见其他主星）
- 特质：不安分、喜欢挑战、横冲直撞、生命周期起伏大
- 适配：武将、江湖、大起大落、果敢性格

**紫府廉武相格局**
- 构成：命宫有紫微、天府、廉贞、武曲、天相（一颗或两颗）
- 特质：
  - 加左辅右弼：君臣庆会格，领导统御、雄才大略
  - 不加左辅右弼：单打独斗、高傲孤僻
- 适配：权贵、政客、谋略、富贵背景

**机月同梁格局**
- 构成：命宫有天机、太阴、天同、天梁（一颗或两颗）
- 特质：温和安逸、心思细密、安分守己、适合繁琐工作
- 适配：文人、工薪、内敛、顺遂人生

**巨日格局**
- 构成：命宫有巨门、太阳（一颗或两颗）
- 特质：外向爽朗、乐观进取、正义感强、喜打抱不平
- 适配：艺人、政客、张扬、中年转运

**命无正曜**
- 构成：命宫无主星
- 特质：主观意识弱、随波逐流、易受环境影响
- 适配：平民、坎坷、被动型人物

### 六吉星与六煞星影响

**六吉星（文昌、文曲、左辅、右弼、天魁、天钺）**
- 增强：才华、贵人运、领导力、学习能力
- 命三方四正吉星多：人生平稳、易得助力

**六煞星（擎羊、陀罗、火星、铃星、地空、地劫）**
- 增加：波折、冲突、变动、挑战
- 命三方四正煞星多：人生起伏大、多磨砺

### 四化象含义

- **化禄**：福禄、财缘、顺利、增加
- **化权**：权力、掌控、强势、竞争
- **化科**：贵人、名声、科甲、缓和
- **化忌**：阻碍、波折、执着、收藏

### 第六步：生成角色外貌（可选）

根据用户需要，基于命盘和面相学原理，生成角色的外貌特征。

**生成内容结构：**

1. **面容特征**
   - 额头、眉毛、眼睛、鼻子、嘴唇、下巴
   - 整体面容轮廓和气质

2. **身材体态**
   - 身高、体型、站姿、动作特点

3. **气质气场**
   - 整体气质、气场强弱、第一印象

4. **细节特征**
   - 特殊标记（疤痕、痣等）
   - 声音特点
   - 穿着风格
   - 体味/香水味

**面相学匹配逻辑：**

1. **主星对应面相**
   - 紫微：帝王之相，高贵威严
   - 天机：谋士之相，温文尔雅
   - 太阳：光明之相，阳光开朗
   - 武曲：财星之相，刚毅果断
   - 天同：福气之相，温和善良
   - 廉贞：复杂之相，神秘艺术
   - 天府：财库之相，稳重包容
   - 太阴：温柔之相，柔美细腻
   - 贪狼：桃花之相，魅力四射
   - 巨门：口才之相，理性冷静
   - 天相：印星之相，谨慎得体
   - 天梁：荫星之相，清高正直
   - 七杀：将星之相，勇猛锐利
   - 破军：耗星之相，变动不定

2. **格局对应气质**
   - 杀破狼：勇猛冒险，眼神锐利
   - 紫府廉武相：高贵领导，气场强大
   - 机月同梁：温和文雅，书卷气浓
   - 巨日格局：阳光正义，感染力强
   - 命无正曜：普通随和，不引人注目

3. **吉星煞星影响**
   - 六吉星：提升外貌气质，眼神明亮
   - 六煞星：增加特殊标记，眼神复杂

4. **性别年龄调整**
   - 男命：强调阳刚、威严、领导力
   - 女命：强调柔美、魅力、亲和力
   - 年龄：少年清澈、青年明亮、中年深邃、老年慈祥

## 输出规范

1. **语言风格**：具象化描述，避免晦涩命理术语，符合剧本创作习惯
2. **内容深度**：每个模块提供3-5个具体、可操作的细节点
3. **逻辑一致性**：所有内容必须符合命盘逻辑，不自相矛盾
4. **创作可用**：直接可用于剧本人物小传，无需二次加工
5. **外貌描述**：细节丰富，有画面感，能为演员提供参考

## 注意事项

1. 所有解读必须基于本地命盘库数据，不编造、不脑补
2. 严格区分男命、女命的不同解读逻辑
3. 命盘匹配结果需向用户展示并确认
4. 生成内容需可溯源至具体星曜格局
5. 面相描述要与性格、命运逻辑一致，相由心生
