# 即时指令系统

> 不用加载整个Skill，一个指令就能搞定。

---

## 核心指令（最常用）

### 给"帮我X"系列

#### 1. 潜台词化
```
输入：直白对白
输出：潜文本版本 + 修改说明

例：
输入："我很害怕失去你"
输出："你走的时候把那件外套带上。外面冷。"
```

#### 2. 加矛盾
```
输入：角色当前状态
输出：Want/Need冲突设计

例：
输入：一个善良的乡村医生
输出：
Want：救更多人
Need：接受自己不是神
冲突：救别人 → 救不了所有人 → 愧疚
```

#### 3. 补伏笔
```
输入：当前页数 + 已知信息
输出：伏笔设计 + 回扣位置

例：
P12提到"打火机"
→ P85回扣：点燃逃生信号
```

#### 4. 压缩场景
```
输入：原文场景
输出：压缩版（保留核心动作）

规则：
- 动作→3行
- 对白→功能句
- 删除解释性内容
```

## 快速生成指令

### 5个开头
```
触发："给我5个开头"
输出：5个不同风格的开场（紧张/悬疑/日常/动作/情绪）
```

### 3个结局
```
触发："3个结局"
输出：成功/失败/妥协三选
```

### 角色速建
```
触发："建角色"
输入：类型（农民/医生/警察...）
输出：Want/Need/Ghost/Lie/Flaw
```

### 对白抛光
```
触发："抛光"
输入：原文
输出：口语化+功能化版本
```

## 词库调用指令

### 场景词库
```
触发："写[场景]场景"
- 村口告别 → 场景06/18参考
- 打斗 → 场景07/19参考
- 亲吻 → 场景01/30参考
```

### 心理/动作
```
触发："写[角色]心理"
- 紧张 → 05参考
- 犹豫 → 17参考
- 爆发 → 07/19参考
```

—— 用具体回忆替代直白表达"

────────────
用户：角色太平，加矛盾

我：
"乡村医生李大夫
Want：让村民都能看病不花钱
