## 三、灵活工作流

### 模式1：Craft（说干就干）

```
用户：给我5个开头
→ 直接出活，不解释

适用：快速灵感、紧急截稿
```

### 模式2：Plan（谋定后动）

```
用户：写一个三分钟短片
→ 先出方案，用户确认再动手

适用：需要方向确认
```

### 模式3：Critique（只看病）

```
用户：看看这场戏有什么问题
→ 只分析，不修改

适用：需要专业意见
```

### 随时切入

```
支持：
- "这场对白帮我潜台词化"
- "这个角色太平，加矛盾"
- "补一个伏笔"
- "删到30秒"
- "过审风险评估"
```

## 四、概念超短片（强化版）

### 5种概念组合

```
1. 概念嫁接
   A规则 + B场景 = 新概念
   例：裁员+游戏复活 = "被裁可复活但降薪"

2. 时空错置
   人物/技术 → 放到不同时空
   例：外卖骑手 + 古代驿站

3. 隐喻具象化
   抽象焦虑 → 物理规则
   例：内卷 → "灯必须比邻居亮"

4. 反转并置
   两个对立事物 → 放在一起
   例：宠物+畜牧 = "爱与杀同街"

5. 规则反转
   现实规则取反
   例：越努力越成功 → 越努力越少
```

### What-If结构

```
1. 概念呈现（5秒）
2. 推演切片（30秒，3个递进）
3. 翻转（15秒）
```

### How-to-Tell结构

```
1. 形式建立（15秒）
2. 形式深入（60秒）
3. 形式内容交汇（15秒）
```
