# 查漏补缺：专业编剧补充

---

## 1. 剧本格式细节

### 标准格式模板

```
【场景编号：场景名 / 地点 / 时间】
【内景/外景】【日/夜/晨/昏/夜】

[动作描写，不超过3行]

角色A：xxx。（对白，不超30字/句）
角色B：xxx。

[反应/动作，1-2行]

角色A：xxx。
```

### 格式禁忌

```
❌ 禁止：
- 超过3行的动作描写
- ()内的演员调度指令
- 镜头角度指示（交给导演）
- 太过详细的心理描写
- 长段落的环境描写
```

### 场景编号规则

```
格式：INT/EXT.地点.时间
示例：
INT.村口.日
EXT.山谷.夜
INT.牢房.昏

连续夜戏：INT./EXT.连续夜
同一场景多日：牢房/第1天 / 第2天
```

## 2. 角色出场统计

### 开场10页必须建立的角色

```
✅ 主角：Want清晰
✅ 对手：动机清晰
✅ 镜像角色：B Plot相关
✅ 导师/帮助者：催化相关
```

### 每个角色必须有

```
1. 首次出场页数
2. 关键词（观众记住的特征）
3. 退出/死亡页数（如果领便当）
4. 功能说明（为什么存在）
```

### 角色弧光预算表

```
角色 | 初始状态 | 终点状态 | 变化页数 | 消耗幅度
----- | -------- | -------- | -------- | --------
主角  | 自卑    | 自信    | P25-P95 | 70页
```

## 3. 分集大纲（剧集补充）

### 集内结构（每集10-15分钟）

```
【集号】《集名》
【Logline】一句话本集核心
【时长】15分钟

【冷开场】（1分钟）
- 没有片头直接进画面
- 与上集钩子呼应或独立钩子

【建置】（3分钟）
- 本集核心冲突引入
- 主角want明确

【发展】（6分钟）
- 障碍1→障碍2
- B Plot推进

【高潮】（3分钟）
- 本集核心选择
- 集末钩子（为下集铺垫）

【结局】（2分钟）
- 解决或未解决
- 留下尾巴
```

### 集间钩子类型

```
1. 悬念型：关键信息不展示
2. 危机型：下集更危险
3. 揭示型：上集真相下集揭示
4. 回归型：下集回到上集
```

## 4. 闪回/非线性叙事

### 闪回使用规则

```
✅ 可用：
- 角色回忆（不超过2次/片）
- 关键证据揭示
- Ghost揭示

❌ 禁止：
- 连续闪回（拖节奏）
- 闪回解释剧情（用动作展现）
- 过多回忆片段
```

### 闪回格式

```
[闪回开始]
【场景X：回忆 / 心理时间】
...
[闪回结束]
【现实继续】
```

## 5. 旁白/画外音规则

### 何时可用

```
✅ 可用：
- 开头建立世界观（只1次）
- 隐藏叙述者视角
- 关键转折点信息

❌ 禁止：
- 解释角色行为
- 代替角色反应
- 过多使用（观众会烦）
```

### 旁白公式

```
"我叫XX，这是XX。我以为..."（开头世界观）
→ 只用一次，建立后不再用
```

## 6. 时间线控制

### 时间标记

```
现实时间：剧本时间 = 1:1
1页 ≈ 1分钟

时间跳跃标记：
【三个月后】
【十年后】
【第二天早晨】
```

### 时间节奏

```
压缩：跳跃式转场（淡入淡出）
延伸：长时间留在一个场景
定格：画面静止，时间停止感
```

## 7. 转场技巧

### 基本转场

```
硬切：直接切 → 紧张/快节奏
淡入淡出：舒缓/时间跳跃
叠化：回忆/平行时间
划入：章节分隔
```

### 转场格式

```
【黑屏】 ← → 【场景】
[淡入]     [淡出]
[切至]
```

### 场景内转场

```
同一场景：
- 角色走出画面 → 角��走进画面（时间过）
- 动作中途停止 → 动作继续（时间停）
```

## 8. 季终/集末设计

### 集末钩子（单集）

```
❌ 不要：
- 强行制造悬念
- 突然死亡/灾难

✅ 要：
- 自然的悬念
- 信息的保留
- 角色决定后的未知
```

### 季终设计（最终集）

```
季终=大高潮+解决+新钩子

公式：
1. 核心冲突解决（3选1）
   - 主角成功 / 主角失败 / 妥协

2. 角色新常态展示
   - 弧光完成后的变化

3. 新季钩子（如果有）
   - 引流向下季
   - 如果没有就完整结束
```

## 9. 营销点/预告片素材

### 预告片需要

```
1. 开场精彩画面（动作/悬念/笑点）
2. 核心冲突展示
3. 角色亮相（有记忆点）
4. 核心对白（1-2句）
5. 高潮片段（不剧透关键）
```

### 宣传口号格式

```
"一个X，必须Y，否则Z"
→ 类似这个格式，好传播
```

## 10. 常见专业问题

### 新手常犯

```
1. 对白太长 → 拆成短句
2. 动作太多 → 删到3行内
3. 心理描写 → 用动作表现
4. 场景太杂 → 合并同类项
5. 节奏太平 → 加障碍
6. 角色太多 → 删到6个内
7. 伏笔太长 → 缩短回扣周期
8. 转折太急 → 加过渡
```

### 剧本医生检查

```
先读Page 1 → 再读Page 25 → 再读Page 55 → 再读Page 85 → 最后读Page 110
快速判断：每25页有没有"事"发生？
```
