# 专业剧本创作模块

> 本模块为"中国民间故事写作Skill"的专业剧本创作核心。
> 以**Save the Cat节拍表**为骨架，融合**山音编剧方法论**。

---

## 一、专业剧本节拍表（Save the Cat 15节拍）

```
第一幕（第1-25页）
├── 1. 开场画面（Page 1）
│   - 影片第一印象：基调/情绪/类型/题材
│   - 建立主角的"旧世界"
│   - 观众的前10分钟关注点
│
├── 2. 主题呈现（Page 5）
│   - 第5页必须有核心主题暗示
│   - 通常由非主角说出
│   - "家庭比金钱重要"类命题
│
├── 3. 铺垫（Page 1-10）
│   - 建立主角和所有人物
│   - 展现主角的6个"缺失"（Want）
│   - 设定期望和情绪基调
│
├── 4. 催化剂（Page 12）
│   - 打破旧世界的事件
│   - 不可逆的改变
│   - "催化时刻"必须清晰
│
├── 5. 争执（Page 12-25）
│   - 主角最后一次"这太疯狂了"
│   - 我该去吗？敢去吗？
│   - 主角必须主动选择
│
└── 6. 第二幕衔接点（Page 25）
    - 主角做出决定进入新世界
    - 必须主动选择，不能被骗入

第二幕（第25-85页）
├── 7. B故事（Page 30）
│   - 通常是爱情线
│   - 承载主题的故事线
│   - 提供"切换镜头"
│
├── 8. 娱乐游戏（Page 30-55）
│   - "大前提约定"部分
│   - 海报/预告片的核心内容
│   - 主角在新世界试错
│
├── 9. 中点（Page 55）
│   - 剧本两半的界限
│   - 伪胜利 或 伪失败
│   - 赌注升级
│
├── 10. 坏人逼近（Page 55-75）
│   - 内外部压力同时收紧
│   - 主角的Flaw开始产生代价
│
├── 11. 一无所有（Page 75）
│   - 最低点
│   - 失去最重要的东西
│   - "死亡气息"
│
└── 12. 灵魂黑夜（Page 75-85）
    - 主角直面自己的Ghost/Lie
    - 弧光转变的临界点

第三幕（第85-110页）
├── 13. 进入第三幕（Page 85）
│   - 主角想通了
│   - 用新方法，不是旧方法升级
│   - A Plot和B Plot融合
│
├── 14. 尾声（Page 85-105）
│   - 五步高潮法
│   - 执行新计划
│
└── 15. 终场画面（Page 110）
    - 与开场对照
    - 展示"新常态"
    - 不解释，用画面
```

## 二、专业人物设计系统

### 完整人物链条

```
Ghost（前史创伤）→ Lie（相信的谎言）→ Flaw（性格缺陷）→ Want（外在追求）
                                                            ↓
                                                         Need（真正需要）
                                                            ↓
                                                      关键选择
                                                            ↓
                                                       新常态（弧光）
```

### Want vs Need 设计

| Want（外在） | Need（内在） | 冲突来源 |
|-------------|-------------|----------|
| 想要钱 | 想要被认可 | 用错误方式追求正确目标 |
| 想要救她 | 直面失去 | 救得了人救不了心 |
| 想要赢 | 接受输 | 赢本身不是终点 |

### Ghost设计

```
Ghost是角色过去的创伤事件，塑造了当前的Lie。

示例：
- Ghost：童年被遗弃 → Lie：不能相信任何人
- Ghost：母亲自杀 → Lie：亲密关系很危险
- Ghost：被嘲笑过 → Lie：成功才能被爱
```

### Flaw设计

```
Flaw = Lie在行为层面的外化

示例：
- Lie：不信任任何人 → Flaw：推开每个想靠近的人
- Lie：成功才被爱 → Flaw：工作狂忽略家庭
```

## 三、潜文本对白系统

### 潜文本三层

```
表层文本：角色说的话
|-----------|
层间含义：角色实际表达的意思（对话）
|-----------|
深层潜文本：角色真正隐瞒的真相（冰山）

好的对白 = 表层文本 + 层间含义 ≠ 深层潜文本（需要观众推断）
```

### 潜文本训练

**❌ 直白对白：**
> "我很害怕失去你。"
> "我 其实一直爱着你。"

**✅ 潜文本对白：**
> "你走的时候把那件外套带上。外面冷。"
> "最近忙，别来了。"

### 对白功能检查清单

每句对白必须有：
- [ ] 推动情节？（WHAT）
- [ ] 揭示性格？（WHO）
- [ ] 制造冲突？（CONFLICT）
- [ ] 角色辨识？（VOICE）

四者必有其一，否则删除。

## 四、视听语言系统

### 基础镜头语言

```
镜头类型：
- 远景（Establishing Shot）建立环境
- 全景（Wide Shot）人物与环境关系
- 中景（Medium Shot）对话场景
- 近景（Close-up）情绪细节
- 特写（Extreme Close-up）强调

运动方式：
- 推（Dolly In）  接近
- 拉（Dolly Out） 远离
- 摇（Pan）     水平移动
- 移（Track）   沿轴移动
- 升（Crab）    垂直移动
```

### 时长估算

```
1页标准剧本 ≈ 1分钟成片
中文字数：约200-250字/页

场景时长标注：
【场景3：医院走廊 / 夜】（约1分30秒）
情节节奏：中 | 情感节奏：重
```

### 场景节奏标注

```
外部情节节奏（剧情密度）：
- 紧：高密度/快剪辑/追逐/对峙
- 松：日常/独处/过渡/风景

内在情感节奏（角色情感）：
- 重：告白/失去/恐惧/狂喜
- 轻：调侃/闲聊/幽默
```

## 六、产品输出线

### 完整剧本产品

```
Level 1：Logline（一句话）
  "一个X，必须Y，否则Z。"

Level 2：Synopsis（200字）
  三幕结构的压缩版

Level 3：Treatment（1000字）
  完整的戏剧前提

Level 4：Character Bible
  所有角色的Want/Need/Ghost/Lie/Flaw

Level 5：Beat Sheet（30-60个情节点）
  完整的节拍表

Level 6：Scene Breakdown
  场景拆解表

Level 7：Full Script
  完整剧本（110页）
```

### 场景拆解表模板

```
场景# | 页数 | 地点 | 时间 | 人物 | 目的 | 核心动作 | 价值转变
------|------|------|------|------|------|--------|----------
1     | 1    | 村   | 日   | 主   | 建立 | 被嘲笑 | 自信→自卑
2     | 3    | 山   | 夜   | 主+仙 | 相逢 | 救狐   | 无→有
...   | ...  | ...  | ...  | ...  | ...  | ...    | ...
```

## 补充：遗漏信息（专业版2.0）

### 场景密度计算
- 110页 = 40-60个场景
- 第1-10页：6-8个（建置期高密度）
- 第31-55页：10-12个（娱乐游戏期）
- 第91-110页：4-6个（高潮+解决）

### 伏笔回扣系统
- 伏笔登记表：内容/首次位置/回扣位置/回扣方式
- 黄金法则：自然出现→必须回扣→"原来如此"

### B Plot交织
- Page 30：B Plot起点
- Page 55：A×B首次交汇
- Page 75：B Plot危机

### 对白控制
- 中文单句≤30字
- 每10页：1快+1慢节奏
- 同一角色3次内必须有反应

### 过审敏感词
- ❌ 涉黑、未成年犯罪、真实地名+敏感事件
- ❌ 警察被收买、邪教

### 快速输出（应急）
- Logline+开场+终场
- 核心角色Want/Need
- 5个关键beat
- 3场核心冲突
