# 八门谋生 - 完整创作工作流

基于奇门遁甲九宫八门体系的权谋商战故事创作流程。

---

## 工作流总览

```
用户提供方向 → 提示词生成 → 大纲规划 → 逐章生成 → 输出&图解
                    ↓              ↓           ↓
              奇门要素映射      章节结构      记忆更新
```

---

## 一、提示词生成（收到用户方向后）

### 第一步：题材定位

根据用户提供的方向，确定：

| 维度 | 选项 | 示例 |
|------|------|------|
| 主类型 | 权谋/商战/职场阴谋 | 商战 |
| 子类型 | 资本博弈/并购争夺/创业奋斗 | 资本博弈 |
| 风格 | 写实/黑色幽默/大男主/群像 | 黑色幽默 |
| 时代 | 当代/近现代 | 当代都市 |

### 第二步：奇门框架设定

根据题材，自动选择奇门要素：

**商战资本博弈 → 核心奇门**

```
宫位：6宫乾（权力）+ 2宫坤（资源）+ 7宫兑（谈判）
八门：生门（融资/扩张）+ 伤门（攻击）+ 开门（上市）
八神：值符（权威）+ 白虎（冲突）+ 玄武（间谍）
```

**职场阴谋 → 核心奇门**

```
宫位：5宫中（决策层）+ 4宫巽（渗透）
八门：杜门（保密）+ 惊门（心理战）+ 死门（绝境）
八神：腾蛇（阴谋）+ 太阴（暗箱）
```

### 第三步：主角人设

生成主角配置：

```markdown
## 主角配置

- 姓名：[默认：陈启明]
- 年龄：28-35岁
- 初始身份：[根据题材设定]
- 性格标签：[如：沉稳、腹黑、隐忍]
- 金手指：[如：奇门知识/重生记忆/神秘人脉]
- 初始状态：[如：被裁员/被架空/创业失败]
- 奇门定位：[如：1宫坎潜伏者→6宫乾权力巅峰]
```

### 第四步：核心冲突设计

**主线矛盾**：

```
[主角目标] → [阻碍势力] → [冲突升级] → [终极对决]
```

**前3章即时冲突**：

| 章节 | 冲突类型 | 奇门要素 |
|------|----------|----------|
| 第1章 | 起始困境 | 1宫坎/休门 |
| 第2章 | 发现机会 | 3宫震/生门 |
| 第3章 | 首次对抗 | 7宫兑/伤门 |

### 第五步：配角框架

自动生成基础配角：

| 角色类型 | 数量 | 奇门属性 | 示例 |
|----------|------|----------|------|
| 对手/反派 | 1-2人 | 白虎/伤门 | 资本对手 |
| 盟友/导师 | 1-2人 | 值符/生门 | 神秘投资人 |
| 红颜/合作 | 1-2人 | 六合/开门 | 投行高管 |
| 炮灰/牺牲 | 若干 | 死门 | 被牺牲的下属 |

### 第六步：开篇钩子

设计第1章吸引力：

```
钩子类型：[选择其一]
- 突发危机：主角被裁员/项目被砍
- 神秘相遇：神秘人物出现
- 情报获取：发现对手致命弱点
- 绝地反击：被逼到绝路后的反击
```

## 二、大纲规划

### 大纲模板

```markdown
# 《剧名》权谋线大纲

## 基本信息
- 题材：[商战/职场阴谋]
- 类型：[资本博弈/并购争夺]
- 预计集数：[12集]
- 每集字数：[3000-4000字]

## 权谋主线

### 核心矛盾
[一句话概括主线]

### 主角奇门定位
- 起始宫位：1宫坎（潜伏者）
- 目标宫位：6宫乾（权力巅峰）
- 成长路径：休门→生门→伤门→死门→惊门→开门

### 八门成长线
- 休门期（第1-2集）：蛰伏、等待、积累
- 生门期（第3-5集）：起步、融资、扩张
- 伤门期（第6-7集）：攻击、对决、危机
- 死门期（第8-9集）：低谷、绝境、反思
- 惊门期（第10-11集）：反击、揭露、逆转
- 开门期（第12集）：终极对决、权力巅峰

## 势力格局

### 主要势力
| 势力 | 核心人物 | 奇门属性 | 目标 |
|------|----------|----------|------|
| 主角方 | 陈启明 | 1宫→6宫 | 权力巅峰 |
| 对手方 | 赵鹏 | 6宫乾 | 维持垄断 |
| 第三方 | 林诗雨 | 2宫坤 | 利益最大化 |

### 势力演变
[每个阶段的势力变化]

## 关键转折点

### 第一次转折（第3-4集）
- 事件：[发现对手弱点]
- 奇门：3宫震（突袭）+ 生门（生机）
- 爽点：[智慧型反击]

### 第二次转折（第7集）
- 事件：[董事会发难]
- 奇门：5宫中（权力中心）+ 伤门（攻击）
- 危机：[内部背叛]

### 第三次转折（第8-9集）
- 事件：[绝地反击]
- 奇门：8宫艮（坚守）→ 4宫巽（渗透）
- 爽点：[置之死地而后生]

### 高潮（第12集）
- 事件：[终极对决]
- 奇门：6宫乾 + 9宫离（权力+舆论）
- 结果：[权力巅峰]

## 支线设计

### 商业线
- 天使轮融资（第4集）
- A轮谈判（第6集）
- 反向收购（第11集）

### 权力线
- 被架空（第1集）
- 安插内线（第3集）
- 董事会战争（第7集）

### 情感线
- 初次交锋（第2集）
- 利益绑定（第4集）
- 信任危机（第8集）

## 伏笔设计

| 伏笔 | 埋设 | 回收 |
|------|------|------|
| 神秘老人身份 | 第2集 | 第10集 |
| 早年案底 | 第3集 | 第9集 |
| 监听设备 | 第4集 | 第8集 |

## 三、逐章生成

### 生成前检查

1. 读取 `.learnings/` 记忆文件
2. 确认当前章节位置
3. 调取相关奇门要素

### 单章模板

```markdown
# 第[X]章：[章名]

## 基本信息
- 权谋阶段：[八门阶段]
- 奇门要素：[宫位+门+神]
- 字数目标：[3000-4000]

## 本章目标
- 主线推进：[...]
- 支线发展：[...]
- 爽点设计：[...]

## 情节点
### 场景1：[场景名]
- 地点：[...]
- 人物：[...]
- 事件：[...]
- 对话：[...]

### 场景2：[...]
...

## 本章金句/权谋感悟
[可选]
```

### 生成后处理

1. 保存到 `output/` 目录
2. 更新 `.learnings/` 记忆
3. 记录新的伏笔

---

## 四、记忆更新流程

### 每次生成后必须更新

#### 1. 更新 CHARACTERS.md

```markdown
## 角色状态追踪表（追加）

| 角色 | 当前状态 | 权势等级 | 位置 | 章节 |
|------|----------|----------|------|------|
```

#### 2. 更新 LOCATIONS.md

```markdown
## 地点转移记录（追加）

| 章节 | 主角位置 | 场景 |
```

#### 3. 更新 PLOT_POINTS.md

```markdown
## 主线剧情（追加）

| 序号 | 情节 | 章节 | 状态 |
```

## 五、质量检查清单

### 提示词阶段

- [ ] 主角有明确的逆袭起点
- [ ] 金手指有清晰规则和限制
- [ ] 前3章有"打脸"场景
- [ ] 奇门框架完整（宫+门+神）

### 大纲阶段

- [ ] 八门阶段划分清晰
- [ ] 每阶段有明确爽点
- [ ] 伏笔设计合理
- [ ] 配角奇门属性明确

### 章节阶段

- [ ] 本章奇门要素体现
- [ ] 对白有"权谋感"
- [ ] 情节推进合理
- [ ] 与前后章节连贯
