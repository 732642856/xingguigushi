## 一、经典商战案例库

### 案例1：宝万之争（2015-2017）

**类型**：恶意收购与反收购

| 要素 | 内容 |
|------|------|
| 收购方 | 宝能系（姚振华） |
| 被收购方 | 万科（王石团队） |
| 核心矛盾 | 资本与管理权的争夺 |


- 突然的资本攻势
- 董事会的内部矛盾

- 监管的角色

## 二、合同博弈场景库

### 场景1：融资条款谈判

**情境**：天使轮/VC谈判

**关键条款**：

| 条款 | 双方关注点 | 博弈点 |
|------|------------|--------|
| 估值 | 创始人的稀释比例 | 高估值vs低稀释 |
| 董事会席位 | 投资人的话语权 | 控制权vs发展 |
| 优先清算权 | 投资人的保障 | 退出保护 |
| 反稀释条款 | 创始人的保护 | 下一轮估值保护 |
| 竞业条款 | 双方约束 | 限制范围 |

**典型对白**：

```
投资人："估值可以谈，但你需要让出董事会席位。"
创始人："席位可以给，但我要求保留一票否决权。"
投资人："一票否决？这不合常规。"
创始人："这是我的底线。没有这个，我宁可不拿这笔钱。"
```

---

### 场景2：并购条款谈判

**情境**：收购方 vs 被收购方

**关键条款**：

| 条款 | 收购方 | 被收购方 |
|------|--------|----------|
| 交易结构 | 现金/股票/混合 | 最大化收益 |
| 对赌条款 | 业绩承诺 | 降低风险 |
| 员工安置 | 优化/裁员 | 保障 |
| 竞业限制 | 防止竞争 | 生存空间 |
| 品牌使用 | 保留/弃用 | 传承 |

**典型对白**：

```
收购方："我们可以用股票支付，估值按现在股价计算。"
被收购方："我需要现金对赌部分。我可以签对赌，但三年业绩目标要合理。"
收购方："这是行业标准。"
被收购方："我的团队值这个价。达不到，我可以退还。"
```

---

### 场景3：股东协议博弈

**情境**：创始团队 vs 投资人

**关键条款**：

| 条款 | 含义 | 博弈点 |
|------|------|--------|
| 优先认购权 | 新融资时的优先权 | 稀释保护 |
| 拖售权 | 投资人可以强制卖股 | 退出机制 |
| 回购权 | 投资人可以要求回购 | 退出保证 |
| 董事会观察员 | 投资人列席 | 信息权 |

**典型对白**：

```
投资人："我们要求拖售权，这样有利于我们退出。"
创始人："拖售权我可以同意，但必须经过董事会2/3同意，不能你们单方面决定。"
投资人："这..."
创始人："这是平衡。我们需要的是战略投资人，不是财务投机者。"
```

## 三、董事会斗争场景库

### 场景1：董事会突然发难

**情境**：CEO在董事会上被突然攻击

**典型流程**：

```
1. 议程突然变更（增加罢免议题）
2. 事先串通的董事发言
3. 外部董事（独立董事）倒戈
4. 董事长不表态（默认）
5. CEO被要求"暂时休息"
```

**典型对白**：

```
董事A："我提议增加一个议题，关于CEO的胜任问题。"
董事长："这个...大家有什么意见？"
董事B："我同意。季度业绩下滑严重，我认为需要有人负责。"
CEO："下滑是因为我们在转型期，这是预期内的。"
董事C："但投资人已经表示不满了。我认为应该让CEO休息一下。"
CEO："（沉默）......"
```

---

### 场景2：投票权争夺

**情境**：势均力敌的投票

**关键要素**：

- 委托投票征集
- 代理人博弈
- 临时动议
- 沉默的大多数

**典型对白**：

```
律师："现在投票。同意的请举手。"
（3只手举起）
律师："不同意的请举手。"
（3只手举起）
律师："弃权的请举手。"
（沉默）
律师："平票。按照公司章程，董事长有一票决定权。"
董事长："......我选择同意。"
```

---

### 场景3：引入外部势力

**情境**：一方引入外部投资者改变格局

**典型场景**：

```
1. 某董事悄悄接触外部资本
2. 外部资本暗中吸筹
3. 突然公开收购要约
4. 原有平衡打破
5. 重新洗牌
```

**典型对白**：

```
董事："我有一个朋友，对我们公司很感兴趣。"
CEO："什么样的朋友？"
董事："他想了解一下我们的情况。我觉得可以聊聊。"
CEO："这不在今天的议程里。"
董事："我提议明天开临时董事会讨论。"
（会后）
CEO："他们要进来了。"
```
