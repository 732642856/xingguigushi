# 权谋角色原型

## 一、权力核心型

### 1.1 幕后巨头型

**特征**：隐藏在背后，操控全局，不到关键时刻不露面。

**奇门定位**：1宫坎（潜伏）+ 九地（深藏）+ 值符（权威）

**台词风格**：云淡风轻中暗藏玄机，习惯用问题回答问题。

```
典型台词：
"你觉得，这盘棋，谁是棋子？"
"有些事，知道就好，不必说破。"
"我给你一个机会，但不是白给的。"
```

**Want**：维持控制权
**Need**：找到真正的继承人
**弧光**：从操控一切 → 放手让后人闯

---

### 1.2 权力巅峰型

**特征**：已经站在最高点，孤独求败，最大的敌人是自己。

**奇门定位**：6宫乾（权力巅峰）+ 开门（公开）+ 九天（高远）

**台词风格**：简短、有力、不容置疑。

```
典型台词：
"这个位置，不是谁都能坐的。"
"我已经赢了，现在只是时间问题。"
"你可以反对，但改变不了结果。"
```

**Want**：守住权力
**Need**：找回失去的人性
**弧光**：从权力巅峰 → 孤独的领悟

---

### 1.3 新贵崛起型

**特征**：从底层杀上来，不择手段，野心勃勃。

**奇门定位**：3宫震（突袭）+ 伤门（攻击）+ 白虎（冲突）

**台词风格**：直接、激进、有攻击性。

```
典型台词：
"我有的选吗？没有。"
"要么赢，要么滚，没有第三选择。"
"他们不让我进来，我就把门砸开。"
```

**Want**：获得权力
**Need**：学会取舍
**弧光**：从不计代价 → 学会选择

---

## 二、智谋型

### 2.1 阴谋家型

**特征**：擅长背后操作，借刀杀人，不留痕迹。

**奇门定位**：4宫巽（渗透）+ 杜门（保密）+ 腾蛇（缠绕）

**台词风格**：绵里藏针，话说一半。

```
典型台词：
"我什么都没做，只是提供了一些信息。"
"有些事，不需要亲自出手。"
"你看到的是我想让你看到的。"
```

**Want**：暗中掌控
**Need**：光明正大赢一次
**弧光**：从阴谋到阳谋

---

### 2.2 谋士型

**特征**：辅佐主公，运筹帷幄，出谋划策。

**奇门定位**：5宫中（核心决策）+ 天相星（辅佐）+ 六合（联盟）

**台词风格**：分析形势，提供方案，忠诚但有底线。

```
典型台词：
"属下有一计，可解此局。"
"主公三思，此举风险太大。"
"我可以帮你，但你要答应我一件事。"
```

**Want**：辅佐明主
**Need**：实现自己的理想
**弧光**：从辅佐到独立

---

### 2.3 破局者型

**特征**：在死局中找到生机，绝地反击。

**奇门定位**：8宫艮（底线）+ 死门（绝境）+ 惊门（逆转）

**台词风格**：冷静分析，不放弃任何希望。

```
典型台词：
"还没输，我们还有机会。"
"最坏的结果是什么？我来兜底。"
"他们以为我完了，但我还有最后一招。"
```

**Want**：打破困局
**Need**：接受不完美的胜利
**弧光**：从绝境逢生 → 学会放弃

---

## 三、情感型

### 3.1 重情重义型

**特征**：为了情义可以放弃权力，背后往往有故事。

**奇门定位**：2宫坤（承载）+ 太阴（情感）+ 九地（隐忍）

**台词风格**：真情流露，但关键时刻不含糊。

```
典型台词：
"我可以不要江山，但不能不要你。"
"当年他帮过我，今天我还他。"
"情义这两个字，不是谁都能承担的。"
```

**Want**：守护情义
**Need**：在权力和情义间找到平衡
**弧光**：从为情义 → 为自己活一次

---

### 3.2 冷血理性型

**特征**：感情是工具，理性到可怕。

**奇门定位**：7宫兑（谈判）+ 玄武（间谍）+ 太阴（暗箱）

**台词风格**：一切以利益出发，情感是可以计算的变量。

```
典型台词：
"感情是最不可靠的东西。"
"我不是没有感情，只是算得很清楚。"
"你对我来说，还有利用价值。"
```

**Want**：利益最大化
**Need**：体验真正的感情
**弧光**：从理性到学会感性

---

### 3.3 亦正亦邪型

**特征**：有底线但会用非常手段，灰色地带游走。

**奇门定位**：9宫离（舆论）+ 景门（公关）+ 值符（权威）

**台词风格**：表面正义，手段强硬。

```
典型台词：
"我做的事，见不得光，但结果是好的。"
"有些正义，要用不正义的方式实现。"
"我不怕被骂，我只看结果。"
```

**Want**：达成目标
**Need**：面对手段的代价
**弧光**：从只看结果 → 承担过程

---

## 四、成长型

### 4.1 复仇者型

**特征）：带着仇恨成长，最终面对仇恨的意义。

**奇门定位**：1宫坎→3宫震（潜伏→爆发）+ 死门（仇恨）

**台词风格**：前期隐忍，后期爆发。

```
典型台词：
"这口气，我咽了十年。"
"我等的，就是这一天。"
"仇恨给了我力量，但也困住了我。"
```

**Want**：复仇
**Need**：放下仇恨
**弧光）：从复仇 → 放过自己

---

### 4.2 背叛者型

**特征）：曾经背叛过，现在寻求救赎。

**奇门定位**：6宫乾→1宫坎（巅峰→坠落）+ 玄武（背叛）

**台词风格）：后悔、赎罪、想要弥补。

```
典型台词：
"当年是我对不起你。"
"我现在做的，都是在还债。"
"我不求你原谅，只求心安。"
```

**Want）：弥补过错
**Need**：真正被原谅
**弧光）：从赎罪 → 重新开始

---

### 4.3 卧底型

**特征）：双重身份，表面一套背后一套。

**奇门定位**：4宫巽（渗透）+ 杜门（隐藏）+ 玄武（间谍）

**台词风格）：真假难辨，关键时刻暴露。

```
典型台词：
"我的身份，只有我自己知道。"
"两边都是我的职责，两边都对不起。"
"总有一天，我要选择一边。"
```

**Want）：完成任务
**Need**：找到真正的归属
**弧光）：从摇摆 → 做出选择

---

## 五、反派型

### 5.1 纯粹恶人型

**特征）：坏得彻底，没有理由，就是坏。

**奇门定位**：7宫兑（冲突）+ 白虎（暴力）+ 死门（毁灭）

**台词风格）：残忍、享受快感、不解释。

```
典型台词：
"我不需要理由。"
"看着别人痛苦，是我的乐趣。"
"这个世界上，只有强弱，没有对错。"
```

**Want**：破坏一切
**Need**：无（纯粹的恶）
**弧光**：恶有恶报

---

### 5.2 可怜之人必有可恨之处型

**特征）：曾经受害者，变成加害者。

**奇门定位**：1宫坎（创伤）+ 死门（执念）+ 玄武（扭曲）

**台词风格）：有理有据的恶，让人不自觉同情。

```
典型台词：
"你们知道当年我经历了什么吗？"
"是这个社会先对不起我的。"
"我不是一开始就是这样 的。"
```

**Want**：讨回公道
**Need**：放下执念
**弧光）：从复仇 → 放下

---

### 5.3 体制内的恶型

**特征）：在体制内作恶，打着正义旗号行恶。

**奇门定位**：5宫中（权力核心）+ 值符（权威）+ 腾蛇（阴谋）

**台词风格）：道貌岸然，冠冕堂皇。

```
典型台词：
"这是规定，我也没有办法。"
"为了更大的利益，牺牲是必要的。"
"我只是在执行命令。"
```

**Want**：维护体制
**Need**：面对体制的恶
**弧光）：从维护 → 打破

---

## 六、喜剧/调剂型

### 6.1 傻白甜富二代型

**特征）：有钱人家的孩子，被保护得太好，不谙世事。

**奇门定位**：2宫坤（资源）+ 天同星（单纯）

**台词风格**：天真、率直、偶尔犯蠢。

```
典型台词：
"有钱了不起吗？不好意思，有钱真的了不起。"
"你们在说什么？我怎么听不懂。"
"我爸说，不懂就问，不要紧的。"
```

**Want**：证明自己
**Need**：接受真实社会的毒打
**弧光）：从天真 → 成长

---

### 6.2 搞笑担当型

**特征）：气氛组，调节紧张的权谋氛围。

**奇门定位**：任何宫位都可，主要功能是调剂

**台词风格**：幽默、吐槽、关键时刻不掉链子。

```
典型台词：
"不好意思，我走错片场了。"
"这也太复杂了，我瓜子都磕完了。"
"别紧张，我就随便看看。"
```

**Want**：融入圈子
**Need**：被认真对待
**弧光）：从搞笑 → 认真
