# 权谋场景模板库

## 一、会议博弈场景

### 1.1 董事会交锋型
```
场景：董事会上各方势力对峙
元素：投票权、利益交换、秘密武器
时长度：3-10分钟
奇门要素：5宫中 + 值符 + 六合
双轨节奏：情节紧+情感轻
```

### 1.2 谈判桌上型
```
场景：双方或多方谈判
元素：筹码、底线、试探、让步
时长度：5-15分钟
奇门要素：7宫兑 + 景门 + 太阴/玄武
双轨节奏：情节紧+情感轻
```

### 1.3 高层密谋型
```
场景：少数人秘密开会
元素：阴谋、交易、承诺
时长度：2-5分钟
奇门要素：1宫坎 + 杜门 + 腾蛇
双轨节奏：情节紧（暗涌）+情感轻
```
