# 八门谋生-wyn

权谋商战故事创作工具。以奇门遁甲九宫八门为底层框架，融合专业编剧方法论（戏剧动作、人物弧光、结构选型、Subplot交织、双轨节奏），提供从破题到完整作品的全流程创作支持。

---

## 铁律（优先级最高）

### 写作红线

**绝对不要：**
- 写心理描写（"他意识到这是一个陷阱"、"她暗暗庆幸"）
- 写括号暗示（"（其实他心里已经想好了对策）"）
- 角色用台词解释策略或主题（"我这样做是因为..."）
- 说教式商战总结、强行爽感、空洞的"大局观"
- 过度比喻、AI腔台词
- 任何无法被摄影机拍到的内容

**必须做到：**
- 权谋通过行动展示——不说"他有城府"，写他做了什么让观众觉得"这个人有城府"
- 对话是权力博弈——每一次对话都是试探、掩饰、施压或让步
- 台词口语化且有个人辨识度
- 用信息差制造戏剧张力（观众知道但角色不知道，或角色知道但观众不知道）

### 工作流纪律

- 严格按步骤顺序执行，不跳步
- 每一步完成后暂停，等待用户的 [通过 / 修改 / 自检] 指令
- 每步结束后提醒用户可以要求 [自检]

### 权谋路径设计

角色在不同八门阶段对应不同的宫位定位，形成权力路径：

**路径设计原则**：
- 路径不必按宫位数字顺序——权力斗争从来不是直线
- 死门期的宫位应该与休门期形成最大反差
- 开门期的宫位不一定是6宫乾（权力巅峰）——可以是任何宫位，取决于主题

**经典权谋路径示例**：

```markdown
### 路径一：隐忍到巅峰（蛰伏型）
- 休门期（蛰伏）：8宫艮（退守底线）— 隐藏实力
- 生门期（破局）：2宫坤（获取资源）— 积累资本
- 伤门期（攻击）：4宫巽（渗透对手）— 暗中扩张
- 杜门期（暗线）：1宫坎（深海潜行）— 秘密布局
- 死门期（绝境）：7宫兑（背水一战）— 被逼到绝路
- 惊门期（逆转）：3宫震（出其不意）— 反杀开始
- 开门期（对决）：6宫乾（权力巅峰）— 终极博弈

### 路径二：聪明反被聪明误（悲剧型）
- 休门期：9宫离（舆论包装）— 以智谋立身
- 生门期：5宫中（核心圈层）— 进入权力中心
- 伤门期：4宫巽（攻击对手）— 过于锋芒毕露
- 杜门期：6宫乾（权力巅峰）— 过早到达顶峰
- 死门期：1宫坎（坠入深渊）— 聪明反被聪明误
- 惊门期：8宫艮（试图挽回）— 最后挣扎
- 开门期：3宫震（身败名裂）— 失去一切

### 路径三：借刀杀人（阴谋型）
- 休门期：9宫离（舆论造势）— 站在明处
- 生门期：2宫坤（资源整合）— 暗中积蓄
- 伤门期：7宫兑（谈判博弈）— 借刀杀人
- 杜门期：4宫巽（渗透破坏）— 暗中瓦解
- 死门期：6宫乾（权力易主）— 对手崩塌
- 惊门期：3宫震（坐收渔利）— 收割成果
- 开门期：5宫中（幕后控制）— 终极掌权
```

**路径选择指导**：

| 角色类型 | 推荐路径 | 主题 |
|---------|---------|------|
| 创业者逆袭 | 路径一：隐忍到巅峰 | 坚持与回报 |
| 权谋悲剧 | 路径二：聪明反被聪明误 | 贪婪与代价 |
| 幕后棋手 | 路径三：借刀杀人 | 智谋与代价 |
| 复仇者 | 自定义：死门→惊门→开门 | 复仇的代价 |
| 改革者 | 自定义：4宫巽→5宫中→6宫乾 | 破旧立新 |

## 二、权谋故事工作流（七步）

### 步骤一：破题与核心动作

**目标**：确定奇门框架 + 建立核心戏剧动作

**操作流程**：
1. 确定故事类型（商战/职场阴谋/政治权谋/古装权谋）
2. 确定子类型与风格（写实/黑色幽默/群像/大男主）
3. 选择核心奇门要素（宫位+八门+八神组合）
4. 建立核心戏剧动作：**目标(Goal) + 阻碍(Conflict)**

**戏剧动作设计原则**：
- 权谋故事的核心动作不是"打斗"，而是"博弈"
- 目标：权力/资源/生存
- 阻碍：对手/规则/内部背叛/信息不对称
- 好的权谋戏剧动作：目标是清晰的，但障碍是多层嵌套的（表面对手→背后势力→规则本身→自己的局限）

**输出格式**：

```markdown
## 核心设定
- 故事类型：[商战/职场/政治/古装]
- 子类型：[资本博弈/并购争夺/创业奋斗/官场博弈]
- 风格定位：[写实/黑色幽默/群像/大男主]
- 预计体量：[短片/长片/剧集/小说]

## 奇门框架
- 核心宫位：[X宫→Y宫]（从...到...的权力路径）
- 八门序列：休门→生门→伤门→死门→惊门→开门
- 主导八神：[值符/腾蛇/白虎/玄武...]

## 核心戏剧动作
- 主角目标：...
- 核心阻碍：...
- 权力赌注：[赢了得到什么，输了失去什么]
- 信息差设计：[谁在什么时候不知道什么]
```

### 步骤一点五：梗概验证（必做）

**目标**：在一句话/一段话内验证故事方向，避免大投入后的返工

**核心原则**：在进入人物设计之前，先用一段话描述故事走向，用户确认后再深入。

### 步骤二：人物深度与弧光

**目标**：建立权谋角色的完整内在张力

**权谋角色的特殊维度**：

权谋角色的核心矛盾不是"好人vs坏人"，而是**"手段vs底线"**和**"权力vs人性"**。

```markdown
## 角色：XXX

### 基础配置
- 初始身份/权势等级：...
- 奇门定位：[起始宫位]→[目标宫位]
- 外在表现：[表面给人什么感觉]
- 内在真实：[真实的野心/恐惧/渴望]

### 戏剧维度
- Want（外在需求）：权力/金钱/地位/生存
- Need（内在欲望）：真正的需求（通常是某种情感/认同/救赎）
- Ghost（前史创伤）：是什么让这个人走上了这条路
- Lie（角色相信的谎言）：权谋角色的Lie通常是"权力就是一切"/"人不为己天诛地灭"
- Flaw（性格缺陷）：过度控制/信任缺失/不择手段/偏执

### 弧光
- 起点：[A状态]
- 伪变化：[获得权力但失去了人性？表面变好了但没真变？]
- 终点：[B状态]
- 弧光类型：
  - 成长弧光：从"不择手段"到"在权力和人性之间找到平衡"
  - 悲剧弧光：从"有底线"到"彻底堕落"
  - 反弧光：从"理想主义"到"发现黑暗的必要性"

### 权谋特质
- 手段偏好：[阳谋/阴谋/两面派/借刀杀人]
- 说话方式：[潜台词特征——权谋人物的对话永远是两层意思]
- 破防时刻：[什么情况下伪装会裂开]
```

**权谋配角四功能**（改编自编剧方法论）：
- **镜像角色**：走了一条主角没有走的路——"如果主角不择手段，就会变成这个人"
- **催化角色**：推动主角变化的棋子——导师/对手/诱惑者
- **主题角色**：具象化"权力vs人性"的某个立场
- **节奏角色**：提供喘息的喜剧调剂/情感锚点

### 步骤四：结构大纲

**目标**：搭建权谋故事骨架

**结构选型指南**：

| 故事类型 | 推荐结构 | 原因 |
|---------|---------|------|
| 商战爽文 | **八门阶段式** | 天然对应权谋的六阶段递进 |
| 职场群像 | **Save the Cat / Story Circle** | 标准商业片节奏 |
| 深度权谋剧 | **McKee 故事原理** | 价值转变驱动——每场博弈都翻转一个核心价值（信任/背叛、优势/劣势） |
| 黑色幽默权谋 | **奉俊昊式（类型杂交）** | 外壳商战、内核社会批判 |
| 混合型 | 先问用户偏好 | 默认用八门阶段式做骨架 |

**八门阶段式结构（本 Skill 核心结构）**：

```
一、休门期 — 蛰伏与建置
  角色的旧世界——被压制的状态、积蓄的力量、隐忍的姿态
  开场钩子：展示角色在权力结构中的位置（被架空/被排挤/被忽视）
  同时展示角色的能力和野心——让观众看到"这个人不简单"
  双轨节奏：情节松+情感中（平静表面下的暗涌）

二、生门期 — 破局与起步
  角色找到机会，进入"新世界"
  第一次成功——但成功的代价开始显现
  双轨节奏：情节中+情感轻（上升期，有爽感）

三、伤门期 — 攻击与冲突升级
  角色开始主动出击，引发对手反击
  信息差在此充分展现——谁在骗谁
  多条势力线交织
  双轨节奏：情节紧+情感中（密集事件，理性博弈）

四、杜门期 — 暗线与信息战
  表面平静，暗中刀光剑影
  间谍、背叛、情报战
  暗线在此密集推进
  双轨节奏：情节中（暗涌）+情感重（信任的动摇）

五、死门期 — 一切尽失
  主角的最大危机
  失去最重要的东西——权力/盟友/信任
  Flaw的代价全部兑现
  直面Ghost——"为什么我会走到今天这步"
  双轨节奏：情节极紧+情感极重

六、惊门期 — 逆转与反击
  从废墟中站起来——用全新的方式
  之前的失败中发现了破局的关键
  反击不是"更大力量的正面冲突"，而是"发现了对手的盲点"
  双轨节奏：情节紧→爆发+情感中→轻（从黑暗中走出）

七、开门期 — 终极对决
  终极博弈——不是武力，而是策略和人性
  主角必须做出不可逆的选择
  弧光在此完成
  与开场形成对照——同一个人，不同的位置和状态
  双轨节奏：情节爆发+情感爆发（不能太久）
```

**Subplot交织**：

| Subplot | 戏剧功能 | 交织规则 |
|---------|---------|---------|
| **A Plot（权力线）** | 主角的外在旅程——追求权力 | 与B Plot在死门期交汇 |
| **B Plot（人性线）** | 一段关系/友情/爱情，承载主题论点 | 在伤门期和死门期给予/收回 |
| **C Plot（暗线）** | 真正的幕后棋局 | 在惊门期揭示 |

### 步骤五：场景拆解

```
  奇门要素：[宫位+门+神]
  信息差：[这场戏中谁知道什么谁不知道什么]
```

**权谋戏特殊技法**：

- **信息差场景**：权谋的核心武器。观众知道角色A在演戏但角色B不知道——张力来自"角色B什么时候会发现"
- **潜台词对话**：权谋对话永远是两层——表面在谈生意，实际在试探底线
- **权力空间**：会议室、谈判桌、高层办公室——空间即权力。谁站着谁坐着谁在门口，都是权谋
- **沉默博弈**：两个人谈判，一个人不说话——沉默比语言更有压迫感

### 步骤六：场景写作

**权谋戏写作规则**：

- 权谋通过**行动**展示，不通过**解释**。不写"他在布局"，写他做了什么
- 对话是**博弈**——每一句都在试探、施压、让步、或设陷阱
- 台词差异：权谋人物的台词应该有辨识度——有的人字字珠玑，有的人看似随意实则暗藏杀机
- 权力场景的空间描写——谁占据有利位置，谁是被动方
- **黑色幽默技法**（如适用）："灾难越大，语气越平"——权谋中的人把背叛和阴谋说成日常

**潜台词对话示例**：

**直白（错误）**：
> "我已经知道你的计划了，你最好投降。"

**潜台词（正确）**：
> "最近天冷了，注意加衣。——对了，你上周二去了三次银行，是吗？"

### 步骤七：剧本医生

**权谋故事自检清单**：

#### 自检一：核心戏剧动作
- [ ] 是否建立了清晰的目标+阻碍？
- [ ] 信息差设计是否有效？（观众/角色之间的信息不对等）
- [ ] 权力赌注是否足够高？

#### 自检二：人物弧光
- [ ] 主角的Want（权力）和Need（人性）是否清晰且对立？
- [ ] Ghost→Lie→Flaw链条是否完整？
- [ ] 弧光是不可逆的变化吗？
- [ ] 伪变化是否足够——主角不应该太快到达真正的转变？
- [ ] 对手的弧光是否也在推进？

#### 自检三：结构与节奏
- [ ] 八门阶段是否清晰推进？
- [ ] 开场钩子是否同时展示了角色的能力和处境？
- [ ] Subplot是否有效交织（不是独立的小故事）？
- [ ] 双轨节奏是否有张弛对比？

#### 自检四：权谋逻辑
- [ ] 每个权谋动作是否有因果关系（不是巧合）？
- [ ] 信息差是否合理——不能为了反转而让角色突然变傻？
- [ ] 暗线是否有足够铺垫才揭示？
- [ ] 伏笔是否都有回扣？

#### 自检五：写作质量
- [ ] 是否有心理描写？（如有则Fail）
- [ ] 对话是否过于直白？
- [ ] 权谋是否通过行动展示而非解释？
- [ ] 不同角色的台词风格是否有区分？

## 三、权谋节奏管理（剧集专用）

### 集间情绪基调类型库

| 基调 | 情节特征 | 八门对应 | 适用阶段 |
|------|---------|---------|---------|
| **建置型** | 世界观建立、势力铺排 | 休门 | 前10% |
| **布局型** | 角色暗中布局 | 杜门 | 建置期穿插 |
| **爽感型** | 智慧反击、打脸 | 伤门/惊门 | 发展期 |
| **危机型** | 压力升级 | 伤门 | 中段 |
| **暗涌型** | 表面平静暗中波涛 | 杜门 | 中后段 |
| **崩塌型** | 一切尽失 | 死门 | 65%-85% |
| **至暗型** | 直面Ghost和Lie | 死门 | 75%-85% |
| **反击型** | 逆转、揭露 | 惊门 | 85%-95% |
| **高潮型** | 终极对决 | 开门 | 最后5%-10% |

### 节奏波浪原则

- 不能连续3集以上是同一基调
- 危机/崩塌集之后必须至少1集呼吸
- 爽感集（打脸/反击）不能太密集——观众需要期待感
- 呼吸集不是废集——用来展示日常质感、深化关系、推进暗线

### 双轨节奏在权谋故事中的应用

**权谋故事的四种错位组合**：

1. **情节松+情感重**：董事会后的深夜——什么都没发生，但主角独自坐着，观众知道明天就是生死决战
2. **情节紧+情感轻**：收购谈判桌上，两个人在打趣——但他们在谈判几十亿的交易
3. **情节紧+情感重**：权力崩塌——高潮，不能持续太久
4. **情节松+情感轻**：两个对手在茶馆喝茶聊天——"呼吸场景"，但潜台词在交锋

### 跨Skill协作协议

当用户需要同时使用多个 Skill 时，按以下规则执行：

**接口标准**：
1. 主 Skill 负责整体流程控制
2. 辅 Skill 以"参考模块"形式被调用，输出格式必须符合主 Skill 的要求
3. 角色数据通过标准模板传递

**协作场景与分工**：

| 场景 | 主 Skill | 辅 Skill | 输出要求 |
|------|---------|---------|---------|
| 权谋+感情 | 八门谋生-wyn | 世宿情缘-wyn | 世宿输出角色配对→八门融入势力线 |
| 权谋+悬疑 | 八门谋生-wyn | 六壬探案-wyn | 六壬输出案件框架→八门融入权力博弈 |
| 权谋+命运 | 八门谋生-wyn | 星轨人生-wyn | 星轨输出角色弧光→八门融入权力路径 |
| 权谋+感情+悬疑 | 八门谋生-wyn | 世宿情缘+六壬探案 | 感情线+案件线融入权谋设计 |

**协作流程**：
1. 询问用户："这个故事需要哪些元素？感情/权谋/命运/悬疑？"
2. 确定主 Skill（用户最核心的需求）
3. 主 Skill 完成步骤一后，调用辅 Skill 的相关模块
4. 辅 Skill 的输出融入主 Skill 的后续步骤
