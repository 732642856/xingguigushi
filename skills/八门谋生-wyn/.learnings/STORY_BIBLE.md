## 权谋商战世界观

基于奇门遁甲体系，构建现代都市权谋/商战故事的背景设定。

## 二、社会规则

### 现代都市权谋法则

1. **利益最大化原则**：所有决策以利益为导向
2. **信息差优势**：掌握信息就是掌握权力
3. **资源整合能力**：谁能整合更多资源，谁就有话语权
4. **人设与真实**：表面一套，背后一套是常态
5. **风险与收益**：高风险高收益，但要先有资本
6. **联盟与背叛**：没有永远的敌人，没有永远的朋友

### 权力结构

```
顶层（Lv6-7）
    ↓ 资源掌控
中层（Lv3-5）
    ↓ 技能/位置
底层（Lv1-2）
    ↓ 努力/机遇
晋升通道
```

## 三、货币/资源体系

### 商业通用资源

| 资源类型 | 描述 | 获取难度 |
|----------|------|----------|
| 资金 | 现金、股票、可变现资产 | 中 |
| 人脉 | 关键人物、渠道、信息源 | 高 |
| 技能 | 专业能力、行业知识 | 低 |
| 品牌 | 个人/公司声誉、口碑 | 高 |
| 情报 | 商业机密、市场信息 | 极高 |
| 权力 | 决策权、审批权、投票权 | 高 |

### 资源换算参考

```
1个核心人脉 ≈ 100万信息价值
1个关键情报 ≈ 50万-500万商业价值
1个决策权位 ≈ 1000万资源调配权
1次顶级联盟 ≈ 改变行业格局
```

## 四、金手指设计（主角特殊能力）

### 类型与示例

| 类型 | 示例 | 限制 | 成长性 |
|------|------|------|--------|
| **知识优势** | 穿越者/重生者，知道未来趋势 | 不能直接改变已发生的事 | 随经验积累 |
| **神秘传承** | 奇门遁甲秘术、神秘老人传授 | 需要悟性/时机激活 | 逐步解锁 |
| **特殊人脉** | 隐藏身份的神秘盟友 | 盟友可能有自己的目的 | 需要信任建立 |
| **信息渠道** | 独家情报网、黑客能力 | 信息不一定准确 | 需要维护 |
| **系统辅助** | 商业分析系统、决策辅助AI | 需要数据输入 | 随数据进化 |

### 设计要点

1. **有边界**：不是全能，有专业领域限制
2. **可升级**：随着剧情发展，能力增强
3. **有代价**：使用能力需要付出代价（时间、人情、风险）
4. **伏笔关联**：与主线剧情紧密相关
5. **平衡性**：不能过于bug，否则失去戏剧张力

## 五、核心势力设定

### 势力类型

| 势力类型 | 特征 | 奇门属性 | 典型代表 |
|----------|------|----------|----------|
| **资本派** | 以资金为核心，追逐利润 | 6宫乾+2宫坤 | 金融集团、投资机构 |
| **技术派** | 以技术为核心，壁垒防御 | 4宫巽+杜门 | 科技公司、专利巨头 |
| **人脉派** | 以关系为核心，资源整合 | 7宫兑+六合 | 政商掮客、联盟核心 |
| **舆论派** | 以媒体为核心，话语权 | 9宫离+景门 | 传媒集团、公关公司 |
| **隐藏派** | 以隐秘为核心，幕后操控 | 1宫坎+太阴 | 神秘组织、隐世家族 |

### 势力关系矩阵

```
          资本派    技术派    人脉派    舆论派    隐藏派
资本派     -       合作      博弈      投资      秘密合作
技术派    合作       -      依赖      警惕      未知
人脉派    博弈      依赖      -       操控      联盟
舆论派    投资      警惕     操控       -      利用
隐藏派    秘密合作  未知     联盟     利用       -
```

## 六、世界观模板示例

### 设定：现代商业都市

```
【时代背景】
2020年代，商业竞争白热化，传统行业转型，互联网巨头格局已定

【核心冲突】
新人崛起 vs 既得利益者
技术创新 vs 资本垄断
个人理想 vs 商业规则

【潜规则】
- 表面公平竞争，实际资源为王
- 合同只是底线，谈判桌下才是真相
- 上市是里程碑，不是终点
- 创始人可能只是一枚棋子

【势力划分】
┌─────────────────────────────────────┐
│           顶层：资本巨头              │
│     （Lv6-7：顶级投资人、控股家族）    │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│           中层：行业领军              │
│    （Lv4-5：CEO、连续创业者）        │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│           底层：上升通道              │
│    （Lv2-3：职业经理人、创业者）      │
└─────────────────────────────────────┘
```

## 七、奇门要素与世界观结合

### 九宫地域设定

| 5宫中 | 总部/董事会 | 权力中心、决策之地 | 高层对决 |
| 6宫乾 | 顶级会所/控股公司 | 权力巅峰、幕后操控 | 终极博弈 |

| 门 | 剧情阶段 | 典型情节 |
|----|----------|----------|
