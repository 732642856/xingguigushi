## 七、场景转换技巧

### 从低权谋区到高权谋区

```
城中村（1宫坎/休门） → 咖啡馆（7宫兑） → 会所（4宫巽/太阴） → 董事会（5宫中）
节奏：平静 → 接触 → 深入 → 决战
```

### 从公开到私密

```
公开场合（9宫离/景门） → 半公开（7宫兑） → 私密（1宫坎/4宫巽）
用途：制造舆论 → 初步接触 → 达成交易
```
