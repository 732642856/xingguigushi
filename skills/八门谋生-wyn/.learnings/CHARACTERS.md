# 八门谋生 - 角色档案模板

## 权谋商战角色体系

基于奇门遁甲九宫系统，为权谋/商战故事提供角色原型。

---

## 一、主角模板（九宫定位）

### 核心配置

| 字段 | 内容 | 示例 |
|------|------|------|
| 姓名 | 主角名称 | 陈启明 |
| 年龄 | 故事开始时的年龄 | 32岁 |
| 初始身份 | 职业/地位 | 某科技公司部门经理 |
| 性格 | 主要性格特点 | 沉稳、腹黑、隐忍 |
| 金手指 | 特殊能力/资源 | 奇门遁甲知识 + 神秘老人传承 |
| 初始状态 | 故事开始时的处境 | 被架空、被裁员边缘 |
| 九宫定位 | 奇门宫位 | 1宫坎（潜伏者）/ 6宫乾（权力追求者）|
| 八门属性 | 核心特质 | 休门（蛰伏）/ 生门（生机）|

### 主角成长线

```
初始（1宫坎-蛰伏）→ 第一阶段突破（3宫震-突袭）→ 危机（4宫巽-渗透失败）
→ 重大转折（5宫中-核心决策）→ 最终形态（6宫乾-权力巅峰 / 9宫离-舆论制高点）
```

### 奇门角色定位表

| 宫位 | 角色类型 | 适合主角定位 | 典型命运 |
|------|----------|--------------|----------|
| 1宫坎 | 潜伏者/密谋者 | 卧底、复仇者、幕后推手 | 长期隐忍，一朝翻盘 |
| 2宫坤 | 资源掌控者 | 资金大亨、联盟核心 | 掌控资源，但被觊觎 |
| 3宫震 | 突袭者/变盘者 | 突然发难、打破格局 | 破局者，但树敌多 |
| 4宫巽 | 渗透者/谣言制造者 | 商业间谍、舆论操控 | 暗中操控，但易暴露 |
| 5宫中 | 核心决策者 | CEO、董事会核心 | 权力中心，也是靶心 |
| 6宫乾 | 权力巅峰者 | 董事长、控股人 | 最终boss或成功者 |
| 7宫兑 | 谈判者/合约高手 | 律师、并购专家 | 成也合约，败也合约 |
| 8宫艮 | 底线坚守者 | 创始人、理想主义者 | 坚守底线，但被边缘化 |
| 9宫离 | 舆论掌控者 | 媒体大亨、公关天才 | 掌控舆论，但被反噬 |

## 六、角色出场记录

### 第1章

- 陈启明：部门经理，被架空
- 赵鹏：CEO，宣布裁员
- 林诗雨：猎头，第一次接触

### 第2章

- 神秘老人：神秘投资人，主动接触
- 小王：被裁员工，陈的追随者

### 第3章

- 陈启明：辞职，开始创业
- 赵鹏：启动对陈的封杀

### 第4章

- 林诗雨：决定投资陈的项目
- 神秘老人：提供第一个关键情报

### 第5章

- 陈启明：完成天使轮融资
- 赵鹏：发现陈的威胁，开始针对性攻击

## 八、角色弧光模板

### 盟友→敌人 弧光

```
阶段1：初始信任 → 阶段2：利益冲突 → 阶段3：试探 → 阶段4：决裂 → 阶段5：对抗
典型案例：小王被赵鹏收买，背叛陈启明
```

### 敌人→盟友 弧光

```
阶段1：初始对立 → 阶段2：意外合作 → 阶段3：利益绑定 → 阶段4：信任建立 → 阶段5：并肩
典型案例：林诗雨从对手变投资人
```

### 导师→对手 弧光

```
阶段1：传授知识 → 阶段2：暗中利用 → 阶段3：露出端倪 → 阶段4：真相揭露 → 阶段5：终极对决
典型案例：神秘老人实际是陈家灭门案的元凶
```
