## 三、四课 → 故事结构层次

| 四课 | 含义 | 故事映射 | 叙事功能 |
|------|------|----------|----------|
| **第一课**（日支上） | 对方/事件的表象 | 故事对手展现给外界看的样子 | 建立表象 |
| **第二课**（日支之阴） | 对方/事件的真相 | 对手隐藏的真实目的 | 揭示真相 |
| **第三课**（日干上） | 当事人的表象 | 主角展现给外界看的样子 | 主角面向 |
| **第四课**（日干之阴） | 当事人的内心 | 主角内心真实的想法和困境 | 内心动机 |

### 上下神关系

| 关系 | 故事映射 |
|------|----------|
| 上克下 | 表面控制实际，外部压力导致内在变化 |
| 下克上 | 内在驱动外在，暗中策划改变现状 |
| 上生下 | 外对内的支持与滋养 |
| 下生上 | 内对外的支持和推动 |
| 比和 | 内外一致、状态稳定 |

## 四、三传 → 情节发展阶段

### 基本映射

| 三传 | 传统含义 | 故事映射 | 占断八门 |
|------|----------|----------|----------|
| **初传** | 事情发端、起因 | 故事的引爆点、第一幕 | 发端门 |
| **中传** | 事情发展、转折 | 故事的转折点、第二幕 | 移易门 |
| **末传** | 事情结局、归着 | 故事的结局、第三幕 | 归计门 |

### 特殊格局 → 情节设计

| 格局 | 条件 | 故事映射 |
|------|------|----------|
| **三合局** | 三传地支成三合 | 多方势力合谋、力量汇聚 |
| **反吟** | 三传地支两两对冲 | 剧情大反转、事态突变 |
| **伏吟** | 三传地支各归本位 | 陷入僵局、停滞不动 |
| **退连茹** | 三传地支逆数相连 | 一波三折、功亏一篑 |
| **进连茹** | 三传地支顺数相连 | 逐步揭开真相、循序渐进 |
| **铸印** | 初中传为戌巳 | 权力获得、掌权时刻 |
| **斫轮** | 初中传为卯戌 | 破旧立新、彻底改变 |

## 七、实战：盘面 → 故事大纲

### 示例盘面

```
日干：寅（主角）
时干：子（案件）
三传：巳→午→未
天将：贵人、白虎、太阴
地支：子、丑、申
```

### 故事化输出

```markdown
【主角】
- 类型：青龙型（权威领导者）
- 性格：坚韧、正统、有领导力
- 设定：警局局长亲自督办

【案件】
- 类型：密室凶杀
- 核心：高档场所（丑）发生命案
- 难点：证据指向不明

【嫌疑人】
1. 白虎型（凶险）→ 可能的杀手
2. 太阴型（阴谋）→ 幕后黑手
3. 螣蛇型（诡异）→ 关键证人

【场景设计】
- 案发现场：高档场所（丑）
- 调查转折：道路拦截（申）
- 高潮：阴暗地下室（酉）

【氛围】
- 主基调：太阴（隐秘阴谋）
- 风格：冷硬派 + 心理博弈

【情节阶段】
- 初传（巳）：案件发生，嫌疑人出现
- 中传（午）：调查深入，迷雾重重
- 末传（未）：真相大白，结局
```
