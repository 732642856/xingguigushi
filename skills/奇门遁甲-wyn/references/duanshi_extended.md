# 张岩客奇门断事扩展指南

> 本文档扩展SKILL.md的断事维度，涵盖健康、学业、出行、官司、失物等常见预测方向。
> 核心原则：仍以值使门为主线，结合各事专用用神。

## 一、健康疾病

### 1.1 用神取用
| 要素 | 代表 | 说明 |
|-----|------|------|
| **天芮星** | 病星、病灶 | 核心用神，看天芮落宫 |
| **日干** | 病人（求测者） | 看日干状态 |
| **时干** | 病情发展 | 看时干生克 |
| **死门** | 死亡、重病 | 凶门，看是否临日干 |
| **白虎** | 伤灾、手术 | 凶神，主血光 |

### 1.2 断病要点

**Step 1：看天芮星**
- 天芮落宫 = 病灶所在部位/脏腑
- 天芮生助日干 → 病轻，易治
- 天芮克制日干 → 病重，难治
- 天芮落空亡 → 病有假象，或已好转

**Step 2：看日干状态**
- 日干旺相 → 身体底子好，抗病力强
- 日干休囚 → 身体虚弱，需调养
- 临死门/白虎 → 病情凶险

**Step 3：看病位对应**
| 天芮落宫 | 对应脏腑/部位 |
|---------|--------------|
| 1宫（坎） | 肾、膀胱、泌尿、耳 |
| 2宫（坤） | 脾胃、腹部、肌肉 |
| 3宫（震） | 肝、胆、足、神经 |
| 4宫（巽） | 胆、股、气管、风疾 |
| 5宫（中） | 脾胃、心腹（寄坤） |
| 6宫（乾） | 肺、大肠、头、骨骼 |
| 7宫（兑） | 肺、口、齿、咽喉 |
| 8宫（艮） | 胃、手、鼻、背 |
| 9宫（离） | 心、小肠、眼、血 |

### 1.3 应期与转归
- 病愈应期：天芮被制、或出空、或日干转旺之时
- 凶险信号：天芮+死门+白虎同宫，且克日干
- 转归判断：值使门生助日干则吉，克制则凶

### 1.4 输出结构（健康）
```
【健康断盘】

病灶定位：天芮星落X宫（X宫对应XX脏腑/部位）
病情判断：
- 天芮与日干关系：...
- 日干状态：...
- 吉凶门神：...

病症描述：...
转归预测：...
应期：...
建议：...
```

## 二、学业考试

### 2.1 用神取用
| 要素 | 代表 | 说明 |
|-----|------|------|
| **天辅星** | 文曲星、学业 | 核心用神 |
| **天任星** | 努力、勤奋 | 辅助用神 |
| **日干** | 考生本人 | 看状态 |
| **时干** | 考试/成绩 | 看结果 |
| **值符** | 考官、录取方 | 看是否赏识 |
| **开门** | 录取、公开 | 看能否录取 |

### 2.2 断学业要点

**Step 1：看天辅星**
- 天辅旺相 + 生助日干 → 学业顺利，成绩好
- 天辅休囚 + 克制日干 → 学业受阻，需努力
- 天辅落空亡 → 考试发挥失常，或有意外

**Step 2：看日干与时干**
- 日干生时干 → 考生努力，但结果一般
- 时干生日干 → 考试有利，成绩理想
- 日干克时干 → 能克服困难，考取

**Step 3：看值符与开门**
- 值符生日干 → 考官赏识，面试顺利
- 开门生日干 → 录取有望
- 开门临日干宫 → 必被录取

### 2.3 考试类型细分
| 考试类型 | 重点看 | 说明 |
|---------|-------|------|
| 升学考试 | 天辅+开门 | 文曲+录取 |
| 资格考试 | 天任+开门 | 努力+通过 |
| 面试选拔 | 值符+日干 | 考官+考生 |
| 出国留学 | 天辅+马星 | 学业+动象 |

### 2.4 输出结构（学业）
```
【学业断盘】

学业基础：天辅星状态...
考试结果：时干与日干关系...
录取可能：值符、开门状态...
应期：...
建议：...
```

## 三、出行远行

### 3.1 用神取用
| 要素 | 代表 | 说明 |
|-----|------|------|
| **马星** | 动、出行 | 核心标志 |
| **日干** | 出行人 | 看状态 |
| **时干** | 目的地/行程 | 看吉凶 |
| **景门** | 道路、信息 | 看路途 |
| **九天** | 远行、高飞 | 吉神，利远行 |
| **九地** | 潜伏、停留 | 利静不利动 |

### 3.2 断出行要点

**Step 1：看马星**
- 马星临日干宫 → 必出行，且速度快
- 马星生助日干 → 出行顺利
- 马星空亡 → 出行有变，可能延期

**Step 2：看景门（道路）**
- 景门旺相 + 生日干 → 道路通畅
- 景门休囚 + 克日干 → 路途不顺
- 景门+白虎 → 道路有险，注意安全

**Step 3：看方向**
- 日干克时干 → 去之有利
- 时干克日干 → 去之不利
- 时干生日干 → 目的地有利

### 3.3 出行吉凶判断
| 组合 | 吉凶 | 说明 |
|-----|------|------|
| 马星+九天 | 大吉 | 远行顺利，高飞远举 |
| 马星+白虎 | 凶 | 出行有险，注意伤灾 |
| 马星+空亡 | 平 | 出行有变，需备选方案 |
| 九地+日干 | 不吉 | 利静不利动，宜推迟 |

### 3.4 输出结构（出行）
```
【出行断盘】

出行时机：马星状态...
路途情况：景门状态...
目的地吉凶：时干与日干关系...
最佳方向：...
注意事项：...
应期：...
```

## 四、官司诉讼

### 4.1 用神取用
| 要素 | 代表 | 说明 |
|-----|------|------|
| **惊门** | 官司、口舌 | 核心用神 |
| **开门** | 法官、裁决 | 官方、公正 |
| **值符** | 原告、主动方 | 看哪方有利 |
| **天乙**（值符对宫） | 被告、被动方 | 看哪方有利 |
| **日干** | 求测者 | 看所处位置 |
| **时干** | 官司结果 | 看胜负 |

### 4.2 断官司要点

**Step 1：确定原被告**
- 日干临值符/生惊门 → 原告有利
- 日干临天乙/惊门克 → 被告有利

**Step 2：看惊门状态**
- 惊门旺相 → 官司激烈，影响大
- 惊门休囚 → 官司影响小，易和解
- 惊门落空亡 → 官司不了了之

**Step 3：看开门（法官）**
- 开门生日干 → 法官偏向求测者
- 开门克日干 → 法官不利求测者
- 开门生值符 → 原告胜
- 开门生天乙 → 被告胜

### 4.3 官司结果判断
| 情况 | 结果 | 说明 |
|-----|------|------|
| 值符生天乙，开门生天乙 | 被告胜 | 原告理亏 |
| 天乙生值符，开门生值符 | 原告胜 | 被告理亏 |
| 惊门落空亡 | 和解/撤诉 | 官司不成立 |
| 开门落空亡 | 悬而未决 | 法官难断 |

### 4.4 输出结构（官司）
```
【官司断盘】

原被告定位：...
官司激烈程度：惊门状态...
法官态度：开门状态...
胜负预测：...
应期：...
建议：...
```

## 五、失物寻找

### 5.1 用神取用
| 要素 | 代表 | 说明 |
|-----|------|------|
| **时干** | 失物 | 核心用神 |
| **日干** | 失主 | 看能否找回 |
| **杜门** | 隐藏、闭塞 | 看失物状态 |
| **天乙**（值符对宫） | 拾获者 | 看能否归还 |

### 5.2 断失物要点

**Step 1：看时干（失物）**
- 时干旺相 → 失物完好
- 时干休囚 → 失物受损
- 时干落空亡 → 难找回

**Step 2：看日干与时干**
- 日干克时干 → 能找回
- 时干克日干 → 难找回
- 时干生日干 → 有人送还

**Step 3：看杜门**
- 杜门临时干 → 失物被藏，难发现
- 杜门空亡 → 失物暴露，易发现

**Step 4：看方位**
- 时干落宫 = 失物所在方位

| 4宫 | 东南 | 草木处、花园、书房 |
| 5宫 | 中 | 家中中心、客厅 |
| 6宫 | 西北 | 高处、金属旁、寺庙 |

## 六、综合断事优先级

当用户同时问多件事时，按以下优先级处理：

1. **第一优先**：用户明确指定的事
2. **第二优先**：值使门所主之事（看值使门落宫五行）
3. **第三优先**：日干所临之事

### 各事五行归类
| 五行 | 对应事项 |
|-----|---------|
| 木 | 事业、升迁、学业 |
| 火 | 名声、文书、考试 |
| 土 | 房产、田产、财运 |
| 金 | 官司、武职、交通 |
| 水 | 出行、流动、桃花 |
