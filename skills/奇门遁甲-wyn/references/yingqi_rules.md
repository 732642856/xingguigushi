# 张岩客奇门应期推算规则（师传口径）

> 本文档记录张岩客师传应期推算方法，区别于传统"值使门落宫"法。
> 核心：六甲、三合、六合原理
> 状态：待吴老师补充具体例题后完善算法细节

## 一、应期推算核心原则

### 1.1 与传统方法的区别
| 方法 | 传统奇门 | 张岩客师传 |
|-----|---------|-----------|
| 核心依据 | 值使门落宫数 | 六甲、三合、六合 |
| 时间单位 | 以宫位定天数/月数 | 以干支关系定应期 |
| 精确度 | 模糊区间 | 可精确到日/时 |

### 1.2 张岩客应期三要素
1. **六甲** - 甲子、甲戌、甲申、甲午、甲辰、甲寅
2. **三合** - 申子辰、寅午戌、巳酉丑、亥卯未
3. **六合** - 子丑、寅亥、卯戌、辰酉、巳申、午未

## 二、六甲应期法

### 2.1 六甲定位
六甲为天干之首，代表"事之始、机之发"。

| 六甲 | 地支 | 五行 | 应期特征 |
|-----|------|------|---------|
| 甲子 | 子水 | 水 | 主开始、萌发、潜藏 |
| 甲戌 | 戌土 | 土 | 主收藏、结束、归藏 |
| 甲申 | 申金 | 金 | 主变革、突破、肃杀 |
| 甲午 | 午火 | 火 | 主鼎盛、光明、显露 |
| 甲辰 | 辰土 | 土 | 主蓄积、转化、酝酿 |
| 甲寅 | 寅木 | 木 | 主生发、启动、扩张 |

### 2.2 六甲应期推算步骤

**Step 1：确定用事干支**
- 以日干为"我"
- 以时干为"事"
- 以值使门所落宫的天盘干为"机"

**Step 2：找六甲关联**
- 查看盘中六甲所在宫位
- 看六甲与用事干支的生克关系

**Step 3：定应期**
- 若六甲生助用事干支 → 应期在六甲当值之时
- 若六甲克制用事干支 → 应期在六甲被制之时
- 若六甲与用事干支比和 → 应期在六甲出现之时

## 三、三合应期法

### 3.1 三合局表
| 三合局 | 地支 | 五行 | 应事类型 |
|-------|------|------|---------|
| 申子辰 | 水局 | 水 | 流动、变化、智慧、财源 |
| 寅午戌 | 火局 | 火 | 光明、名声、快速、冲突 |
| 巳酉丑 | 金局 | 金 | 收获、决断、肃杀、变革 |
| 亥卯未 | 木局 | 木 | 生长、合作、扩展、文化 |

### 3.2 三合应期推算

**原则**：
- 用事干支落入三合局中的某一地支 → 应期在该三合局成局之时
- 三合局成局 = 三个地支在盘中同时出现，或流月/流日走到该地支

**推算步骤**：
1. 确定日干/时干/值使门所在宫的地支
2. 查看该地支属于哪个三合局
3. 观察三合局中其他两个地支的位置
4. 应期在"三合全"的时间点

## 四、六合应期法

### 4.1 六合配对表
| 六合 | 配对 | 五行关系 | 应事特征 |
|-----|------|---------|---------|
| 子丑合 | 水+土 | 克合 | 合作中有制约，暗中交易 |
| 寅亥合 | 木+水 | 生合 | 贵人相助，暗中支持 |
| 卯戌合 | 木+土 | 克合 | 表面合作，内有矛盾 |
| 辰酉合 | 土+金 | 生合 | 金玉良缘，正式结盟 |
| 巳申合 | 火+金 | 克合 | 冲突中合作，博弈 |
| 午未合 | 火+土 | 生合 | 光明正大，公开合作 |

### 4.2 六合应期推算

**原则**：
- 六合主"合和、成事、姻缘、合作"
- 应期在"六合成"的时间点

**推算步骤**：
1. 确定用神所在宫的地支
2. 找出该地支的六合配对地支
3. 观察配对地支在盘中的位置
4. 应期在两支"相遇"之时

**"相遇"的定义**：
- 流月/流日走到配对地支
- 或配对地支被冲动（如子午冲，午日冲动子）

## 五、综合应期判断流程

### 5.1 优先级
1. **第一优先**：六甲（事之发端）
2. **第二优先**：三合（事之发展）
3. **第三优先**：六合（事之成就）

### 5.2 综合判断步骤

```
┌─────────────────────────────────────────┐
│  Step 1: 读取九宫信息                    │
│  - 日干、时干、值使门位置                │
│  - 六甲位置                             │
│  - 各宫地支                             │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  Step 2: 六甲分析                        │
│  - 哪个六甲与用事相关？                  │
│  - 生克关系如何？                        │
│  - 初步应期：X甲当值之时                 │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  Step 3: 三合验证                        │
│  - 用事地支入哪个三合局？                │
│  - 三合何时成局？                        │
│  - 验证/修正应期                         │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│  Step 4: 六合确认                        │
│  - 是否有合和之象？                      │
│  - 六合何时成就？                        │
│  - 最终应期确定                          │
└─────────────────────────────────────────┘
```

### 5.3 应期表达规范

**信息充足时**：
```
应期：甲子日（公历202X年X月X日）
依据：
- 六甲：甲子当值，生助日干
- 三合：申子辰水局成，利于...
- 六合：子丑合，贵人出现
```

**信息不足时**：
```
应期窗口：
- 最早：甲子日（X月X日前后）
- 最可能：申月申日（三合水局）
- 触发条件：遇到属龙或属猴的贵人
```

## 六、特殊应期情况

### 6.1 空亡中的应期
- 用神落空亡 → 应期在"出空"之时
- 出空 = 冲空（如空亡在戌，辰日冲之）或填空（戌日填之）

### 6.2 马星中的应期
- 马星主"动、速、变"
- 用神临马星 → 应期快，在马星当值之时
- 马星被冲 → 事情加速

### 6.3 伏吟/反吟（张岩客口径）
- 张岩客师传：六甲时辰**不出现**大伏吟
- 若遇伏吟象 → 事有反复，应期延后
- 若遇反吟象 → 事有突变，应期提前
