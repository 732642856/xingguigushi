# App Store 元数据模板

## 中文模板

### 应用名称（30字符内）
```
{App名} - {核心功能}
```

### 副标题（30字符内）
```
{卖点1} · {卖点2} · {卖点3}
```

### 关键词（100字符）
```
关键词1,关键词2,关键词3,关键词4,关键词5,关键词6,关键词7,关键词8,关键词9,关键词10
```

### 应用描述（4000字符内）
```
【一句话介绍】

【核心功能】
• 功能1：详细描述
• 功能2：详细描述
• 功能3：详细描述

【为什么选择我们】
• 优势1
• 优势2
• 优势3

【适合人群】
• 人群1
• 人群2

【联系我们】
如有问题或建议，请联系：
• 邮箱：xxx@example.com
• 网站：https://example.com
```

### 新功能说明
```
版本 {X.X.X}

✨ 新功能
• 新增功能A
• 新增功能B

🔧 优化
• 优化了性能
• 改进了界面

🐛 修复
• 修复了问题C
• 修复了问题D
```

## English Template

### App Name (30 chars)
```
{AppName} - {CoreFeature}
```

### Subtitle (30 chars)
```
{Benefit1} · {Benefit2} · {Benefit3}
```

### Keywords (100 chars)
```
keyword1,keyword2,keyword3,keyword4,keyword5,keyword6,keyword7,keyword8,keyword9,keyword10
```

### Description (4000 chars)
```
[One-line intro]

[Key Features]
• Feature 1: Description
• Feature 2: Description
• Feature 3: Description

[Why Choose Us]
• Benefit 1
• Benefit 2
• Benefit 3

[Who It's For]
• User type 1
• User type 2

[Contact Us]
For questions or suggestions:
• Email: xxx@example.com
• Website: https://example.com
```

Version {X.X.X}

✨ New
• Added feature A
• Added feature B

• UI improvements
