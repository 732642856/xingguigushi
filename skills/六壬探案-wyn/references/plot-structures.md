# 叙事结构参考手册

适配都市悬疑 + 黑色幽默 + 家族恩怨类型，融合三幕剧 / 英雄之旅 / 韩国编剧6段式大纲法。

---

## 一、两种结构模型的取舍

### 模型A：三幕剧（适合强情节、悬疑推进型）

适合：刑侦主线强、案件驱动、每集有明确破案进展的剧

### 模型B：英雄之旅（适合人物弧光驱动型）

适合：主角从困境中成长、情感线重于案件线的剧

### 模型C（推荐）：三幕 + 韩国6段式混合

```
三幕剧的外壳（建制→对抗→解决）
  +
韩国编剧的"受力分析"内核（每一场戏都有外部事件×内部状态）
  +
四线交叉结构（主线/支线A/支线B/伏笔线）
```

---

## 二、四线叙事结构

### 四线分配

```
【主线】刑侦/解谜 —— 推动故事的第一引擎
【支线A】家族恩怨/历史秘密 —— 揭示主题深度
【支线B】人物情感/关系 —— 提供情感落点
【伏笔线】民间奇术/历史谜团 —— 在大结局引爆

四线关系：
不是简单的"先走主线再走支线"，
而是像DNA双螺旋一样缠绕推进。
每一集，三到四条线都要有进展。
```

### 四线时间分配（以12集为例）

| 阶段 | 集数 | 主线 | 支线A | 支线B | 伏笔线 |
|---|---|---|---|---|---|
| 建局 | 1-2集 | 30% | 30% | 30% | 10% |
| 铺垫 | 3-4集 | 40% | 25% | 20% | 15% |
| 深化 | 5-7集 | 35% | 30% | 20% | 15% |
| 中点反转 | 8集 | 50% | 20% | 15% | 15% |
| 高潮 | 9-10集 | 30% | 30% | 20% | 20% |
| 落幕 | 11-12集 | 25% | 25% | 25% | 25% |

## 三、12集剧本结构（详细节拍）

### 第1集：建局

**核心任务**：用极小的异常戳中角色的内在创伤

```
【建制】
- 建立主角的日常（他的"底色"）
- 通过一个极小的事件，暴露他的"后遗症"/创伤反应
- 建立三条线的起点

【节拍】
00:00-05:00  主角日常出场（底色建立）
05:00-10:00  一个小事件触发创伤反应
10:00-20:00  主线开始（出现第一个案件/谜团）
20:00-30:00  支线A开始（家族/历史线铺垫）
30:00-40:00  支线B开始（情感线铺垫）
40:00-50:00  三线出现第一个交织点
50:00-60:00  第一集钩子（必须是一个让人想继续看的"不对"）
```

### 第2集：纠缠

**核心任务**：主角被动卷入，三条线各自向下延伸

### 第3-4集：铺垫

**核心任务**：建立所有主要人物，三线开始出现交集

```
关键：第3集结尾 或 第4集开头，
必须出现一个小高潮——让观众确认"这个剧不水"
```

### 第5-7集：深化

**核心任务**：主角开始主动行动，支线开始揭示深度

### 第8集：中点反转（最关键的一集）

```
这一集的反转，必须死死击中主角在第1集就建立起来的深层创伤。

不是"突然发生了意想不到的事故"——
而是"他终于被迫面对了自己一直在逃避的东西"。

中点反转测试：
把第8集的核心事件单独拎出来问——
这个事件，是否让主角的内在创伤再次被戳中？
如果答案是否，这一集需要重写。
```

### 第9-10集：高潮

**核心任务**：所有线索收束，真相浮出水面，代价开始显现

### 第11-12集：代价与落幕

```
⚠️ 注意：
这是最容易写崩的两集。
核心陷阱：
- 为了大团圆强行和解
- 为了"爽"强行反转
- 为了"深刻"强行悲剧

✅ 正确方向：
"带着伤疤继续生活的从容"

参考：《机智的医生生活》结尾——
没有高谈阔论的宏大升华，
只是几个经历了生死考验的中年人，
下班后聚在办公室里抢着吃一锅热腾腾的泡面。
```

## 四、单集内部节拍（每15分钟一个爆点）

### 45分钟/集（台剧标准）

```
【前5分钟】黄金开场
任务：完成"这一集的核心悬念是什么"的建立
禁忌：前5分钟是日常生活流水账

【5-15分钟】第一波冲突/推进
第一爆点（10分钟左右）

【15-25分钟】第二波深化/转折
第二爆点（20分钟左右）

【25-35分钟】第三波危机/揭示
第三爆点（30分钟左右）

【35-40分钟】缓冲（温情/幽默/喘息）
让观众消化前面35分钟的密度

【40-45分钟】本集钩子
留下下一集的悬念
```

### 黑色幽默风格的缓冲技巧

```
缓冲≠注水，缓冲段落的写法：

✅ 正确：
主线推进到紧张点，突然切到配角在吃饭——
这个配角的存在本身是荒诞的（他完全不知道主线正在发生什么），
他的日常和主角的惊险形成反差，产生黑色幽默效果。

❌ 错误：
在紧张场面的间隙插入一段"角色A谈感想"——
这是注水，不是缓冲。
```

## 五、幕间过渡技巧

### 从第一幕到第二幕（第一个转折点）

```
❌ 常见错误：
"于是他决定去追查真相。"

✅ 正确写法：
他不想追查。
但如果不追查，他连现在这口饭都吃不上。
——转折不是来自外部压力，而是来自生存本能的推动。

韩国编剧的做法：
第一幕结尾的转折，必须切断角色"退回日常"的可能性。
让他没有退路。
```

### 从第二幕到第三幕（第二个转折点）

```
这一次的转折，是角色从"被动承受"转向"主动选择"的节点。

关键问题：
在第二幕结尾之前，角色是：
- 在别人的推动下行动（被动）
- 还是有了一个真正属于自己的"选择"（主动）？

如果他一直在被动承受，
第三幕的转折点就是让他"Own这个选择"——
不是别人让他做，是他"我要做"。
```

## 七、"受力分析"实操案例

### 示范：第3集大纲

```markdown
# 《阴门铃医》第3集大纲

## 基本信息
- 核心事件：阿河用遥视能力看到第一名死者生前最后一个画面

---

## 【外部事件】
阿河跟随父亲到受害者家中"驱邪"，
意外触发遥视能力，看到了死者生前最后五分钟的画面——
一个蒙着面具的人，从背后接近他。

## 【内部状态】
阿河第一次主动"使用"了他的能力，不是被动触发。
他开始意识到：这个能力可以找到真相。
但他也开始恐惧：真相意味着什么。

## 【受力分析】
之前，阿河一直把能力当作诅咒——
因为它带来的都是他不想看到的东西。
但这一次，他发现了能力的另一面：
它不仅仅是诅咒，它也可以是"答案"。
这个认知击中了原来的创伤：
过去他一直在逃避这个能力，
现在他面临选择——用还是不用？
```

## 八、高潮设计原则

### 终极问题

> 你的主角在故事结尾，必须回答一个问题。
> 这个问题从第一集就埋在那里。
> 大结局，就是他给出答案的时刻。

```
示例：《无证之罪》骆闻
终极问题：当法律无法给你正义，你是否有权自行执法？
答案：骆闻用死亡给出了他的答案。
（不是"是"，也不是"否"——是更复杂的灰色。）

示例：《隐秘的角落》朱朝阳
终极问题：你会为了维持你想要的"好孩子"形象而牺牲别人吗？
答案：他一步一步yes，最终失去了所有亲人。
（开放结局：他变成了另一个张东升。）
```

### 大结局三忌

```
❌ 忌大团圆：每个人都活着/和解了
❌ 忌强行反转：所有人都死/都坏
❌ 忌主题升华：结尾突然来一段旁白总结"这个故事告诉我们……"
```
