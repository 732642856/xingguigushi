# 反侦察技法库

为悬疑推理故事提供犯罪者的反侦察手段，增加案件复杂度。

---

## 一、时间类反侦察

### 1.1 时间证人制造
```
手段：制造不在场证明
- 故意在公共场所露脸（视频/目击）
- 伪造消费记录/通行记录
- 使用他人身份证件
- 调快/调慢手表时间，制造时间差

创作提示：真正的反侦察不是完美的不在场证明，而是"差一点"的证明
```

### 1.2 时间混淆
```
手段：干扰警方时间判断
- 故意破坏受害者手表/手机时间
- 误导警方对死亡时间的判断（冷藏/冷冻）
- 利用时区/跨地区制造混乱

创作提示：时间混淆要合理，不能过于巧合
```

### 1.3 时间拖延
```
手段：延缓尸体发现
- 故意制造受害者"还活着"的假象
- 将尸体藏于难以发现处
- 利用地理因素延长发现时间

创作提示：拖延发现不等于掩盖真相
```

## 二、空间类反侦察

### 2.1 密室制造
```
手段：制造不可能现场
- 利用现场结构（通风管道、窗户）
- 制造"伪密室"（先进入后锁门）
- 使用外部工具（长杆、绳索）

创作提示：密室不需要"不可能"，只需要"被忽视"
```

### 2.2 现场伪造
```
手段：伪造犯罪现场
- 伪造自杀现场
- 伪造事故现场
- 伪造第二现场（移尸）

创作提示：伪造现场总会有痕迹，细节决定成败
```

### 2.3 轨迹消除
```
手段：消除来去痕迹
- 戴手套、不留指纹
- 清除足迹、轮胎印
- 处理现场视频（删除/覆盖）

创作提示：过度清除本身就是可疑痕迹
```

---

## 三、证据类反侦察

### 3.1 证据转移
```
手段：转移关键证据
- 将证据放入他人场所（嫁祸）
- 销毁证据（焚烧、腐蚀）
- 藏匿证据（多年后被发现）

创作提示：转移的证据最终会成为铁证
```

### 3.2 证据污染
```
手段：干扰证据有效性
- 在现场留下误导性证据
- 故意制造混乱干扰取证
- 利用警方内部人员

创作提示：污染证据需要"自己人"或"巧合"
```

### 3.3 证据伪造
```
手段：制造假证据
- 伪造文书、合同
- 制造伪证（书信、录音）
- 故意留下"证据"

创作提示：假证据最终会被揭穿，伏笔可以埋很远
```

## 四、心理类反侦察

### 4.1 误导警方画像
```
手段：故意提供错误特征
- 伪装外貌（妆容、假发、身高）
- 改变行为模式（故意用不习惯的手）
- 误导声音（变声、模仿）

创作提示：误导需要"真实感"，越刻意越容易被识破
```

### 4.2 制造不在场性格
```
手段：预先建立"不可能犯罪"的人设
- 表现得懦弱/胆怯
- 制造完美公民形象
- 预先设置"不可能"

创作提示：性格转变是最大的可疑点
```

### 4.3 利用证人心理
```
手段：操控证人证词
- 提前与证人"对台词"
- 利用证人的情感偏见
- 制造证人的"认知盲区"

创作提示：证人的"选择性记忆"是最难破解的
```

## 五、身份类反侦察

### 5.1 身份伪造
```
手段：使用虚假身份
- 假身份证、假学历
- 冒名顶替
- 多重身份（间谍）

创作提示：身份越复杂，破绽越多
```

### 5.2 角色扮演
```
手段：扮演不同角色接近目标
- 伪装成服务人员
- 伪装成受害者家属
- 长期潜伏

创作提示：扮演需要"沉浸式"，越像越难自拔
```

### 5.3 双重身份
```
手段：白天的身份与夜晚的身份
- 表面正经，暗地犯罪
- 白天是好人，晚上是另一面
- 利用身份便利犯罪

创作提示：双重身份是"日常"与"异常"的对比
```

## 六、团伙类反侦察

### 6.1 缄默规则
```
手段：团伙之间不说
- 提前警告
- 利益绑定
- 恐惧压制

创作提示：缄默需要"共同利益"，破除需要"分化"
```

### 6.2 分工合作
```
手段：每人只知一部分
- 单线联系
- 分段执行
- 信息隔离

创作提示：分工越细，破绽越多在"接口"处
```

### 6.3 弃车保帅
```
手段：牺牲一人保护全体
- 主动顶罪
- 伪造证据
- 利益交换

创作提示："弃车"时往往有"不舍"，细节会出卖
```

---

## 七、技术类反侦察

### 7.1 网络匿名
```
手段：利用网络技术隐藏
- VPN、代理IP
- 加密通讯
- 删除浏览记录

创作提示：技术手段越高级，留下的"数字痕迹"越明显
```

### 7.2 通讯干扰
```
手段：干扰警方追踪
- 临时手机、一次性手机
- 心理暗示（"别打这个号码"）
- 巧妙利用"巧合"

创作提示：通讯干扰需要"计划性"，巧合太多就是计划
```

### 7.3 物理手段
```
手段：物理干扰追踪
- 屏蔽器
- 干扰GPS
- 物理破坏追踪器

创作提示：物理手段往往有"痕迹"，如拆除痕迹
```

---

## 八、高端反侦察（慎用）

### 8.1 利用警方内部人员
```
手段：警局有内鬼
- 提前得知行动
- 关键证据消失
- 案件调查被干扰

创作提示：内鬼是"双刃剑"，一旦暴露就是死穴
```

### 8.2 利用媒体舆论
```
手段：引导舆论影响司法
- 故意制造舆论
- 利用媒体压力
- 误导公众判断

创作提示：舆论是把"双刃剑"，反转时更致命
```

### 8.3 利用"正义"外衣
```
手段：以正义之名犯罪
- 私家正义
- 替天行道
- 执法过当

创作提示：外衣越正义，暴露时越讽刺
```
