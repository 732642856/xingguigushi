# 韩剧编剧核心方法论

整合自韩国编剧内部资料（2026年4月收录），适配都市悬疑 + 黑色幽默 + 家族题材的创作框架。

---

## 一、底层哲学：视为人意

### 核心原则

| 陷阱（人为事） | 正确（视为人意） |
|---|---|
| 设定好情节，让角色去执行 | 所有情节都是人物性格的延伸 |
| 为了推进情节让角色"降智" | 角色的每一个选择都必须符合其性格逻辑 |
| 反派负责坏，主角负责正确 | 每个角色都有自己的欲望和创伤 |

### 实操判断标准

写每一场戏之前，问自己：
> 这个角色在这里做的选择，是**他本人**在这个情境下会做的选择，还是"剧情需要"他做的选择？

如果不是前者，这一场戏需要重写。

## 二、人物构建三步法：底色 → 创伤 → 欲望

### 第一步：底色（出场设置）

**禁止**：用形容词写底色（"他是一个善良的普通人"）
**要求**：用**具体的可感知的行为细节**写底色

```
❌ 错误示范：
"他是一个善良的便利店店员，总是乐于助人。"

✅ 正确示范：
他是一家便利店的夜班店员。
不用问，他就会把常客最爱喝的那款咖啡递过去。
每天下班前，他把快过期的便当留给后巷的流浪猫。
他说话带点乡下口音。
遇到任何争执，他的第一反应永远是先低头道歉。
```

**底色 = 观众能在生活中找到对应人的真实感**

### 第二步：创伤（击碎底色的那一下）

**要求**：
1. 创伤必须**彻底击碎**刚才建立的底色
2. 必须留下**不可逆的、可在日常中反复发作**的后遗症
3. 创伤必须被**具象化为一个生理习惯或物理动作**

```
✅ 优秀示范：《黑暗荣耀》文东恩
创伤事件：高中时被电烫棒大面积烫伤
后遗症：即使在隆冬，也习惯用冰冷的水洗手
——这个微小的日常动作，直接外化了她的皮肉与心理双重创伤
冰水能缓解潜意识里那种被灼烧的痛楚
```

```
❌ 常见错误：
创伤只停留在台词里（"那段经历让我很痛苦"）
或者创伤太轻/太模糊，观众感受不到
```

### 第三步：欲望（从创伤中自然生长）

**禁止**：
- 悬浮的欲望（"我想拯救世界"）
- 口号式欲望（"我要为正义而战"）

**要求**：欲望必须是弥补创伤/掩盖后遗症的**具体行动目标**

```
✅ 正确示范：
创业者的创伤：亲眼看着父亲因付不起医药费死在医院门口
欲望：赚钱，赚很多钱——不是因为贪心，是因为只有钱能给他安全感
（这个欲望会驱使他做出一系列选择，有些对，有些错，有些灰色）

✅ 《黑暗荣耀》文东恩
欲望：复仇
深层逻辑：不是为了"赢"，是为了穿越时空去抱一抱那个满身伤痕的18岁的自己
——复仇不是为了爽，是为了找回被偷走的人生
```

## 三、MBTI 人格模型：锁定行为逻辑一致性

### 用法

韩国编剧用 MBTI 作为角色的**底层行为逻辑代码**，确保长线叙事中角色行为前后一致。

### 与角色的对应

| MBTI | 行为特征 | 适用角色 |
|---|---|---|
| INTJ | 遇危机第一反应：冷酷逻辑计算得失 | 高智商反派、谋略家 |
| INFP | 内在价值观驱动，行动迟缓但坚定 | 理想主义者、殉道者 |
| ESTP | 结果导向，行动派，忽视情感 | 现实主义者、行动派 |
| ESFJ | 人际导向，照顾他人感受 | 温暖角色、牺牲者 |
| ISTJ | 规则至上，不变通 | 执法者、体制内角色 |

### 反脸谱化技巧

人格模型只是**骨架**，让角色反脸谱化的是：
1. **微小的反差**（善中带私，恶中藏情）
2. **两难的困境**（逻辑和情感的撕裂）

```
✅ 反派也有脆弱瞬间：
《秘密森林》反派检察长在外人面前翻手为云覆手为雨，
但深夜独自看着被害儿子的照片，流露出一丝极其隐秘的脆弱。

✅ 正面角色也有私心：
《请回答1988》德善作为家里老二，善良但也会嫉妒姐姐、委屈于父母偏心。
```

## 四、日常破局 + 冲突叠加公式

### 日常破局（第一幕核心）

**不需要**上来就炸白宫。用一个小到不能再小的异常，去戳角色的**内在创伤**。

```
✅ 示范：《请回答1988》德善第一幕危机
不是生离死别，而是作为家里老二，总是被要求让着姐姐和弟弟。
仅仅因为家里只剩两个荷包蛋，没有她的份儿。
她终于爆发了——这暴露了她内心深处"渴望被唯一关注"的软肋。
```

### 冲突叠加公式（高阶技巧）

在极小的物理空间内，同时引爆**四根引线**：

| 层级 | 引线类型 | 示例（车内爆胎场景） |
|---|---|---|
| 第一层 | 外部压力 | 荒郊爆胎 + 下大雨 |
| 第二层 | 内部挣扎 | 丈夫刚被裁员，银行卡没钱叫拖车 |
| 第三层 | 关系矛盾 | 丈夫用暴怒掩盖无能，与妻子激烈争吵 |
| 第四层 | 秘密曝光 | 妻子翻出手套箱里的房屋催收单 |

```
✅ 这一技巧的核心：
没有黑帮追杀，没有连环爆炸，
但观众的呼吸会停滞——因为情感张力被拉到了极限。

✅ 对本项目的启示：
不必每场戏都用大场面。
在一个出租屋里、一张饭桌上，
同样可以制造出令人窒息的戏剧张力。
```

## 五、情感传递：拒绝直白抒情

### 三条铁律

1. **非语言互动替代台词**
   - 国内流水线写法：男主把女主拉到墙角，深情说"我爱你"
   - 韩剧写法：整理围巾时指尖极度克制的无意触碰

2. **符号化道具承载情感**
   ```
   《请回答1988》阿泽：边缘磨损的旧棋谱 → 诉说常年孤独
   《鬼怪》金侁：常喝的香蕉牛奶 → 内心渴望纯真与少女感
   ```

3. **轻量化救赎**
   ```
   ❌ 传统写法：一场大雨洗刷所有罪恶 / 谈个恋爱就改变命运
   ✅ 韩剧写法：记住对方的小禁忌
     两人点外卖，其中一人极其自然地说"不要放葱啊"——
     因为他记得另一个人对葱过敏。
   这种带着各自伤痕、仍能在破碎生活里互相给予的不完美安慰，
   才是真实世界里的治愈。
   ```

### 台词提纯技巧

**永远不要直接说出情绪词，把它们转换成带时间跨度和动作细节的具象表达。**

```
❌ 灾难级台词：
"我等你等得好心寒。"

✅ 高级台词：
"桌上的饭我热了三次，你一条消息都没有。"
（饭热了三次 = 时间流逝 + 从期待到担心到死心的心理过程）
（观众脑海中会自动补全那个落寞的背影）
```

## 六、节拍控制：15分钟一个爆点

### 标准16集韩剧的节奏分布

| 集数 | 阶段 | 核心任务 |
|---|---|---|
| 第1-2集 | 建局 | 建立所有人物 + 三线各自铺垫 |
| 第3-4集 | 铺垫 | 三线开始出现交集点 |
| 第5-7集 | 深化 | 支线深化，主线出现新线索 |
| 第8集 | **中点反转** | **反转必须死死击中第一集就建立的深层创伤** |
| 第9-12集 | 高潮 | 所有线索收束，真相浮出水面 |
| 第13-15集 | 代价 | 角色弧光完成，牺牲与选择 |
| 第16集 | 落幕 | 带着伤疤继续生活的从容（克制的大结局） |

### 单集内部节拍

```
要求：每15分钟一个小冲突或情绪爆点

但——全程高能 = 全程疲惫
解法：快慢交替
一场极其密集的动作冲突/智斗后，
必定紧接一段慢节奏的情感渲染镜头，
用舒缓的音乐、特写、甚至看似无关紧要的散步，
给观众的神经一个缓冲带。
```

## 七、群像戏节奏控制

### 核心原则：确立节奏主导者

每一场群像戏开写之前，脑子里必须明确：
> **谁是这场戏里目的最强/性格最外向的人？**

这个人就是这场戏的**节奏主导者**（交响乐团的指挥）。

```
✅ 节奏主导者抛出激进提议
✅ 性格稳重的角色用缓慢的长句去反驳
✅ 性格急躁的角色用短句快节奏去附和/挖苦
✅ 甚至用语气词或肢体动作打断别人的长篇大论

效果：干瘪的文字文本瞬间有了高低错落的听觉节奏。
```

## 八、台词设计：符号互动替代直白

### 韩剧 vs 国内流水线

| 国内流水线 | 韩剧写法 |
|---|---|
| 角色说自己冷，男主说"谁让你不多穿" | 女主：我冷。男主：（沉默地脱下外套披在她身上） |
| 女主表白说"我真的很喜欢你" | 雨天，女主没带伞，男主出现，撑伞的手冻得发红 |
| 角色哭诉自己的悲惨身世 | 角色在别人家吃完一顿饭后，默默把碗洗了 |

### 本项目的台词借鉴

```
✅ 种老四风格（仪式型话语）：
"有些事，不是药能治的。你这病，在命里，不在身上。"
（实际是拖延时间观察对方反应，但听起来像玄学）

✅ 阿河风格（感知型沉默）：
他不说"我很难过"，他只是把烟灰缸里的烟头一个个按灭，
然后很轻地说一句："那就好。"

✅ 郭羽风格（堕落者的自嘲）：
"你知道律师和算命先生的区别吗？
律师给你编一个可信的故事。
算命先生给你编一个你自己想信的故事。
我？我现在两个都干。"
```

## 九、6段式大纲法（受力分析图）

**核心**：不是把剧情流水账罗列，而是写每一个外部事件与内部状态的对应关系。

```
❌ 错误写法：
"第三集：男主被陷害入狱。"

✅ 正确写法：
"第三集：男主被陷害入狱。
【外部事件】男主被关押
【受力分析】这彻底击碎了他对现有体制的盲目信任，
迫使他从被动承受转向主动利用体制漏洞——这是他内心蜕变的受力点。"
```

**原则**：大纲展示的是结构的强韧和情感流动的脉络，不是窗户上挂什么颜色的窗帘。

## 十、留白：韩国编剧最核心的竞争力

### 留什么

- 未解决的矛盾（不强行给每个人一个干净的结局）
- 明明到嘴边却死死咽下去的台词
- 拼图的最后一块——交给观众去填满

### 判断标准

> 如果你写完一场戏，回头看，发现你把角色的感受、动机、结局全部用台词说了——那就说过头了。

```
✅ 优秀的留白：
阿河用能力看到了一切，但他什么都没说。
他只是回家，给父亲倒了一杯酒。
父亲接过酒，看了儿子一眼，什么也没问。
两人沉默地喝完了那顿晚饭。

（所有该说的都在酒里、在眼神里。）
```
