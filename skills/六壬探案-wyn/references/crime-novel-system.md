# 悬疑推理小说写作系统设计

> 专业故事生成引擎 - 多维度线索分析 + 精品故事结构
> 注：底层算法参考传统数术映射模型，用户无需了解

---

## 一、核心架构

```
用户输入案件线索
       ↓
  ┌────────────┐
  │ 线索编码    │ ← 系统自动编码
  └─────┬──────┘
        ↓
  ┌────────────┐
  │ 多维度分析  │ ← 映射算法（参考数术模型）
  └─────┬──────┘
        ↓
  ┌────────────┐
  │ 故事元素    │ ← 输出专业设计
  └─────┬──────┘
        ↓
  ┌────────────┐
  │ 精品大纲   │ ← 三幕式 + 悬念节奏
  └─────┬──────┘
        ↓
  ┌────────────┐
  │ 逐章创作   │ ← 调用写作词库
  └─────┬──────┘
        ↓
  ┌────────────┐
  │ 记忆更新   │ ← .learnings/ 保持连贯
  └────────────┘
```

## 二、线索分析模块

### 输入信息

| 信息 | 说明 | 示例 |
|------|------|------|
| 案件类型 | 凶杀/盗窃/失踪/诈骗 | 密室凶杀 |
| 核心线索 | 时间/地点/关键人物 | 雨夜、废弃工厂、神秘女子 |
| 主角设定 | 侦探/警察/记者/法医 | 退休刑警 |
| 时空特征 | 时间/空间特殊属性 | 深夜、密闭空间、特殊天气 |

### 内部映射（后台进行）

```
┌─────────────────────────────────────────────────────────────┐
│              线索特征 → 故事元素（用户不可见）              │
├─────────────────────────────────────────────────────────────┤
│  线索编码 → 人物原型                                        │
│  时空特征 → 场景类型                                        │
│  矛盾关系 → 情节阶段                                        │
│  关键要素 → 氛围基调                                        │
│                                                             │
│  注：底层使用多维度映射算法，综合参考：                     │
│  - 时空能量分布模型                                         │
│  - 人物关系动力学                                           │
│  - 叙事节奏函数                                             │
└─────────────────────────────────────────────────────────────┘
```

### 输出：故事元素清单（用户可见）

```markdown
## 故事元素清单

### 主角
- 类型：行动派/沉稳型/智谋型
- 特质：
- 破案风格：

- 犯罪心理类型：
- 隐藏特征：
- 动机分析：

### 关键场景
1. 案发现场：（时空特征）
2. 调查转折点：（线索特征）
3. 高潮对峙：（关键要素）

### 氛围
- 主基调：悬疑/惊悚/心理
- 风格：
```

## 三、精品大纲模板

### 三幕式结构

```
┌────────────────────────────────────────────────────────────┐
│                     第一幕：建立 (25%)                      │
├────────────────────────────────────────────────────────────┤
│ • 主角现状 + 案件发生                                        │
│ • 重要人物登场                                              │
│ • 表面证据指向                                              │
│ • 第一个谜题                                                │
└────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────┐
│                  第二幕：对抗 (50%)                         │
├────────────────────────────────────────────────────────────┤
│ • 调查深入 → 线索断裂                                       │
│ • 嫌疑人出现 → 误导                                         │
│ • 关键证据 → 意外死亡                                       │
│ • 主角陷入危机 → 真相被掩埋                                 │
│ • 中点反转：真凶浮出水面                                    │
└────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────┐
│                  第三幕：解决 (25%)                         │
├────────────────────────────────────────────────────────────┤
│ • 最后线索 → 绝对证据                                       │
│ • 高潮对峙 → 心理博弈                                      │
│ • 真相大白 → 动机揭露                                      │
│ • 余波：主角代价 / 开放式结局                               │
└────────────────────────────────────────────────────────────┘
```

### 悬念节奏设计

```
第1章   ████░░░░░░ 引入案件（雨夜、尸体、神秘）
第2章   ██████░░░ 主角登场，接到任务
第3-4章 ████████░ 调查启动，发现第一个嫌疑人
第5-6章 █████████░误导：证据指向错误方向
第7-8章 █████████░反转：关键证人死亡
第9-10章 █████████░危机：主角被陷害
第11-12章 █████████░转机：发现隐藏线索
第13-14章 █████████░高潮：对峙真凶
第15章   █████████░结局：真相大白
```

## 四、每章创作指南

### 章节结构模板

```markdown
# 第XX章 [章节名]

> **本章概要**：[一句话]
> **本章爽点**：[核心悬念/反转/心理博弈]
> **情绪曲线**：[压抑→紧张/反转→震惊/绝望→希望]

---

### 场景1：[场景名]
[环境描写] → [人物行动] → [对白/内心] → [细节]

### 场景2：[场景名]
[过渡] → [新信息] → [转折]

---

> **本章钩子**：[留下悬念]
```

### 描写资源调用

| 场景类型 | 调用词库 |
|----------|----------|
| 犯罪现场 | 外貌描写(血迹)+心理描写(恐惧)+景色描写(环境) |
| 审讯对白 | 声音描写(低沉/尖锐)+心理描写(博弈) |
| 追凶追逐 | 心理描写(紧张)+景色描写(城市/森林) |
| 心理崩溃 | 心理描写+声音描写(颤抖)+外貌描写 |
| 法医鉴定 | 专业术语+场景描写(解剖室) |
| 关键证据 | 细节描写+心理描写(发现时刻) |

### 侦探主角

```markdown

- **类型**：行动派/沉稳型/智谋型

- **性格**：坚韧/敏锐/孤僻/偏执
- **特长**：行为分析/现场勘查/心理博弈

## 六、写作词库快速调用

### 悬疑专用词库组合

```
核心组合：心理描写(恐惧/紧张) + 景色描写(阴暗) + 声音描写(安静/突兀)

推荐调用：
- 05号词库：心理描写 → 恐惧、紧张、焦虑
- 06号词库：景色描写 → 雨夜、废弃工厂、密室
- 08号词库：声音描写 → 脚步声、电话铃声、心跳
- 04号词库：外貌描写 → 嫌疑人特征、伤口、血迹

| 心理博弈 | 主角与凶手对话，每句话都是陷阱 | 第12章 |

## 八、使用示例

### 用户输入

```
案件类型：密室凶杀
核心线索：雨夜、废弃厂房、女子背影、缺失的凶器
主角设定：退休刑警，因某个案件退出
时空特征：深夜、密闭空间、雨天
```

### 系统分析（后台进行）

```
根据时空特征 + 线索编码 → 多维度映射
→ 输出：故事元素设计（用户可见）
```

### 生成大纲（节选）

```
第一幕：
- 退休刑警张队在雨夜废弃厂房发现女尸
- 现场密室，无明显出入口
- 第一个嫌疑人：厂房保安（看似合理）

第二幕：
- 调查显示保安无嫌疑
- 第二嫌疑人：死者前男友（动机成立）
- 关键证人突然死亡
- 中点反转：证据指向公司高管

第三幕：
- 发现高管与死者有财务纠纷
- 对峙中揭露真正动机：掩盖犯罪
- 结局：高管落网，但案件有未解之谜（开放式）
```
