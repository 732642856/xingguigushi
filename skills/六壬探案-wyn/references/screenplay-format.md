# 场景/剧本格式参考手册

适配都市悬疑 + 黑色幽默 + 家族恩怨类型，参考《无证之罪》《隐秘的角落》《阴门铃医》剧本格式。

---

## 一、剧本格式标准

### 场次标题格式

```markdown
## 【场1】时间·地点·内/外
**人物**：角色A、角色B

△ 场景描述（环境氛围先行，3-5句话）

角色A：（动作描述）
角色B：（动作描述）

角色A：（台词，黑色幽默风格）
角色B：（台词）

角色A：（内心独白或沉默反应）
```

### 格式要素说明

| 要素 | 写法要求 |
|---|---|
| 时间 | 白天/夜晚/凌晨/黄昏 + 具体时段 |
| 地点 | 具体地点名称 |
| 内/外 | 内景（INT.）/ 外景（EXT.） |
| 人物 | 首场必须注明，后续可省略 |
| △ | 场景描述符号，后面跟氛围描写 |

## 二、场景描写标准

### 氛围优先原则

每一场戏的第一个任务：建立一个让观众相信的空间。

```markdown
✅ 正确示范：
【场1】深夜·城中村出租屋·内

△ 楼下的烧烤摊刚收摕，空气里还留着孜然味和炭火燎过的焦香。
墙上贴着一张八卦图，图下面是两排塑料红绸。
电视机里播着《今日说法》，声音开到最大，
因为隔壁在吵架——准确地说，是单方面在骂人。

（建立了：城中村/底层/父亲江湖人设/嘈杂环境）

❌ 错误示范：
【场1】深夜·出租屋·内

△ 他在家里。
```

### 细节堆砌法

用细节建立地域感和时代感，而不是直接说明。

```markdown
✅ 正确示范（不需要说"这是东北"，观众自然知道）：
【场1】冬夜·东北某县城·街道·外

△ 街边的小卖部亮着惨白的日光灯。
门口摆着一箱冰柜，旁边拴着一条脏兮兮的田园犬，
狗腿上缠着红布条——开春前不能杀狗的讲究。
远处传来唢呐声，不知道是办白事还是喜事。
（这片土地上，生死好像没那么分明。）

❌ 错误示范：
【场1】冬夜·东北县城
△ 这是一个寒冷的冬夜，我们的主角走在东北的街道上……
```

## 三、台词格式与风格

### 基本格式

```markdown
角色名：（动作/表情/心理）台词

角色名：（动作）"台词"（适用于需要精确引号的情况）

（旁白/内心独白——用括号，与动作描述区分）
```

### 台词风格：角色声音即性格

```markdown
✅ 种老四风格（江湖话术 + 文化包装）：
种老四：（翻着白眼看罗盘）此处风水……
（停顿一拍）
嗯，有点意思。
（不说"有问题"，说"有点意思"——给自己留余地）

✅ 阿河风格（自嘲 + 观察锐利 + 沉默胜于开口）：
阿河：（看着父亲数钱，数完把钱包塞进裤兜里）
"……又赚了？"
（不是疑问句，是陈述句。他早看出来了。）

阿河：（看着那个蒙面人的背影）
（他没有说话。
他只是把看到的一切，默默记在心里。）
（这种沉默比任何台词都更有力量。）

✅ 黑色幽默风格（平静地说可怕的事）：
骆闻：（看着手里的照片）这是第六个了。
（停顿）
不是我想杀他们。
是没人愿意查他们的案子。

✅ 温情风格（不说，说不出口）：
种老四：（看着阿河）你以后……
（停顿很久。）
（本来想说"以后要照顾好自己"。）
（最后变成了：）
"以后少熬夜。脸上全是痘。"
```

### 对话节奏：留白与沉默

```markdown
✅ 沉默的力量：
阿河：（看着父亲递过来的钱）
（没有接。）
（父亲也没有说话。）
（两个人就这样沉默地对峙了三秒。）
（然后父亲把钱收回裤兜，说：）
"行，我替你存着。"

✅ 台词之间的"呼吸"：
郭羽：（站起来，走向窗边）
"我不是好人。"
（看着窗外，很久。）
"但我也不是最坏的那个。"
（回过头，看着对方）
"你猜猜，我这辈子最羡慕什么人？"
```

## 四、动作描写标准

### 禁止 vs 要求

```markdown
❌ 禁止：
"他生气地站了起来。"
"她伤心地哭了。"
"他微笑着点了点头。"

✅ 要求（具体 + 可视化）：
"他站起来，把椅子往后一推——椅子腿在水泥地上划出一道刺耳的声音。"

"她没有哭。她只是把那张照片翻了过去，照片面朝下扣在桌上。"

"他没有点头。他只是把自己的酒杯和对方的碰了一下，仰头干了。"
```

### 微动作外化心理创伤

```markdown
✅ 示范：文东恩式的冰冷习惯外化
（不是直接写"她很痛苦"，而是写她的身体反应）

阿河：（在厨房洗东西，突然停下来）
（水龙头开着，水哗哗地流。
他盯着自己的手，看了很久。
然后他把手伸进冰凉的水里，像是在确认什么。）

父亲：（从后面进来，看到这一幕）
（没有说话。
只是把灶上的火关了，开水的蒸汽慢慢散了。）

（两人都知道那个习惯意味着什么。
但两人都没有提。）

✅ 示范：用物件的"反应"代替角色的"表达"：
种老四：（看着阿河递过来的那个盒子）
（盒子很旧，是十几年前的铁皮盒子，上面印着"上海手表厂"。）
（他的手抖了一下。
然后他用另一只手压住这只抖的手。）
"放下吧。回头再弄。"

（他不说"我不想要"，
但他的手抖了。
这个细节，比任何台词都有分量。）
```

## 五、内心独白使用规范

### 内心独白 = 最后的手段

能用动作/细节/沉默说清楚的，绝不用内心独白。

```markdown
✅ 正确（几乎不用内心独白）：
骆闻：（在黑暗里坐着，没有开灯）
（电视里在播新闻：又一起失踪案。
他听完了整条新闻，没有换台，也没有动。）
（这种案子，永远不会有人查。）
（但他知道他要找的那个人在哪里。）

✅ 滥用内心独白（反面教材）：
骆闻：（心想）这个案子已经死了六个人了，
（心想）凶手用"法律"来制裁这些人，
（心想）这种案子，只有我会去查……
（变成了流水账，毫无质感）
```

## 六、群戏写法

### 节奏主导者原则

每一场群像戏，必须先确定：
> **这场戏谁是节奏主导者？**

然后，其他人的反应必须与主导者形成**节奏对位**。

```markdown
✅ 示范：四个人在讨论事情
场景：派出所会议室，四个警察+一个线人

周警官：（站起来，把照片钉在墙上）这个人，三天前在建材市场出现过。

严良：（蹲在椅子上，嘴里嚼着槟榔）三天前？他那时候应该已经离开省了。

周警官：（没有理会严良，继续）我们查了建材市场的监控——

线人：（突然插嘴）你们查不到的。那个市场监控一年前就坏了，没人修。

（所有人的目光都看向线人。）

严良：（慢悠悠地）哦？你怎么知道的这么清楚？

线人：（脸色微变）我……我就是听人说的。

周警官：（与严良对视一眼）……继续说。

（节奏主导者是周警官，他控制这场戏的走向。
但严良用他的"慢悠悠"形成了节奏反差，制造紧张感。
线人则是一个变数，他的出现打破了原有的节奏。）
```

## 七、黑色幽默场景写法

### 场景：葬礼

```markdown
✅ 黑色幽默的葬礼：
【场1】下午·殡仪馆·内

△ 告别厅外停着十几辆黑色轿车，花圈堆到门口。
有人在门口发烟，有人低头看手机，有人站在一旁面无表情。
空气里是香烛味和来不及散的冷风。

（角色A走进告别厅，在遗体前站了两秒。）

角色A：（自言自语，声音很小）老李，上次见面你还说等出院了请我喝酒。
（停顿）
……行吧，这次算我请。你记着。

（然后他苹了一躬，转身出去了。
走到门口时，外面的人问："这么快？"
他："还有事。"
他："份子钱你帮我给了吗？"
对方："给了，多包了两百。"
角色A：（没回头）"……行。回头还你。"

（他走了。
没有人知道，他今天中午刚被公司开除。）
```

### 场景：医院

```markdown
✅ 黑色幽默的医院：
【场1】深夜·医院走廊·内

△ 走廊里有消毒水的味道，混着某种廉价方便面的香气。
护士站传来按铃声，响了三下，停了。
某间病房里传来哭声，哭了一会儿，停了。

种老四：（坐在走廊的塑料椅子上，面前放着一个CT袋子）
（他在等人。）

阿河：（走过来，在他旁边坐下）
（看了一眼那个CT袋子）"有问题？"

种老四：（把袋子收起来）"老了，都有。"
（停顿）"你吃饭了吗？"

阿河：（摇头）"没有。"

种老四：（站起来，把CT袋子塞进外套里）
"走，请你吃面。医院门口的别吃，不新鲜。
过两条街，有一家……"

阿河：（打断他）"你又没钱了？"

种老四：（停顿）
（然后笑了）"小看人。我有钱。今天刚挣的。"

阿河：（看着他手里的CT袋子）
（那个袋子，比他上次拿的那个要薄得多。）

（两人都没有再说话。
阿河站起来，跟父亲往外走。
走到楼梯口，种老四突然停下来。）
"今天那个事……算了，没什么。"

阿河：（没有追问）
（"算了"两个字，在这个家庭里，有时候比很多东西都重。）
```

## 八、转场技巧

### 声音转场

```markdown
✅ 声音转场：
骆闻放下电话。
（电话那头是诈骗团伙的下一串号码。）
与此同时，画面外——
△ 同一个城市的另一边，一辆车在深夜的街道上飞驰。
（用声音把两个空间连起来：电话那头的声音 = 车上的人的声音）

✅ 时间转场：
（在医院）
种老四：（把病历本合上）
"行，我知道了。"
他站起来，把病历本塞进外套里。

△ 当他再次出现，是在三天后的城中村。
（用黑色淡入淡出，而不是硬切）

✅ 物件转场：
骆闻：（打开抽屉，拿出一个信封）
△ 信封里是一张照片——死者生前的照片。

△ 画面定格在这张照片上。
然后画面慢慢放大——
变成了真实的第一现场。
（照片变成了叙事中的"窗口"，连接回忆和现实）
```

## 九、集与集之间的连接

### 留钩子的位置

```
最佳位置：每集的最后5分钟

❌ 错误：
"第一集结束了，大家下集再见！"

✅ 正确（用动作/事件做钩子）：
种老四：（看着阿河）你今天……看到什么了？

阿河：（沉默了很久）
（然后他摇了摇头）
"没什么。"

（画面定格在阿河的脸上。
他的眼神里有什么。
但他选择了不说。）

（下集预告不需要说出来——
这个沉默，就是最好的预告。）
```

### 黑屏前的最后一句台词

```markdown
✅ 最佳钩子台词（来自《无证之罪》）：
"有些债，是用命还的。"

✅ 最佳钩子动作（来自《隐秘的角落》）：
张东升：（在窗边，看着楼下的三个小孩）
（他笑了。）
（镜头慢慢拉远。）

（这个笑容，胜过任何台词。）

✅ 最佳钩子沉默（来自本项目参考）：
阿河：（看着父亲离去的背影）
（他没有追上去。
他只是站在原地，看着父亲的背影消失在街角。）
（然后他低头，看了看自己的手。）

他的手在抖。

（不是因为冷。）
```

## 十、快速检查清单

写完每一场戏后，自问：

```
□ 这场戏的环境描写是否能让人"看见"这个空间？
□ 角色的动作是否足够具体（不是"生气地"而是具体的身体语言）？
□ 台词是否是这个角色独有的声音？
□ 这场戏里有没有一个"动作"可以代替"说"？
□ 群像戏里，谁是节奏主导者？
□ 这场戏的情感是冷还是热？是否符合当前集/整个故事的节奏需要？
□ 这场戏的结尾是否有钩子？
□ 是否需要内心独白？（尽量不要用）
```
