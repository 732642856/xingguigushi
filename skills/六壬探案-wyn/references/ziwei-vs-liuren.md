# 紫微斗数 vs 大六壬 对比表

> 用于故事创作时的选型参考

---

## 核心定位对比

| 系统 | 定位 | 擅长领域 | 适合题材 |
|------|------|----------|----------|
| **紫微斗数** | 命理预测系统 | 人物命运、性格特质、人生轨迹 | 角色设计、人物命运、性格刻画 |
| **大六壬** | 事态占卜系统 | 事件发展、时机判断、人际关系 | 悬疑推理、案件推理、情节推进 |

---

## 紫微斗数（角色设计）

### 核心要素

| 要素 | 数量 | 说明 |
|------|------|------|
| 十四主星 | 14颗 | 紫微、天机、太阳、武曲、天同、廉贞、天府、巨门、天相、天梁、七杀、破军、贪狼、禄存 |
| 辅星杂曜 | 108颗 | 文昌、文曲、左辅、右弼、天魁、天钺等 |
| 十二宫 | 12个 | 命宫、兄弟宫、夫妻宫、子女宫、财帛宫、疾厄宫、迁移宫、奴仆宫、官禄宫、田宅宫、福德宫、父母宫 |
| 四化 | 4化 | 化禄、化权、化科、化忌 |

### 故事创作应用

```
输入：角色基础属性（性别/年龄/性格/职业/背景）
      ↓
匹配：144种标准命盘
      ↓
输出：完整人物设定
      - 性格特质
      - 命运走向
      - 人生节点
      - 与其他角色的关系
```

### 适用场景

- 创建有深度的角色
- 设计角色的命运轨迹
- 构建人物之间的互动关系
- 角色的成长弧线

---

## 大六壬（悬疑推理）

### 核心要素

| 要素 | 数量 | 说明 |
|------|------|------|
| 十二天将 | 12个 | 贵人、腾蛇、朱雀、六合、勾陈、青龙、天空、白虎、太常、玄武、太阴、天后 |
| 十二地支 | 12个 | 子丑寅卯辰巳午未申酉戌亥 |
| 四课 | 4课 | 事体现状、隐藏因素、互动关系、结果 |
| 三传 | 3传 | 初传（启动）、中传（发展）、末传（终结） |
| 五行 | 5行 | 金木水火土 |

### 故事创作应用

```
输入：案件线索（时间/地点/人物/事件特征）
      ↓
起盘：大六壬排盘算法
      ↓
映射：盘面→故事元素
      输出：
      - 嫌疑人画像（天将）
      - 场景设计（地支）
      - 情节阶段（三传）
      - 氛围基调（神将组合）
```

### 适用场景

- 悬疑推理小说
- 犯罪侦查故事
- 复杂案件设计
- 情节反转设计

---

## 对比总结

| 维度 | 紫微斗数 | 大六壬 |
|------|----------|--------|
| **核心关注点** | "人" - 命运与性格 | "事" - 事件与发展 |
| **输出形式** | 静态角色设定 | 动态情节发展 |
| **时间维度** | 人生轨迹（一生） | 事件发展（当下） |
| **使用场景** | 角色设计 | 推理设计 |
| **适合题材** | 都市/古言/仙侠 | 悬疑/推理/犯罪 |

---

## 组合使用建议

可以同时使用两个系统：

```
紫微斗数 → 角色设计
大六壬 → 情节推理

案例：
1. 用紫微斗数设计角色性格与命运
2. 用大六壬设计角色之间的互动与事件发展
```
