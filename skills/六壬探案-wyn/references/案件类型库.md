# 悬疑推理案件类型库

## 一、凶杀案类

### 1.1 密室杀人案
```
特征：不可能犯罪，现场封闭，凶手如何进出
线索类型：物理痕迹、时间线、密道、假象
经典模式：不可能进→不可能出→心理密室→时间差
难度等级：高
适用类型：古典推理
```

### 1.2 完美犯罪案
```
特征：看似完美的犯罪，最终被细节击败
线索类型：完美中的微小瑕疵、证词矛盾、动机隐藏
经典模式：无懈可击→百密一疏→细节决定
难度等级：中-高
适用类型：心理悬疑
```

### 1.3 模仿作案案
```
特征：模仿经典案例，增加破案难度
线索类型：模仿痕迹、差异点、原始案件
经典模式：多起案件→发现关联→找到原型→破解
难度等级：中
适用类型：动作惊险
```

### 1.4 连锁杀人案
```
特征：多个受害者，相关但非同一凶手
线索类型：共同点、差异点、真实动机
经典模式：独立案件→发现关联→真凶另有其人
难度等级：中-高
适用类型：心理悬疑/社会派
```

### 1.5 复仇杀人案
```
特征：多年后的复仇，动机深藏
线索类型：时间线、旧案、人物关系
经典模式：当下案件→陈年旧案→真相浮出
难度等级：低-中
适用类型：社会派
```

## 二、经济犯罪类

### 2.1 诈骗案
```
特征：高智商犯罪，涉及金额巨大
线索类型：账户、IP、话术、受害者证词
经典模式：小额试试→大额入局→人间蒸发
难度等级：中
适用类型：商战悬疑
```

### 2.2 贪污腐败案
```
特征：内部人员，证据难找
线索类型：账目、审批、假象
经典模式：匿名举报→查无所获→内部突破
难度等级：中-高
适用类型：反腐/商战
```

### 2.3 洗钱案
```
特征：合法外衣下的非法资金
线索类型：交易链、假公司、资金流向
经典模式：异常交易→追踪资金→证据链
难度等级：高
适用类型：商战/犯罪纪实
```

## 三、失踪类

### 3.1 儿童失踪案
```
特征：紧急、感人、社会关注度高
线索类型：目击者、时间线、社交圈
经典模式：紧急搜寻→线索中断→意外发现
难度等级：低-中
适用类型：社会派/动作惊险
```

### 3.2 成年人失踪案
```
特征：可能是自愿离开，可能是被迫
线索类型：通讯、设备、财务、动机
经典模式：失踪报告→调查轨迹→两种可能
难度等级：中
适用类型：心理悬疑
```

## 四、盗窃类

### 4.1 艺术品盗窃案
```
特征：高智商，目标价值连城
线索类型：安防系统、时间、销赃渠道
经典模式：案发→排查→江洋大盗→追回
难度等级：中-高
适用类型：动作惊险
```

### 4.2 珠宝盗窃案
```
特征：目标明确，嫌疑人范围小
线索类型：现场痕迹、目击、保险
经典模式：精心策划→意外变故→身份暴露
难度等级：低-中
适用类型：古典推理
```

## 五、网络犯罪类

### 5.1 黑客攻击案
```
特征：虚拟世界，证据易销毁
线索类型：IP、代码、攻击模式
经典模式：攻击→溯源→真假线索→反击
难度等级：高
适用类型：现代悬疑
```

### 5.2 网络诈骗案
```
特征：跨境、匿名、受害者众
线索类型：账户、话术、IP、受害者
经典模式：小案→串并→背后组织→收网
难度等级：中-高
适用类型：社会派
```

### 5.3 信息泄露案
```
特征：隐私暴露，引发连锁反应
线索类型：泄露源、传播链、动机
经典模式：泄露→恐慌→追查→真相
难度等级：中
适用类型：现代悬疑/社会派
```

## 六、社会事件类

### 6.1 校园事件
```
特征：涉及未成年人，社会敏感
线索类型：校园关系、社交媒体、动机
经典模式：事件发生→保护or调查→真相or更多谜
难度等级：低-中
适用类型：社会派
```

### 6.2 医患纠纷
```
特征：专业性强，灰色地带
线索类型：病例、医疗记录、动机
经典模式：意外or谋杀→专业调查→两难选择
难度等级：中-高
适用类型：社会派/心理
```

### 6.3 工伤事故
```
特征：责任认定困难，利益相关
线索类型：现场、记录、利益链
经典模式：意外→伪造→真相→追责
难度等级：低-中
适用类型：社会派/商战
```

## 七、心理犯罪类

### 7.1 精神控制案
```
特征：看不见的操控，受害者不自知
线索类型：行为异常、关系、操控者
经典模式：异常行为→发现操控→营救or更多谜
难度等级：高
适用类型：心理悬疑
```

### 7.2 诱导自杀案
```
特征：看似自杀实则他杀
线索类型：心理状态、通讯、异常
经典模式：自杀结论→疑问→重新调查→他杀
难度等级：中-高
适用类型：心理悬疑
```

### 7.3 的人格分裂案
```
特征：嫌疑人有不同人格，真相复杂
线索类型：目击矛盾、时间线、行为差异
经典模式：证据指向分裂→治疗→人格对话→真相
难度等级：高
适用类型：心理悬疑
```

## 八、历史悬案类

### 8.1 陈年旧案
```
特征：多年未破，证据缺失
线索类型：旧档案、新技术、利益相关人
经典模式：旧案重提→新线索→重新调查→翻案
难度等级：中
适用类型：社会派
```

### 8.2 考古/历史谜案
```
特征：与历史事件相关，真相可能改变认知
线索类型：文物、历史文献、专业知识
经典模式：发现→研究→争议→真相
难度等级：中-高
适用类型：历史悬疑
```
