# 大六壬·故事编织器

> 用古代占卜体系驱动现代故事创作的交互式 Web App

---

## 产品定位

用户通过一系列选择（模拟大六壬起课过程），让系统根据六壬卦象定位出「天时地利人和」的情境框架，再通过不断的选择和引导，辅助用户写出结构完整、命运感强烈的故事情节。

**核心价值：** 不是算命，是借用六壬的哲学框架（时空交叉、三传递进）为故事提供「命运骨架」，用户在骨架上填肉。

---

## 用户流程

### 第一阶段：定位（起课）
用户通过选择来确定六壬模型：

1. **选择时间维度**
   - 年支（12选1：子丑寅卯辰巳午未申酉戌亥）
   - 月将（12选1，对应节气）
   - 时支（12选1）

2. **选择问题类型**（事件性质）
   - 感情 / 事业 / 财运 / 出行 / 争讼 / 求职 / 其他

3. **选择主角身份**
   - 性别、年龄段、身份标签（如：都市白领 / 落魄侦探 / 小镇青年…）

系统根据以上输入，自动计算：
- **天地盘**（天盘加临地盘）
- **四课**（第一课到第四课）
- **三传**（初传、中传、末传）
- **贵人方位**（吉凶方向）
- **旬空**（空亡的干支）

---

### 第二阶段：解读（卦象 → 故事要素）

系统把六壬卦象翻译成故事结构：

| 六壬要素 | 对应故事元素 |
|---|---|
| 初传（发用） | 故事开端 / 触发事件 |
| 中传 | 核心矛盾 / 转折点 |
| 末传 | 结局走向 / 命运归宿 |
| 天盘神将 | 外部力量（命运/对手/机遇） |
| 地盘支 | 主角内在状态 / 环境底色 |
| 贵人方位 | 贵人角色 / 转机来自哪里 |
| 旬空 | 虚空之物 / 注定失去或得不到的 |
| 课式类型 | 整体故事基调（如：元首课=顺势、重审课=迂回、八专课=孤独…） |

---

### 第三阶段：选择（用户定制故事）

用户面对一系列故事选择题，每一题都基于六壬解读：

**示例选择题：**
> 初传「太阳临午」预示强烈的起点。你的主角会以哪种方式登场？
> - A. 在人群中突然接到一个神秘电话
> - B. 深夜独自发现了某个不该知道的秘密  
> - C. 一次意外让他/她站在了命运的岔路口

每次选择都会：
1. 推进故事情节
2. 根据六壬中传调整下一步的选项
3. 积累「故事碎片」

---

### 第四阶段：生成（输出成品）

所有选择完成后，系统将：
1. 把用户的选择串联成**完整故事大纲**
2. 生成**精彩的故事开头段落**（AI 润色）
3. 提供**续写提示**（基于末传走向的写作建议）
4. 可选：导出为 Markdown / 复制到剪贴板

---

## 技术栈（建议）

- **前端**：React + TypeScript（或 Vue 3）
- **样式**：Tailwind CSS，风格：暗色调 + 水墨纹理，古典与现代融合
- **大六壬计算引擎**：纯 JavaScript 实现（无需后端）
- **AI 润色**：调用 Gemini API（google/gemini-2.0-flash）
- **数据存储**：localStorage（本地保存用户故事）
- **部署**：静态页面，可直接部署到 Vercel / GitHub Pages

## 大六壬计算规则（核心算法）

### 天地盘起法
- 以月将加时支，顺布十二支于地盘
- 天盘以子起，顺数到月将所在位置，其余依次排列

### 四课取法
1. 第一课：以日干寄宫为地，其上天盘支为天
2. 第二课：以第一课天盘支为地，其上天盘支为天  
3. 第三课：以日支为地，其上天盘支为天
4. 第四课：以第三课天盘支为地，其上天盘支为天

### 三传取法（常用：贼克法/比用法/涉害法）
- **元首课**：上克下，取上为用
- **重审课**：下克上，取下为用
- **知一课**：一上克一下（或反之）
- 中传：从初传支起，按五行生克关系取
- 末传：从中传支起，继续推

### 十二天将对应
| 天将 | 对应神兽 | 故事意象 |
|---|---|---|
| 贵人 | 玉堂 | 贵人相助、转机 |
| 螣蛇 | 蛇 | 惊扰、虚惊、噩梦 |
| 朱雀 | 朱雀 | 文书、消息、争吵 |
| 六合 | 青龙之子 | 合作、婚姻、和解 |
| 勾陈 | 土 | 纠纷、阻滞、官司 |
| 青龙 | 青龙 | 财运、机遇、喜事 |
| 天空 | 空 | 空洞、谎言、失约 |
| 白虎 | 白虎 | 凶险、疾病、意外 |
| 太常 | 礼 | 宴会、喜庆、衣食 |
| 玄武 | 玄武 | 盗窃、暗算、隐秘 |
| 太阴 | 阴 | 隐藏、女性、幕后 |
| 天后 | 后 | 女主角、感情、柔情 |

## 故事基调对照（课式）

| 课式 | 基调 | 适合的故事类型 |
|---|---|---|
| 元首课 | 顺势而为，主动出击 | 英雄崛起、创业逆袭 |
| 重审课 | 被动承受，以退为进 | 悬疑反转、以弱胜强 |
| 知一课 | 单线突破，专注一事 | 爱情故事、单一目标 |
| 涉害课 | 层层阻碍，步步惊心 | 冒险、犯罪、悬疑 |
| 遥克课 | 隔空影响，幕后推手 | 阴谋论、权谋故事 |
| 八专课 | 孤立无援，自力更生 | 孤独旅人、末日生存 |
| 伏吟课 | 原地踏步，内心挣扎 | 心理剧、困境突围 |
| 返吟课 | 大起大落，命运弄人 | 黑色幽默、荒诞喜剧 |
