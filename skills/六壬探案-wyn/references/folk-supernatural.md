# 民间玄学/奇术元素设计手册

适配都市悬疑 + 民间奇术类型，基于《阴门铃医》世界观整理。

---

## 一、设计原则（三条铁律）

### 铁律1：能力必有代价

没有代价的能力 = 没有戏剧张力的超能力。

```
好的设计：
遥视能力
  ↔ 代价：
  - 无法主动关闭，被动接收
  - 感知≠理解，容易误读（大脑会"脑补"）
  - 使用后剧烈头痛
  - 感知到的惨剧无法改变，只能看它发生

差的设计：
遥视能力
  ↔ 代价：偶尔头疼（太轻了，没戏剧性）
```

### 铁律2：体系要有"中间地带"

```
不是"真灵"   ←————————→   不是"纯骗"
     ↓
    戏剧空间在这里
    （有多少是真的？有多少是大脑脑补的？
      有多少是骗子父亲的包装话术？
      这个模糊地带，就是最好看的戏）
```

### 铁律3：能力只是线索，不是答案

```
玄学 → 提供碎片/方向（但不能是唯一线索）
         ↓
刑侦 → 验证/推翻/复杂化
         ↓
真相 → 往往在"信玄学"和"信科学"之间
```

## 二、民间奇术类型库

### 类型A：感知/诊断型

**定义**：能"看见"或"感知"某些常人无法感知的信息。

**代表**：《阴门铃医》阿河的"遥视"

**设计框架**：

```
感知类型：接触性 / 共情性 / 场域性 / 预言性

触发条件（信标）：
- 物品（接触性）：触碰与目标关联的物品
- 情绪（共情性）：周围有极度强烈的情绪信号
- 空间（场域性）：进入残留强能量的场所

感知形式：
- 感官碎片：视觉/听觉/嗅觉/触觉的零散片段
- 象征性图像：信息以意象出现而非直接画面
- 情绪共振：直接感受到某种强烈的情绪

解读方式：
- 自己解读（容易出错）
- 借助他人解读（形成依赖关系）
- 记录原始碎片后由他人帮助分析

代价类型：
- 生理代价：头痛、流鼻血、短暂失明、昏迷
- 心理代价：被动接收负面情绪导致心理创伤
- 道德代价：看见了却无法改变，眼睁睁看惨剧发生
```

### 类型B：仪式/表演型

**定义**：不是真正的超自然能力，而是将心理暗示、江湖话术、行业知识包装成"神秘仪式"。

**代表**：种老四的"铃医驱邪"

**设计框架**：

```
"医术"类型：
- 诊断型：看人就能知道"病"（基于观察和经验）
- 仪式型：复杂的仪式过程（实际是在做心理治疗/拖延/观察）
- 混合型：真本事（部分）+ 包装（部分）

"诊所"设定：
- 物理空间：破旧的出租屋、墙上的人体经络图
- 符号系统：符咒、器物、香烛、特殊服饰
- 话术体系：专业术语 + 民俗文化 + 心理暗示

表演的"真实"部分：
-江湖经验：对人性的洞察
- 行业知识：医学/民俗/心理学的通俗版应用
- 观察能力：通过对细节的观察推断信息

表演穿帮的风险：
- 制造情节危机的来源
- 穿帮后角色如何应对，是戏剧性的重要来源
```

### 类型C：命理/推算型

**定义**：紫微、八字、风水等传统命理体系，用作宿命感制造和悲剧预言。

**代表**：《阴门铃医》中可能涉及的命理元素

**设计框架**：

```
用法一：宿命预言
- 用命理暗示悲剧结局
- 但留有"解法"的空间（改命/逆命）
- 宿命感 + 人为选择之间的张力

用法二：信息提取
- 通过命盘/八字获取角色背景信息
- 刑侦工具（查人背景）
- 但信息不完整，需要实地验证

用法三：叙事锚点
- 每个角色有一个命理标签（命宫主星/命格）
- 命格与性格形成对照
- 命格与命运形成反讽（"注定"悲剧 vs 人的选择）

命理的局限：
- 算命先生可能出错（人有自由意志）
- 算命先生可能故意说谎（控制对方）
- 解读方式不同，结论不同
```

### 类型D：民俗/禁忌型

**定义**：利用地方民俗、民间禁忌、仪式传统来制造悬疑氛围。

**代表**：《阴门铃医》中的祭祀、驱邪场面

**设计框架**：

```
民俗类型：
- 丧葬仪式：停尸、守夜、出殡的规矩
- 祭祀文化：祭祖、请神、送鬼的流程
- 民间禁忌："正月不能理发"、"怀孕不能参加葬礼"
- 地方传说：本地的鬼故事/灵异事件（作为背景调料）

禁忌的叙事功能：
- 禁忌被打破 = 情节转折点
- 禁忌被遵守 = 人物牺牲/代价
- 禁忌的真相 = 揭示世界观深层逻辑

氛围构建：
- 感官描写：气味（香烛味/纸钱味）、声音（诵经声）、光线（烛火）
- 空间描写：祠堂/墓地/荒野祭祀点
- 人物描写：仪式主持者的服饰/道具/姿态
```

## 三、能力与刑侦的结合模式

### 模式一：能力→线索→验证

```
能力使用者提供碎片信息
     ↓
刑侦/调查者实地验证
     ↓
部分验证 + 部分推翻
     ↓
真相在两者之间浮现
```

**示例**：
```
遥视碎片：铁锈味 + 弯曲形状 + 冰冷感
     ↓
调查者解读：可能是"沾血的扳手"（骗子的包装）
     ↓
实地调查：确实有一把扳手，但不在案发现场，
          而在嫌疑人父亲（电工）的工作间里
     ↓
真相：嫌疑人父亲才是凶手，嫌疑人是被冤枉的
```

### 模式二：能力→误导→反转

```
能力使用者提供看似清晰的信息
     ↓
所有人都以为真相已明
     ↓
调查深入后发现能力被误导（情绪/记忆污染）
     ↓
真相颠覆，之前建立的一切都要重审
```

**示例**：
```
遥视碎片：模糊的人影 + 车的声音 + 尖叫
     ↓
结论：车祸
     ↓
调查后发现：是肇事逃逸，但不是意外，是故意的
遥视没有看到动机——因为动机是人内心的东西，
感知碎片只能呈现表面信息
```

### 模式三：能力→道德困境

```
能力使用者"看见"了某人的未来（可能是悲剧）
     ↓
他面临选择：说出来（可能改变）vs 不说（接受命运）
     ↓
说出来：改变了对方的命运，但未必是好的改变
不说：目睹悲剧发生，承担道德重压
```
